## 2. 背景与应用场景

飞轮储能系统是转动中功与能转换的典型工程应用。当电力富余时，电机将飞轮加速到数万转/分钟，将电能转化为飞轮的转动动能储存；需要时飞轮带动发电机将动能回收为电能。飞轮设计的关键参数是转动惯量和转速——储存的能量为 $\frac12 J\omega^2$，提高 $\omega$ 比增大 $J$ 更有效（平方关系），但受材料强度限制。

> **直觉理解**：平推一个物体，速度越快动能越大（$E_k = \frac12 mv^2$）。转动同理——角速度越快、转动惯量越大，转动动能越大（$E_k = \frac12 J\omega^2$）。力矩做功和平动中力做功的公式完全对应：$F \to M$，$x \to \theta$，$v \to \omega$。

从能量角度分析转动问题往往比直接用力矩和转动定律更简单——尤其在涉及变力矩或多物体系统时，能量守恒可以避免复杂的力矩分析。

## 3. 核心概念与符号约定

### 力矩的功

当力矩 $M$ 使刚体转过 $\mathrm{d}\theta$ 角位移时，力矩做的元功：

$$
\mathrm{d}A = M\,\mathrm{d}\theta
$$

从 $\theta_1$ 到 $\theta_2$ 的总功：

$$
A = \int_{\theta_1}^{\theta_2} M\,\mathrm{d}\theta
$$

- 若 $M$ 为常量：$A = M(\theta_2 - \theta_1) = M\Delta\theta$
- 若 $M$ 随 $\theta$ 变化：需积分求解

力矩做功的物理本质：力 $\mathbf{F}$ 在作用点切向位移上的累积——$\mathrm{d}A = F_\tau \mathrm{d}s = F_\tau \cdot r\mathrm{d}\theta = M\mathrm{d}\theta$。

### 转动动能

刚体绕定轴转动的动能等于各质元动能之和：

$$
E_k = \sum \frac12 \Delta m_i v_i^2 = \sum \frac12 \Delta m_i (r_i\omega)^2 = \frac12 \left(\sum \Delta m_i r_i^2\right) \omega^2 = \frac12 J\omega^2
$$

### 转动动能定理

**合外力矩对刚体做的功等于刚体转动动能的增量**：

$$
A = \frac12 J\omega_2^2 - \frac12 J\omega_1^2
$$

### 刚体的重力势能

刚体在均匀重力场中的重力势能等于总质量集中在质心处的势能：

$$
E_p = mgh_C
$$

其中 $h_C$ 为质心到重力势能零点的垂直高度。这一结论源于 $E_p = \sum \Delta m_i gh_i = g\sum \Delta m_i h_i = mgh_C$。

### 纯滚动中的总动能

车轮做无滑滚动时，总动能 = 质心平动动能 + 绕质心转动动能：

$$
E_k = \frac12 mv_C^2 + \frac12 J_C\omega^2
$$

利用无滑条件 $v_C = \omega R$ 可统一为单一变量表示。

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $A$ | 力矩做功 | $\mathrm{J}$ |
| $M$ | 力矩 | $\mathrm{N \cdot m}$ |
| $E_k$ | 转动动能 | $\mathrm{J}$ |
| $J$ | 转动惯量 | $\mathrm{kg \cdot m^2}$ |
| $\omega$ | 角速度 | $\mathrm{rad/s}$ |
| $h_C$ | 质心高度 | $\mathrm{m}$ |

## 4. 原理与方法

### 平动与转动的功-能关系对比

| 概念 | 平动 | 转动 |
|------|------|------|
| 做功的物理量 | 力 $F$ | 力矩 $M$ |
| 位移 | 线位移 $\mathrm{d}x$ | 角位移 $\mathrm{d}\theta$ |
| 元功 | $\mathrm{d}A = F\,\mathrm{d}x$ | $\mathrm{d}A = M\,\mathrm{d}\theta$ |
| 动能表达式 | $E_k = \frac12 mv^2$ | $E_k = \frac12 J\omega^2$ |
| 动能定理 | $A = \Delta(\frac12 mv^2)$ | $A = \Delta(\frac12 J\omega^2)$ |
| 势能表达 | $E_p = mgh$（质点） | $E_p = mgh_C$（质心） |

### 刚体转动能量问题的解题策略

**方法一：转动动能定理**
1. 确定研究对象和转轴
2. 计算所有外力矩做功（或合外力矩的功）
3. 写出初末状态的转动动能
4. 列方程 $A = \frac12 J\omega_2^2 - \frac12 J\omega_1^2$

**方法二：机械能守恒**
1. 判断是否有非保守力矩做功（如摩擦力矩、阻力矩）
2. 若只有保守力矩做功，机械能守恒
3. 写出初末状态的总机械能（平动动能 + 转动动能 + 重力势能）
4. 列守恒方程

**方法选择**：
- 已知力矩随角度变化关系 → 用动能定理（积分求功）
- 只有重力（或弹力）做功 → 用机械能守恒更简便
- 涉及纯滚动的多体系统 → 用能量法可避免分析内力

### 纯滚动中平动与转动动能的分配

以均匀圆盘（$J = \frac12 mR^2$）无滑滚动为例：

$E_k = \frac12 mv^2 + \frac12 \cdot \frac12 mR^2 \cdot \frac{v^2}{R^2} = \frac12 mv^2 + \frac14 mv^2 = \frac34 mv^2$

即：平动动能占 $2/3$，转动动能占 $1/3$。不同形状的刚体分配比例不同：

| 刚体形状 | $J_C$ | 总动能（纯滚动） | 转动动能占比 |
|---------|------|----------------|:----------:|
| 圆环 | $mR^2$ | $mv^2$ | $1/2$ |
| 圆盘/圆柱 | $\frac12 mR^2$ | $\frac34 mv^2$ | $1/3$ |
| 实心球 | $\frac25 mR^2$ | $\frac{7}{10}mv^2$ | $2/7$ |

## 5. 例题

### 例题 1：力矩做功与转动动能定理

一个飞轮的转动惯量 $J = 4.0\,\mathrm{kg \cdot m^2}$，在恒定力矩 $M = 8.0\,\mathrm{N \cdot m}$ 作用下从静止开始转动。飞轮同时受到恒定的阻力矩 $M_f = 2.0\,\mathrm{N \cdot m}$。

(1) 求飞轮转过 $20\,\mathrm{rad}$ 后的角速度
(2) 若此时撤去驱动力矩，飞轮在阻力矩作用下还能转多少圈才停止？

**解**：

**建模分析**：合外力矩 $M_{\text{合}} = M - M_f = 6.0\,\mathrm{N \cdot m}$。

**(1)** 恒力矩做功：$A = M_{\text{合}} \cdot \theta = 6.0 \times 20 = 120\,\mathrm{J}$

由转动动能定理：$120 = \frac12 \times 4.0 \times \omega^2$

$\omega^2 = \frac{2 \times 120}{4.0} = 60$

$\omega = \sqrt{60} \approx 7.75\,\mathrm{rad/s}$

**(2)** 撤去驱动力矩后，只有阻力矩 $M_f = -2.0\,\mathrm{N \cdot m}$ 做功。设还能转动 $\theta'$ 后停止：

$A' = M_f \cdot \theta' = 0 - \frac12 J\omega^2$

$-2.0 \cdot \theta' = -\frac12 \times 4.0 \times 60$

$\theta' = \frac{120}{2.0} = 60\,\mathrm{rad}$

约 $60/2\pi \approx 9.55$ 圈

**验证**：全过程总功 $A_{\text{总}} = 6.0 \times 20 + (-2.0) \times 60 = 120 - 120 = 0\,\mathrm{J}$，初末动能均为零，动能定理一致。✅

### 例题 2：细棒下摆——机械能守恒

一质量为 $m = 2.0\,\mathrm{kg}$、长度 $L = 1.2\,\mathrm{m}$ 的均匀细棒，一端用光滑铰链固定在水平轴上。棒从水平位置由静止释放，求棒经过竖直位置时的角速度和末端线速度。

**解**：

**建模分析**：只有重力做功（铰链光滑，不做功），机械能守恒。取水平位置为重力势能零点。

**初状态**（水平，$\theta = 0^\circ$，速度为零）：

$E_{k1} = 0$，$E_{p1} = 0$（已设零点）

**末状态**（竖直，$\theta = 90^\circ$）：

质心高度下降了 $\frac{L}{2}$，重力势能 $E_{p2} = -mg\frac{L}{2}$

转动动能 $E_{k2} = \frac12 J\omega^2$，$J = \frac13 mL^2$

机械能守恒：$0 = \frac12 J\omega^2 - mg\frac{L}{2}$

$\frac12 \cdot \frac13 mL^2 \cdot \omega^2 = mg\frac{L}{2}$

$\frac16 mL^2 \omega^2 = mg\frac{L}{2}$

$\omega^2 = \frac{3g}{L}$

代入数值：$\omega = \sqrt{\frac{3 \times 9.8}{1.2}} = \sqrt{24.5} \approx 4.95\,\mathrm{rad/s}$

末端线速度 $v = \omega L = 4.95 \times 1.2 \approx 5.94\,\mathrm{m/s}$

**验证**：用转动定律和积分方法也可得相同结果，但机械能守恒只需一行方程。

**量纲检查**：$\sqrt{g/L}$ 量纲：$\sqrt{\mathrm{(m/s^2)/m}} = \sqrt{1/\mathrm{s^2}} = 1/\mathrm{s} = \mathrm{rad/s}$ ✅

### 例题 3：圆盘沿斜面纯滚动

一均匀实心圆柱体（$m = 3.0\,\mathrm{kg}$，$R = 0.1\,\mathrm{m}$，$J = \frac12 mR^2$）从倾角 $\theta = 30^\circ$ 的斜面顶端由静止开始纯滚动。斜面长度 $L = 2.0\,\mathrm{m}$。求圆柱体到达斜面底端时的质心速度。

**解**：

**建模分析**：纯滚动条件 $v_C = \omega R$，静摩擦力不做功（接触点瞬时静止），只有重力做功，机械能守恒。

总动能 = 平动动能 + 转动动能：

$E_k = \frac12 mv_C^2 + \frac12 J\omega^2 = \frac12 mv_C^2 + \frac12 \cdot \frac12 mR^2 \cdot \frac{v_C^2}{R^2} = \frac12 mv_C^2 + \frac14 mv_C^2 = \frac34 mv_C^2$

初态（顶端，静止）：

$E_{k1} = 0$，$E_{p1} = mgL\sin\theta = 0$（取顶端为势能零点）

末态（底端）：

$E_{k2} = \frac34 mv_C^2$，$E_{p2} = -mgL\sin\theta$

机械能守恒：$0 = \frac34 mv_C^2 - mgL\sin\theta$

$v_C^2 = \frac{4}{3}gL\sin\theta = \frac{4}{3} \times 9.8 \times 2.0 \times 0.5 = \frac{4}{3} \times 9.8$

$v_C^2 = 13.07$

$v_C \approx 3.61\,\mathrm{m/s}$

**对比**：若圆柱体沿光滑斜面**滑动**（无转动），$v = \sqrt{2gL\sin\theta} = \sqrt{2 \times 9.8 \times 2 \times 0.5} = \sqrt{19.6} = 4.43\,\mathrm{m/s}$。

纯滚动时速度更小（$3.61 < 4.43$），因为部分重力势能转化为了转动动能。

**结论解释**：同样的高度势能，纯滚动时分配到转动和平动两个自由度，所以平动速度小于纯滑动。这解释了为什么圆盘滚下斜坡比同样质量的滑块滑下斜坡慢。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|------|---------|
| "力矩做功就是力乘以位移" | 力矩做功 $A = \int M\,\mathrm{d}\theta$，角位移 $\mathrm{d}\theta$ 是绕轴转过的角度，不是力的作用点的线位移 |
| "纯滚动中摩擦力不做功，所以机械能守恒" | 正确——纯滚动时接触点瞬时静止，静摩擦力不做功，但前提是无其他非保守力 |
| "转动动能定理和平动动能定理是独立的" | 两者可以统一——对刚体的总动能（平动+转动）应用动能定理或机械能守恒 |
| "刚体的重力势能是所有质元势能之和，必须逐点计算" | $E_p = \sum \Delta m_i gh_i = mg h_C$，只需知道质心高度即可 |
| "恒力矩 $M$ 做功就是 $M\theta$" | 正确——仅当 $M$ 为常数时成立。若 $M$ 随角度变化，必须积分 $A = \int M\,\mathrm{d}\theta$ |

### 检查点

- [ ] 计算力矩做功时是否用对了角位移 $\theta$ 的单位（rad，不是圈数）？
- [ ] 纯滚动问题中是否同时计入了平动动能和转动动能？
- [ ] 刚体的重力势能是否按质心高度计算？
- [ ] 使用机械能守恒前是否确认了非保守力不做功（或做功为零）？
- [ ] 转动动能定理的表达式 $A = \Delta(\frac12 J\omega^2)$ 与平动动能定理 $A = \Delta(\frac12 mv^2)$ 的对应关系是否清晰？

## 练习题

### 练习1（基础巩固）

一个质量为 $m = 2.0\,\mathrm{kg}$、半径为 $R = 0.25\,\mathrm{m}$ 的均匀圆盘（$J = \frac12 mR^2$）可绕中心轴自由转动。一条轻绳绕在圆盘边缘，绳端悬挂 $M = 1.5\,\mathrm{kg}$ 的物体。物体从静止下落 $h = 0.8\,\mathrm{m}$。不计轴承摩擦，求：

(1) 物体下落 $0.8\,\mathrm{m}$ 时的速度
(2) 此时圆盘的角速度

<details>
<summary><b>参考答案</b></summary>

**建模分析**：系统由物体 $M$、圆盘 $m$ 和地球组成。只有重力做功（绳与盘之间不打滑，但绳张力是内力，做功代数和为零），系统机械能守恒。

取物体初始位置为重力势能零点。

**初状态**（静止）：

总机械能 $E_1 = 0$

**末状态**（物体下落 $h$，速度为 $v$）：

物体动能：$\frac12 Mv^2$

圆盘转动动能：$\frac12 J\omega^2 = \frac12 \cdot \frac12 mR^2 \cdot \frac{v^2}{R^2} = \frac14 mv^2$

重力势能：$-Mgh$

机械能守恒：$0 = \frac12 Mv^2 + \frac14 mv^2 - Mgh$

$Mgh = \frac12 Mv^2 + \frac14 mv^2$

$Mgh = v^2\left(\frac{M}{2} + \frac{m}{4}\right)$

$v^2 = \frac{Mgh}{\frac{M}{2} + \frac{m}{4}} = \frac{1.5 \times 9.8 \times 0.8}{\frac{1.5}{2} + \frac{2.0}{4}} = \frac{11.76}{0.75 + 0.5} = \frac{11.76}{1.25}$

$v^2 = 9.408$

$v \approx 3.07\,\mathrm{m/s}$

**(2)** $\omega = \frac{v}{R} = \frac{3.07}{0.25} \approx 12.28\,\mathrm{rad/s}$

**验证**：用转动定律+牛顿定律求解——$Mg - T = Ma$，$TR = J\alpha = \frac12 mR^2 \cdot a/R$，可得 $a = \frac{Mg}{M + m/2} = \frac{1.5 \times 9.8}{1.5 + 1.0} = 5.88\,\mathrm{m/s^2}$。由 $v^2 = 2ah = 2 \times 5.88 \times 0.8 = 9.408$，$v = 3.07\,\mathrm{m/s}$，结果一致。✅

### 练习2（迁移应用）

一个实心球（$m = 0.5\,\mathrm{kg}$、$R = 0.05\,\mathrm{m}$、$J = \frac25 mR^2$）和一个圆环（$m = 0.5\,\mathrm{kg}$、$R = 0.05\,\mathrm{m}$、$J = mR^2$）从同一斜面顶端同时由静止开始纯滚下。斜面倾角 $\theta = 30^\circ$，斜面长度 $L = 1.5\,\mathrm{m}$。

(1) 哪个先到达底端？为什么？
(2) 分别求两者到达底端时的质心速度

<details>
<summary><b>参考答案</b></summary>

**建模分析**：两者质量相同、半径相同，但转动惯量不同。纯滚动时机械能守恒。

设刚体对质心的转动惯量为 $J_C = \beta mR^2$（$\beta$ 为形状因子：实心球 $\beta = 2/5$，圆环 $\beta = 1$）。

纯滚动总动能：$E_k = \frac12 mv^2 + \frac12 (\beta mR^2) \cdot \frac{v^2}{R^2} = \frac12 (1 + \beta) mv^2$

机械能守恒：$mgL\sin\theta = \frac12 (1+\beta) mv^2$

底端速度：$v = \sqrt{\frac{2gL\sin\theta}{1+\beta}}$

**(1)** $\beta$ 越小则 $v$ 越大，到达时间越短。实心球 $\beta = 0.4$，圆环 $\beta = 1.0$，所以**实心球先到达**。

物理直觉：圆环的质量分布在边缘，转动惯量大，相同速度下转动动能占比更大，因此平动速度更小。

**(2)** 实心球：

$v_{\text{球}} = \sqrt{\frac{2 \times 9.8 \times 1.5 \times 0.5}{1 + 0.4}} = \sqrt{\frac{14.7}{1.4}} = \sqrt{10.5} \approx 3.24\,\mathrm{m/s}$

圆环：

$v_{\text{环}} = \sqrt{\frac{2 \times 9.8 \times 1.5 \times 0.5}{1 + 1.0}} = \sqrt{\frac{14.7}{2}} = \sqrt{7.35} \approx 2.71\,\mathrm{m/s}$

**结论解释**：两者从同一高度释放，重力势能相同。圆环的 $\beta$ 更大，更多的能量分配给了转动动能，所以平动速度更小。这一结论在设计竞速类运动装备（如滑板轮子、自行车轮）时有实际意义——**转动惯量越小，滚动中的平动速度损失越小**。

</details>
