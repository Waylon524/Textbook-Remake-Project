# 20. 功与功率

> 所属切块：01-力学  |  块内序号：20  |  难度：基础
> 常见混淆点：变力做功的积分表达式常出错

## 1. 学习目标与先修前置

### 学习目标
- 理解功是力在位移上的积累效应
- 掌握恒力做功 $A = \mathbf{F}\cdot\Delta\mathbf{r}$ 和变力做功 $A = \int\mathbf{F}\cdot\mathrm{d}\mathbf{r}$ 的计算
- 理解功率的物理意义及 $P = \mathbf{F}\cdot\mathbf{v}$ 的应用
- 能正确判断功的正负——力与位移夹角决定

### 先修知识
- 力的矢量点积
- 定积分的基本概念
- 匀速运动和匀加速运动公式

## 2. 背景与应用场景

汽车发动机的功率参数是衡量汽车性能的核心指标。$P = Fv$ 揭示了为什么汽车上坡时需要降档：功率一定时，要增大牵引力 $F$ 就必须降低速度 $v$。这就是为什么卡车爬陡坡时只能用很慢的速度。

> **直觉理解**：推着一个箱子走 $10$ 米——用力越大、走得越远，做的功越多。但如果推着一堵墙（纹丝不动），就算使再大的力，功也是零（位移为零）。功的本质是"力在位移方向上的累积"。

不同情境下做功的差异：
- 水平推物体：推力做功等于推力乘以水平位移
- 竖直提物体：拉力做功等于拉力乘以上升高度
- 圆周运动中向心力不做功：因为力始终与速度方向垂直

## 3. 核心概念与符号约定

### 恒力做功

力 $\mathbf{F}$ 为恒力，物体发生位移 $\Delta\mathbf{r}$：

$$
A = \mathbf{F}\cdot\Delta\mathbf{r} = F\Delta r\cos\theta
$$

$\theta$ 为 $\mathbf{F}$ 与 $\Delta\mathbf{r}$ 的夹角。

| $\theta$ 范围 | 功的正负 | 物理含义 |
|-------------|---------|---------|
| $0^\circ \le \theta < 90^\circ$ | $A > 0$ | 动力做功 |
| $\theta = 90^\circ$ | $A = 0$ | 力的方向与位移垂直，不做功 |
| $90^\circ < \theta \le 180^\circ$ | $A < 0$ | 阻力做功 |

### 变力做功

力随位置变化时，沿路径积分：

$$
A = \int_{a}^{b} \mathbf{F}\cdot\mathrm{d}\mathbf{r} = \int_{a}^{b} F\cos\theta\,\mathrm{d}s
$$

在直角坐标系中：

$$
A = \int_{x_1}^{x_2} F_x\,\mathrm{d}x + \int_{y_1}^{y_2} F_y\,\mathrm{d}y + \int_{z_1}^{z_2} F_z\,\mathrm{d}z
$$

### 功率

平均功率 $\bar{P} = \frac{\Delta A}{\Delta t}$，瞬时功率 $P = \frac{\mathrm{d}A}{\mathrm{d}t} = \mathbf{F}\cdot\mathbf{v}$

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $A$ | 功 | $\mathrm{J}$（焦耳） |
| $P$ | 功率 | $\mathrm{W}$（瓦特） |
| $\mathbf{F}$ | 力 | $\mathrm{N}$ |
| $\theta$ | 力与位移夹角 | $\mathrm{rad}$ |

$1\,\mathrm{J} = 1\,\mathrm{N\cdot m}$，$1\,\mathrm{W} = 1\,\mathrm{J/s}$

## 4. 原理与方法

### 常见力做功的特点

| 力 | 做功特点 | 说明 |
|----|---------|------|
| 重力 | 做功与路径无关，只与高度差有关 | $A_G = \pm mgh$ |
| 弹力 | 做功与形变量有关 | $A_{\text{弹}} = -\frac12 k(x_2^2 - x_1^2)$ |
| 摩擦力 | 与路径有关，总是做负功 | $A_f = -f\cdot s$（$s$ 为路径长度）|
| 向心力 | 始终不做功 | $A=0$，$F\perp v$ |
| 洛伦兹力 | 始终不做功 | $A=0$，$F\perp v$ |

### 多力做功的处理

**方法一**：先求合外力，再求合外力的功：$A = \mathbf{F}_{\text{合}}\cdot\Delta\mathbf{r}$

**方法二**：分别求每个力的功，再求和：$A = A_1 + A_2 + \cdots + A_n$

## 5. 例题

### 例题 1：恒力做功的计算

$m = 10\,\mathrm{kg}$ 的物体在水平地面上受到与水平方向成 $30^\circ$ 角斜向上的拉力 $F = 50\,\mathrm{N}$ 作用，在水平面上移动 $s = 5\,\mathrm{m}$。物体与地面的摩擦系数 $\mu_k = 0.2$。求：

(1) 拉力做的功
(2) 摩擦力做的功
(3) 重力和支持力做的功
(4) 合外力做的总功

**解**：

**(1)** 拉力做功：$A_F = Fs\cos30^\circ = 50\times5\times0.866 = 216.5\,\mathrm{J}$

**(2)** 正压力 $N = mg - F\sin30^\circ = 10\times9.8 - 50\times0.5 = 98 - 25 = 73\,\mathrm{N}$

摩擦力 $f = \mu_k N = 0.2\times73 = 14.6\,\mathrm{N}$

$A_f = -fs = -14.6\times5 = -73\,\mathrm{J}$

**(3)** 重力竖直向下、支持力竖直向上，位移水平，$\theta=90^\circ$，两者做功均为零。

**(4)** 总功 $A = A_F + A_f = 216.5 - 73 = 143.5\,\mathrm{J}$

验证：$A = F_{\text{合}}s$，$F_{\text{合}} = F\cos30^\circ - f = 50\times0.866 - 14.6 = 43.3 - 14.6 = 28.7\,\mathrm{N}$

$A = 28.7\times5 = 143.5\,\mathrm{J}$ ✅

### 例题 2：变力做功（弹簧）

劲度系数 $k = 200\,\mathrm{N/m}$ 的弹簧，从原长拉伸到 $x = 0.1\,\mathrm{m}$。求弹力做的功。

**解**：

弹力 $F = -kx$，方向与位移方向相反。

弹力做功 $A = \int_{0}^{0.1} (-kx)\,\mathrm{d}x = -\frac12 kx^2\Big|_{0}^{0.1} = -\frac12\times200\times0.01 = -1\,\mathrm{J}$

负号表示弹力做负功（外力拉伸弹簧需要克服弹力做功）。弹性势能增加 $1\,\mathrm{J}$。

### 例题 3：功率计算

一辆质量为 $m = 1500\,\mathrm{kg}$ 的汽车，发动机额定功率 $P = 60\,\mathrm{kW}$。在水平路面上以 $v = 72\,\mathrm{km/h}$ 匀速行驶。

(1) 求此时汽车的牵引力
(2) 如果汽车上坡，需要牵引力增大到 $5000\,\mathrm{N}$，要保持相同速度，发动机功率应为多少？若功率保持 $60\,\mathrm{kW}$ 不变，最大速度是多少？

**解**：

$v = 72\times\frac{1000}{3600} = 20\,\mathrm{m/s}$

**(1)** 由 $P = Fv$：$F = \frac{P}{v} = \frac{60000}{20} = 3000\,\mathrm{N}$

匀速行驶时牵引力等于阻力，所以阻力 $f = 3000\,\mathrm{N}$。

**(2)** 若需要 $F=5000\,\mathrm{N}$ 且 $v=20\,\mathrm{m/s}$：$P = 5000\times20 = 100\,\mathrm{kW}$——超过额定功率。

若功率保持 $60\,\mathrm{kW}$ 不变：$v = \frac{P}{F} = \frac{60000}{5000} = 12\,\mathrm{m/s} = 43.2\,\mathrm{km/h}$

这就解释了为什么汽车上坡时必须降速——功率一定时，更大的牵引力只能用更小的速度换取。

## 6. 常见误区与检查点

| 误区 | 正确理解 |
|------|---------|
| "做功等于力乘以路程" | 功是力与**位移**的点积，不是力乘以路程 |
| "正功增大能量，负功减小能量" | 不一定，需看是什么力做功以及分析什么系统 |
| "摩擦力总做负功" | 静摩擦力可以不做功或做正功（如传送带上的物体） |
| "$F$ 大则功率一定大" | 功率 $P=Fv$，还与速度有关 |

## 练习题

### 练习1（基础巩固）

$F=10\,\mathrm{N}$ 的水平力拉物体在粗糙水平面上匀速移动 $5\,\mathrm{m}$。求拉力和摩擦力做的总功。

<details>
<summary><b>参考答案</b></summary>

匀速运动，合力为零，摩擦力 $f = F = 10\,\mathrm{N}$。

拉力做功 $A_F = Fs = 10\times5 = 50\,\mathrm{J}$（正功）

摩擦力做功 $A_f = -fs = -10\times5 = -50\,\mathrm{J}$（负功）

总功 $A = 50 - 50 = 0\,\mathrm{J}$

匀速运动动能不变，合外力做功为零——与动能定理一致。

</details>

### 练习2（迁移应用）

汽车质量 $m=1500\,\mathrm{kg}$，以 $v=54\,\mathrm{km/h}$ 匀速行驶时发动机功率 $P=30\,\mathrm{kW}$。

(1) 求阻力大小
(2) 若汽车以 $0.5\,\mathrm{m/s^2}$ 加速前进，此时发动机需输出多大功率（阻力不变）？

<details>
<summary><b>参考答案</b></summary>

**(1)** $v = 54\times\frac{1000}{3600} = 15\,\mathrm{m/s}$

$P = Fv = fv$（匀速时牵引力等于阻力）

$f = \frac{P}{v} = \frac{30000}{15} = 2000\,\mathrm{N}$

**(2)** 加速时牵引力 $F = f + ma = 2000 + 1500\times0.5 = 2000 + 750 = 2750\,\mathrm{N}$

功率 $P = Fv = 2750\times15 = 41250\,\mathrm{W} = 41.25\,\mathrm{kW}$

加速时需更大功率，这就是为什么超车时需要深踩油门。

</details>
