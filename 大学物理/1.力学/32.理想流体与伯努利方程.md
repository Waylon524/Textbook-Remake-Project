# 32. 理想流体与伯努利方程

> 所属切块：01-力学  |  块内序号：32  |  难度：进阶
> 常见混淆点：伯努利方程的适用条件（理想流体、定常流动）；流速大处压强小这一结论的滥用

### 学习目标
- 理解理想流体模型的两个基本假设及其适用场景
- 掌握连续性方程 $S_1v_1 = S_2v_2$（质量守恒在流体中的体现）
- 理解伯努利方程 $p + \frac12\rho v^2 + \rho gh = \text{常量}$ 是机械能守恒在流体中的体现
- 能用伯努利方程解释日常现象（机翼升力、喷雾器、水流速度与压强关系）

### 先修知识
- 机械能守恒定律（文件 23）
- 基本微积分概念

## 2. 背景与应用场景

飞机机翼的上表面曲率大于下表面，空气流经上表面的速度更快、压强更小，上下表面的压强差产生向上的升力——这正是伯努利方程的直接应用。同样的原理还解释了两艘平行行驶的船为什么会相互靠近（中间水流快、压强小），以及为什么在火车站台需要设置安全线（列车通过时带动周围空气快速流动，靠近列车处压强降低）。

> **直觉理解**：在一条河里，河道窄的地方水流快，河道宽的地方水流慢——这是连续性方程告诉我们的。而在水平管道中，水流快的地方压强小——这是伯努利方程的核心结论。两者结合，可以解释大量流体现象。

伯努利方程本质上是**单位体积流体的机械能守恒**——在理想流体定常流动中，压强能、动能和重力势能之和沿流线为常数。

---

## 3. 核心概念与符号约定

### 理想流体模型

| 假设 | 含义 | 何时近似满足 |
|------|------|-------------|
| 不可压缩 | $\rho = \text{常量}$，密度不随压强变化 | 液体、低速气体（$v < 0.3Ma$） |
| 无黏性 | 内部无摩擦，无黏滞力 | 稀薄流体、远离边界处 |

### 定常流动 (steady flow)

流场中各点的流速不随时间变化——$\partial\mathbf{v}/\partial t = 0$。注意定常不意味着各点速度相同，只是各点的速度不随时间改变。

流线：在流场中画出的曲线，其上每一点的切线方向与该点的流速方向一致。定常流动中流线与迹线重合。

### 连续性方程 (continuity equation)

对于不可压缩流体的定常流动，流经管道任意横截面的体积流量（或质量流量）相等：

$$
S_1v_1 = S_2v_2 \quad \text{或} \quad Sv = \text{常量}
$$

其中 $S$ 为横截面积，$v$ 为平均流速。截面越窄，流速越大。

### 伯努利方程 (Bernoulli's equation)

在理想流体的定常流动中，沿同一流线有：

$$
p + \frac12\rho v^2 + \rho gh = \text{常量}
$$

三项的物理意义（单位体积的能量）：

| 项 | 名称 | 物理含义 |
|---|------|---------|
| $p$ | 压强能（静压） | 单位体积流体因压强而具有的做功能力 |
| $\frac12\rho v^2$ | 动能密度（动压） | 单位体积流体的动能 |
| $\rho gh$ | 势能密度 | 单位体积流体的重力势能 |

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $p$ | 压强 | $\mathrm{Pa}$（$\mathrm{N/m^2}$） |
| $\rho$ | 流体密度 | $\mathrm{kg/m^3}$ |
| $v$ | 流速 | $\mathrm{m/s}$ |
| $g$ | 重力加速度 | $\mathrm{m/s^2}$ |
| $h$ | 高度 | $\mathrm{m}$ |
| $S$ | 横截面积 | $\mathrm{m^2}$ |
| $Q = Sv$ | 体积流量 | $\mathrm{m^3/s}$ |

---

## 4. 原理与方法

### 连续性方程的推导

取一段流管，在 $\Delta t$ 时间内，从截面 $S_1$ 流入的流体体积为 $S_1v_1\Delta t$，从截面 $S_2$ 流出的流体体积为 $S_2v_2\Delta t$。不可压缩流体要求流入=流出，得 $S_1v_1 = S_2v_2$。

### 伯努利方程的推导

沿流线取一小段流体，分析其在流动过程中机械能的变化。对单位体积流体：

- 外力（压强差）做功 = 动能增量 + 势能增量
- 推导得：$p_1 + \frac12\rho v_1^2 + \rho gh_1 = p_2 + \frac12\rho v_2^2 + \rho gh_2$

### 解题步骤

1. **确认条件**：流体是否可视为理想流体？流动是否定常？
2. **选择流线**：沿同一流线应用伯努利方程（不同流线可能常数不同）
3. **应用连续性方程**：如果涉及截面变化，用 $S_1v_1 = S_2v_2$ 求速度关系
4. **列伯努利方程**：代入已知量，求解未知量
5. **结果检验**：量纲检查、极限情况验证

### 伯努利方程的常见形式

| 情形 | 简化形式 | 说明 |
|------|---------|------|
| 水平管（$h_1 = h_2$） | $p_1 + \frac12\rho v_1^2 = p_2 + \frac12\rho v_2^2$ | 流速大处压强小 |
| 静止流体（$v=0$） | $p + \rho gh = \text{常量}$ | 静力学压强公式 |
| 开口容器小孔出流 | $\frac12\rho v^2 = \rho g\Delta h$ | 托里拆利定律 |

## 5. 例题

### 例题 1：水平管中的流速与压强

水在水平管道中流动。截面 $S_1 = 0.03\,\mathrm{m^2}$ 处的流速 $v_1 = 1.5\,\mathrm{m/s}$，压强 $p_1 = 2.5 \times 10^5\,\mathrm{Pa}$。管道变窄处截面 $S_2 = 0.015\,\mathrm{m^2}$。水的密度 $\rho = 1000\,\mathrm{kg/m^3}$。求 $S_2$ 处的流速和压强。

**解**：

**建模分析**：水可视为不可压缩理想流体，水平管 $h_1 = h_2$。

**列式**：

连续性方程：$S_1v_1 = S_2v_2$

伯努利方程：$p_1 + \frac12\rho v_1^2 = p_2 + \frac12\rho v_2^2$

**计算**：

$v_2 = \frac{S_1}{S_2}v_1 = \frac{0.03}{0.015} \times 1.5 = 3.0\,\mathrm{m/s}$

$p_2 = p_1 + \frac12\rho(v_1^2 - v_2^2) = 2.5 \times 10^5 + \frac12 \times 1000 \times (2.25 - 9.0)$

$p_2 = 2.5 \times 10^5 + 500 \times (-6.75) = 2.5 \times 10^5 - 3375$

$p_2 = 2.466 \times 10^5\,\mathrm{Pa}$

**结论**：截面减半，流速翻倍（从 $1.5$ 到 $3.0\,\mathrm{m/s}$），压强下降 $3375\,\mathrm{Pa}$。

**量纲检查**：$\frac12\rho v^2$ 量纲：$\mathrm{kg/m^3 \cdot (m/s)^2 = kg \cdot m^{-1} \cdot s^{-2} = N/m^2 = Pa}$，与压强单位一致。✅

### 例题 2：托里拆利定律——容器小孔出流

一个大水箱中水面高度为 $H = 2.0\,\mathrm{m}$，箱底有一个小孔。求水从小孔流出的速度。

**解**：

**建模分析**：取水面上一点 1 和小孔处一点 2，沿流线应用伯努利方程。水面面积远大于小孔面积，水面下降速度可忽略（$v_1 \approx 0$）。水面和小孔处均为大气压 $p_0$。

**列式**：

$p_1 + \frac12\rho v_1^2 + \rho gh_1 = p_2 + \frac12\rho v_2^2 + \rho gh_2$

$p_0 + 0 + \rho gH = p_0 + \frac12\rho v_2^2 + 0$

$\rho gH = \frac12\rho v_2^2$

**计算**：

$v_2 = \sqrt{2gH} = \sqrt{2 \times 9.8 \times 2.0} = \sqrt{39.2} \approx 6.26\,\mathrm{m/s}$

**结论解释**：小孔出流速度等于质点从水面高度自由下落到小孔高度的末速度——这就是托里拆利定律。这个结果说明流体的重力势能完全转化为动能。

**注意**：实际出流速度略小于理论值（因黏性和孔口收缩效应），工程中常引入流速系数 $C_v \approx 0.97$。

### 例题 3：文丘里流量计

文丘里流量计由一段渐缩管和渐扩管组成。入口截面直径 $d_1 = 0.1\,\mathrm{m}$，最窄处（喉部）直径 $d_2 = 0.05\,\mathrm{m}$。U 形管中水银（$\rho_{\text{Hg}} = 13600\,\mathrm{kg/m^3}$）的液面高度差 $\Delta h = 0.12\,\mathrm{m}$。求水的体积流量。

**解**：

**建模分析**：水平文丘里管，$h_1 = h_2$。由入口和喉部的压强差（通过 U 形管测量）结合连续性方程和伯努利方程求流速。

入口截面积 $S_1 = \pi(d_1/2)^2 = \pi \times 0.05^2 = 7.85 \times 10^{-3}\,\mathrm{m^2}$

喉部截面积 $S_2 = \pi(d_2/2)^2 = \pi \times 0.025^2 = 1.96 \times 10^{-3}\,\mathrm{m^2}$

由 U 形管压强差：$p_1 - p_2 = (\rho_{\text{Hg}} - \rho_{\text{水}})g\Delta h$

$p_1 - p_2 = (13600 - 1000) \times 9.8 \times 0.12 = 12600 \times 9.8 \times 0.12 \approx 14818\,\mathrm{Pa}$

连续性方程：$S_1v_1 = S_2v_2$，$v_2 = \frac{S_1}{S_2}v_1 = \frac{7.85}{1.96}v_1 \approx 4.0v_1$

伯努利方程：$p_1 + \frac12\rho v_1^2 = p_2 + \frac12\rho v_2^2$

$p_1 - p_2 = \frac12\rho(v_2^2 - v_1^2) = \frac12\rho(16v_1^2 - v_1^2) = \frac{15}{2}\rho v_1^2$

$v_1 = \sqrt{\frac{2(p_1-p_2)}{15\rho}} = \sqrt{\frac{2 \times 14818}{15 \times 1000}} = \sqrt{\frac{29636}{15000}} = \sqrt{1.976} \approx 1.41\,\mathrm{m/s}$

体积流量：$Q = S_1v_1 = 7.85 \times 10^{-3} \times 1.41 \approx 1.11 \times 10^{-2}\,\mathrm{m^3/s} = 11.1\,\mathrm{L/s}$

**结论**：水的体积流量约为 $11.1$ 升/秒。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|------|---------|
| "伯努利方程对所有流体都适用" | 仅适用于**理想流体**（不可压缩、无黏性）的**定常流动**。粘性流体会因内部摩擦而耗散机械能 |
| "流速大则压强一定小" | 伯努利方程中 $p + \frac12\rho v^2 + \rho gh = \text{常量}$，流速大则压强小**仅当高度不变时成立**。若高度也变化，需全面考虑 |
| "流速快的地方一定压强小（用于解释所有升力现象）" | 机翼升力的完整解释需要考虑环流和库塔条件，伯努利方程只是部分解释 |
| "连续性方程对所有流体都成立" | 仅对**不可压缩流体**成立。可压缩气体（高速时）需考虑密度变化 |
| "伯努利方程中常量对所有流线相同" | 仅在无旋流动中不同流线的常量相同。一般有旋流动中，不同流线的常量不同 |

### 检查点

- [ ] 应用伯努利方程前是否确认了理想流体和定常流动条件？
- [ ] 连续性方程和伯努利方程是否对同一流管（同一流线）应用？
- [ ] 截面变化时是否先用连续性方程求出速度关系？
- [ ] 所有项的单位是否一致（均为 $\mathrm{Pa}$ 或均为 $\mathrm{J/m^3}$）？
- [ ] 涉及高度变化时，是否注意了 $\rho gh$ 项的符号？（高度增加，此项增大）

## 练习题

### 练习1（基础巩固）

水在水平管中流动，截面 $S_1 = 0.04\,\mathrm{m^2}$ 处流速 $v_1 = 1.0\,\mathrm{m/s}$，压强 $p_1 = 2.0 \times 10^5\,\mathrm{Pa}$。截面 $S_2 = 0.01\,\mathrm{m^2}$ 处压强 $p_2$ 为多少？水的密度 $\rho = 1000\,\mathrm{kg/m^3}$。

<details>
<summary><b>参考答案</b></summary>

**建模分析**：水平管，$h_1 = h_2$，应用连续性方程 + 伯努利方程。

连续性方程：$S_1v_1 = S_2v_2$

$v_2 = \frac{S_1}{S_2}v_1 = \frac{0.04}{0.01} \times 1.0 = 4.0\,\mathrm{m/s}$

伯努利方程：$p_1 + \frac12\rho v_1^2 = p_2 + \frac12\rho v_2^2$

$p_2 = p_1 + \frac12\rho(v_1^2 - v_2^2) = 2.0 \times 10^5 + \frac12 \times 1000 \times (1 - 16)$

$p_2 = 2.0 \times 10^5 + 500 \times (-15) = 2.0 \times 10^5 - 7500 = 1.925 \times 10^5\,\mathrm{Pa}$

**结论**：截面缩小到 $1/4$，流速增大到 4 倍，压强降低 $7500\,\mathrm{Pa}$。

</details>

### 练习2（迁移应用）

一栋大楼的水箱位于地面以上 $H = 30\,\mathrm{m}$ 处。一根水管垂直向下连接到一楼的水龙头，水龙头距地面 $h = 1.0\,\mathrm{m}$。水龙头全开时，出口直径 $d = 1.5\,\mathrm{cm}$。

(1) 忽略黏性，求水龙头出口处的流速
(2) 求体积流量（单位：$\mathrm{L/min}$）

<details>
<summary><b>参考答案</b></summary>

**建模分析**：选水箱水面为点 1，水龙头出口为点 2。取水龙头所在高度为 $h = 0$ 零点，则水箱水面高度 $h_1 = H - h = 29\,\mathrm{m}$。水面处 $v_1 \approx 0$（水箱截面远大于出口截面），水面和出口均通大气 $p_1 = p_2 = p_0$。

**(1)** 伯努利方程：

$p_0 + 0 + \rho g(H-h) = p_0 + \frac12\rho v_2^2 + 0$

$v_2 = \sqrt{2g(H-h)} = \sqrt{2 \times 9.8 \times 29} = \sqrt{568.4} \approx 23.84\,\mathrm{m/s}$

**(2)** 出口截面积：$S = \pi(d/2)^2 = \pi \times (0.0075)^2 = \pi \times 5.625 \times 10^{-5} \approx 1.767 \times 10^{-4}\,\mathrm{m^2}$

体积流量：$Q = Sv = 1.767 \times 10^{-4} \times 23.84 \approx 4.21 \times 10^{-3}\,\mathrm{m^3/s}$

换算为 $\mathrm{L/min}$：$Q = 4.21 \times 10^{-3} \times 1000 \times 60 \approx 252.6\,\mathrm{L/min}$

**注意**：实际流量会小于此理论值，因为：
1. 水管中的摩擦（黏性）会消耗机械能
2. 水龙头处的收缩效应（流量系数 $< 1$）
3. 工程估算是通常会取 $C_d \approx 0.6$-$0.8$ 的流量系数

**量纲检查**：$\sqrt{g(H-h)}$ 量纲 $\sqrt{\mathrm{m/s^2 \cdot m}} = \mathrm{m/s}$ ✅

</details>
