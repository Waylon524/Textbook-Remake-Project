## 2. 背景与应用场景

F1 赛车的轮胎与赛道之间的摩擦力决定了赛车的极限性能。轮胎经过预热后，橡胶变软，与沥青的静摩擦系数 $\mu_s$ 可达 $1.5$ 以上。这意味着赛车可以在水平弯道中以 $1.5g$（约 $14.7\,\mathrm{m/s^2}$）的横向加速度过弯。相比之下，普通家用车的轮胎 $\mu_s \approx 0.8$，这就是为什么 F1 赛车可以在不减速的情况下通过家用车必须大幅减速的弯道。

> **直觉理解**：推一个沉重的木箱——一开始用力但箱子不动（静摩擦力与推力平衡），继续加力，突然动了（推力超过最大静摩擦力），推起来后感觉轻松一些（滑动摩擦力小于最大静摩擦力）。这就是 $\mu_s$ 总是大于 $\mu_k$ 的生活直觉。

弹力的生活例子：弹簧秤称重时，弹簧被拉伸产生弹力平衡重力。如果超过弹簧的弹性限度，弹簧发生永久形变，弹力不再与形变成正比。

## 3. 核心概念与符号约定

### 重力

$$
G = mg
$$

重力加速度 $g$ 在地球表面变化：赤道 $9.78$、两极 $9.83$、标准值 $9.80665\,\mathrm{m/s^2}$。方向竖直向下。

**注意**：重力是地球对物体的万有引力的一个分力（另一分力提供物体随地球自转所需的向心力）。在大多数力学问题中，$g$ 取 $9.8\,\mathrm{m/s^2}$ 足够精确。

### 弹力

发生弹性形变的物体要恢复原状而对与之接触的物体产生的力。

**胡克定律**（弹簧的弹力）：

$$
F = -kx
$$

$k$ 为劲度系数（$\mathrm{N/m}$），$x$ 为形变量（伸长或压缩），负号表示弹力方向与形变方向相反。

**绳的张力**：
- 理想轻绳（质量不计、不可伸长）：绳上各点张力相等，张力沿绳方向
- 实际有质量的绳：悬挂重物时绳上端张力大于下端（需支撑绳本身重量）
- 跨过定滑轮的绳：不计滑轮质量和摩擦时，两侧张力相等

**支持力**：
- 方向：垂直于接触面指向被支持物体
- 大小：由平衡条件或动力学方程决定——不一定等于重力

### 摩擦力

**静摩擦力**：

$$
0 \le f_s \le f_{\max} = \mu_s N
$$

- 方向：与物体**相对接触面**的运动趋势方向相反
- 大小：由平衡条件决定（$0$ 到 $\mu_s N$ 之间）

**滑动摩擦力**：

$$
f_k = \mu_k N
$$

- 方向：与物体**相对接触面**的运动方向相反
- 大小：$f_k = \mu_k N$

**关键数值关系**：$\mu_s > \mu_k$。常见材料摩擦系数：

| 材料 | $\mu_s$ | $\mu_k$ |
|------|---------|---------|
| 钢-钢 | $0.74$ | $0.57$ |
| 轮胎-干燥沥青 | $1.0$ | $0.8$ |
| 轮胎-湿沥青 | $0.7$ | $0.5$ |
| 冰-冰 | $0.1$ | $0.03$ |
| 木材-木材 | $0.5$ | $0.3$ |

### 万有引力

任何两个有质量的物体之间都存在相互吸引力：

$$
F = G\frac{m_1 m_2}{r^2},\qquad
G = 6.67\times10^{-11}\,\mathrm{N\cdot m^2/kg^2}
$$

- **适用条件**：两个物体可视为质点（或球对称分布）
- **方向**：沿两质点的连线

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $G$ | 重力 / 万有引力常量 | $\mathrm{N}$ / $\mathrm{N\cdot m^2/kg^2}$ |
| $g$ | 重力加速度 | $\mathrm{m/s^2}$ |
| $k$ | 劲度系数 | $\mathrm{N/m}$ |
| $x$ | 形变量 | $\mathrm{m}$ |
| $\mu_s,\mu_k$ | 静/滑动摩擦系数 | 无量纲 |
| $N$ | 支持力（正压力） | $\mathrm{N}$ |

## 4. 原理与方法

### 摩擦力方向的精讲——最容易出错的地方

核心原则：**摩擦力以"相对接触面的运动（趋势）"为准，不以"相对地面"为准**。

**例1：传送带上的物体**——物体轻轻放在向右运动的传送带上，物体相对地面静止，但相对传送带向左滑动——传送带给物体的滑动摩擦力方向**向右**。正是这个摩擦力使物体加速到与传送带同速。

**例2：人走路**——脚向后蹬地，脚相对地面有向后运动的趋势——地面对脚的静摩擦力方向**向前**。人前进的动力来自地面向前的静摩擦力。如果没有摩擦力（冰面），人无法行走。

**例3：汽车转弯**——汽车在水平路面上右转弯，有沿半径向外侧滑的趋势——路面对轮胎的静摩擦力指向弯道**内侧**（圆心），提供向心力。湿滑路面上摩擦力减小，车速必须降低以防侧滑。

### 受力分析的标准流程

1. **确定研究对象**：明确分析哪个物体
2. **画重力**：方向竖直向下，标出大小 $mg$
3. **找接触面**：每一个接触面可能同时存在支持力（弹力）和摩擦力
4. **画其他外力**：拉力、推力、空气阻力等
5. **建立坐标系**：一般将多数力投影到坐标轴上，沿运动方向和垂直运动方向分解
6. **列方程**：$\sum F_x = ma_x$，$\sum F_y = ma_y$

## 5. 例题

### 例题 1：斜面上的摩擦力分析

质量为 $m = 5\,\mathrm{kg}$ 的物体放在倾角 $\theta = 30^\circ$ 的斜面上。物体与斜面之间的 $\mu_s = 0.4$，$\mu_k = 0.3$。

(1) 物体能否在斜面上保持静止？
(2) 如果不能静止，下滑加速度为多大？
(3) 斜面倾角增加到多大时物体刚好开始下滑？

**解**：

**建模**：物体受三个力——重力 $mg$、支持力 $N$、摩擦力 $f$。取沿斜面向下为 $x$ 方向，垂直斜面向上为 $y$ 方向。

**(1)** 判断是否静止：

重力沿斜面分量（使物体下滑的力）：

$mg\sin\theta = 5\times9.8\times\sin30^\circ = 49\times0.5 = 24.5\,\mathrm{N}$

最大静摩擦力（阻止下滑的最大能力）：

$f_{\max} = \mu_s N = \mu_s mg\cos\theta = 0.4\times5\times9.8\times\cos30^\circ = 0.4\times49\times0.866 = 16.97\,\mathrm{N}$

$24.5\,\mathrm{N} > 16.97\,\mathrm{N}$——重力沿斜面分量超过了最大静摩擦力，**物体不能静止，会下滑**。

**(2)** 下滑时滑动摩擦力替代静摩擦力：

$f_k = \mu_k N = 0.3\times5\times9.8\times0.866 = 12.73\,\mathrm{N}$

沿斜面方向应用牛顿第二定律：

$mg\sin\theta - f_k = ma$

$a = g\sin\theta - \mu_k g\cos\theta = 9.8\times0.5 - 0.3\times9.8\times0.866$

$= 4.9 - 2.55 = 2.35\,\mathrm{m/s^2}$

**(3)** 刚好开始下滑时，重力沿斜面分量等于最大静摩擦力：

$mg\sin\theta_c = \mu_s mg\cos\theta_c$

$\tan\theta_c = \mu_s = 0.4$

$\theta_c = \arctan(0.4) \approx 21.8^\circ$

**结论**：倾角超过 $21.8^\circ$ 时物体开始下滑。$30^\circ > 21.8^\circ$，所以物体必然下滑。

### 例题 2：传送带上的滑动摩擦

水平传送带以 $v = 2\,\mathrm{m/s}$ 匀速运行。一个 $m = 10\,\mathrm{kg}$ 的箱子轻放在传送带一端。$\mu_k = 0.4$。

(1) 箱子刚放上时受到的摩擦力大小和方向
(2) 箱子加速到与传送带同速需要多长时间？
(3) 在此过程中，箱子相对传送带滑行了多远？

**解**：

**(1)** 箱子刚放上时速度为 $0$，相对传送带向左滑动，所以滑动摩擦力方向向右。箱子在水平方向只受摩擦力：

$f_k = \mu_k mg = 0.4\times10\times9.8 = 39.2\,\mathrm{N}$

加速度 $a = f_k/m = 39.2/10 = 3.92\,\mathrm{m/s^2}$

**(2)** 从 $0$ 加速到 $v=2\,\mathrm{m/s}$：

$t = \frac{v}{a} = \frac{2}{3.92} = 0.51\,\mathrm{s}$

**(3)** 箱子对地位移：$x_{\text{箱}} = \frac12 at^2 = \frac12\times3.92\times(0.51)^2 = 0.5\times3.92\times0.26 = 0.51\,\mathrm{m}$

传送带对地位移：$x_{\text{带}} = vt = 2\times0.51 = 1.02\,\mathrm{m}$

相对滑动距离：$\Delta x = x_{\text{带}} - x_{\text{箱}} = 1.02 - 0.51 = 0.51\,\mathrm{m}$

这段时间内摩擦力做功 $A = f_k \cdot \Delta x = 39.2\times0.51 = 20.0\,\mathrm{J}$，全部转化为内能。

## 6. 常见误区与检查点

| 误区 | 正确理解 |
|------|---------|
| "支持力总等于重力" | 支持力是约束力，由力学条件决定——斜面上 $N=mg\cos\theta$，超重时 $N>mg$ |
| "摩擦力一定与运动方向相反" | 摩擦力与**相对运动（趋势）**方向相反——人走路时摩擦力方向与人运动方向相同 |
| "绳中张力处处相等" | 仅对轻绳（质量不计）成立。有质量的绳悬挂重物时上端张力更大 |
| "质量越大万有引力越大" | 万有引力由 $m_1$ 和 $m_2$ 共同决定，$F \\propto m_1m_2$ |
| "静摩擦力总是 $\\mu_s N$" | $\\mu_s N$ 是最大值，实际静摩擦力在 $0$ 到最大值之间由平衡条件决定 |

### 检查点

- [ ] 能否区分静摩擦和滑动摩擦的计算公式？
- [ ] 是否理解为什么 $\mu_s > \mu_k$？
- [ ] 能否独立完成一个包含重力、支持力、摩擦力的受力分析图？

## 练习题

### 练习1（基础巩固）

弹簧原长 $l_0 = 20\,\mathrm{cm}$，劲度系数 $k = 100\,\mathrm{N/m}$。竖直悬挂 $m = 2\,\mathrm{kg}$ 的物体静止后，弹簧长度为多少？若换成 $m = 3\,\mathrm{kg}$，结果如何？如果弹簧的弹性限度为形变不超过 $30\,\mathrm{cm}$，$5\,\mathrm{kg}$ 的物体会不会损坏弹簧？

<details>
<summary><b>参考答案</b></summary>

**$m=2\,\mathrm{kg}$**：由胡克定律 $k\Delta l = mg$

$\Delta l = \frac{mg}{k} = \frac{2\times9.8}{100} = \frac{19.6}{100} = 0.196\,\mathrm{m} = 19.6\,\mathrm{cm}$

弹簧总长 $l = l_0 + \Delta l = 20 + 19.6 = 39.6\,\mathrm{cm}$

**$m=3\,\mathrm{kg}$**：$\Delta l = \frac{3\times9.8}{100} = 0.294\,\mathrm{m} = 29.4\,\mathrm{cm}$

弹簧总长 $l = 20 + 29.4 = 49.4\,\mathrm{cm}$

**$m=5\,\mathrm{kg}$**：$\Delta l = \frac{5\times9.8}{100} = 0.49\,\mathrm{m} = 49\,\mathrm{cm}$

弹性限度为 $30\,\mathrm{cm}$，而 $\Delta l = 49\,\mathrm{cm} > 30\,\mathrm{cm}$，弹簧会发生永久形变损坏。

</details>

### 练习2（迁移应用）

水平拉力 $F = 50\,\mathrm{N}$ 拉 $m = 10\,\mathrm{kg}$ 的物体在粗糙水平面上运动。$\mu_s = 0.4$，$\mu_k = 0.3$。

(1) 物体能否被拉动？
(2) 若能拉动，从静止开始 $t = 5\,\mathrm{s}$ 内移动的距离是多少？
(3) 如果将 $F$ 改为与水平方向成 $30^\circ$ 向下推，$5\,\mathrm{s}$ 内的移动距离如何变化？为什么？

<details>
<summary><b>参考答案</b></summary>

**(1)** 最大静摩擦力 $f_{\max} = \mu_s mg = 0.4\times10\times9.8 = 39.2\,\mathrm{N}$

$F = 50\,\mathrm{N} > 39.2\,\mathrm{N}$——可以被拉动。

**(2)** 物体运动后受滑动摩擦力：

$f_k = \mu_k mg = 0.3\times10\times9.8 = 29.4\,\mathrm{N}$

水平方向合力 $F - f_k = 50 - 29.4 = 20.6\,\mathrm{N}$

加速度 $a = 20.6/10 = 2.06\,\mathrm{m/s^2}$

$5\,\mathrm{s}$ 内位移 $x = \frac12 at^2 = \frac12\times2.06\times25 = 25.75\,\mathrm{m}$

**(3)** $F$ 向下 $30^\circ$ 时，竖直方向多了向下的分力：

正压力 $N = mg + F\sin30^\circ = 98 + 50\times0.5 = 123\,\mathrm{N}$

滑动摩擦力 $f_k = \mu_k N = 0.3\times123 = 36.9\,\mathrm{N}$

水平方向合力 $F\cos30^\circ - f_k = 50\times0.866 - 36.9 = 43.3 - 36.9 = 6.4\,\mathrm{N}$

加速度 $a = 6.4/10 = 0.64\,\mathrm{m/s^2}$

$x = \frac12\times0.64\times25 = 8.0\,\mathrm{m}$

**分析**：下压时正压力增大导致摩擦力增大，虽然水平分力仍有 $43.3\,\mathrm{N}$，但净力大幅减小。$5\,\mathrm{s}$ 内的位移从 $25.75\,\mathrm{m}$ 降到 $8.0\,\mathrm{m}$——力方向的微小改变对运动效果有显著影响。

</details>
