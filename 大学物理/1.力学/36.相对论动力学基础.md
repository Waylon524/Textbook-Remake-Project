## 2. 背景与应用场景

核电站利用铀-235 核裂变释放能量。一个铀-235 核吸收中子后裂变为两个中等质量的核，裂变产物的总静质量略小于裂变前的总静质量——亏损的质量 $\Delta m$ 转化为能量 $E = \Delta m \cdot c^2$。$1\,\mathrm{kg}$ 的铀-235 全部裂变释放约 $8.2 \times 10^{13}\,\mathrm{J}$ 的能量，相当于 $2500$ 吨标准煤完全燃烧——质能关系的威力在此彰显。

> **直觉理解**：经典力学中，质量和能量是"两码事"——质量是物质的量，能量是运动或相互作用的量度。爱因斯坦的 $E = mc^2$ 把它们统一了起来：**质量本身就是一种高度浓缩的能量形式**。$1\,\mathrm{kg}$ 任何物质包含的能量约为 $9 \times 10^{16}\,\mathrm{J}$——相当于全人类数天的能源消耗。

相对论动力学的核心在于：牛顿定律在高速（$v \sim c$）下需要修正，修正的核心是将质量和动量概念相对论化。

---

## 3. 核心概念与符号约定

### 相对论质量

物体的质量随运动速度增大而增大：

$$
m = \gamma m_0 = \frac{m_0}{\sqrt{1 - v^2/c^2}}
$$

其中 $m_0$ 为**静质量**（物体静止时的质量），$m$ 为**相对论质量**（运动时的质量）。

### 相对论动量

$$
\mathbf{p} = m\mathbf{v} = \gamma m_0 \mathbf{v}
$$

相对论动量守恒定律：在惯性系中，系统的总相对论动量守恒。

### 相对论动能

$$
E_k = mc^2 - m_0c^2 = (\gamma - 1)m_0c^2
$$

当 $v \ll c$ 时，利用泰勒展开 $\gamma \approx 1 + v^2/(2c^2)$，得：

$$
E_k \approx \frac12 m_0 v^2
$$

——低速下退化为经典动能公式。

### 质能关系 (mass-energy equivalence)

**静能**：$E_0 = m_0c^2$（物体静止时蕴含的能量）

**总能**：$E = mc^2 = \gamma m_0c^2$

**质能关系**：$E = E_k + E_0 = mc^2$

### 能量-动量关系

$$
E^2 = p^2c^2 + m_0^2c^4
$$

这个关系非常有用——它将能量、动量和静质量联系在一个公式中。

对于光子（$m_0 = 0$）：$E = pc$

### 符号约定

| 符号 | 含义 | 单位 |
|------|------|------|
| $m_0$ | 静质量 | $\mathrm{kg}$ |
| $m$ | 相对论质量 | $\mathrm{kg}$ |
| $E$ | 总能 | $\mathrm{J}$ |
| $E_k$ | 动能 | $\mathrm{J}$ |
| $E_0$ | 静能 | $\mathrm{J}$ |
| $p$ | 相对论动量 | $\mathrm{kg \cdot m/s}$ |
| $\gamma$ | 洛伦兹因子 | — |

## 4. 原理与方法

### 相对论动力学的基本方程

相对论中牛顿第二定律的形式需要修正：

$\mathbf{F} = \frac{\mathrm{d}\mathbf{p}}{\mathrm{d}t} = \frac{\mathrm{d}(\gamma m_0 \mathbf{v})}{\mathrm{d}t}$

注意：当力使速度增大时，$\gamma$ 和 $\mathbf{v}$ 都在变化，所以 $\mathbf{F} \neq m\mathbf{a}$（$m$ 也在变）。只有当 $v \ll c$ 时，$\gamma \approx 1$，退化为 $\mathbf{F} = m_0\mathbf{a}$。

### 质能关系的物理含义

$E = mc^2$ 是物理学中最著名的公式之一。其含义：

1. **质量是能量的一种形式**——质量和能量不是独立的概念，而是同一物理实在的两种表现
2. **质量可以转化为能量，能量可以转化为质量**——核反应、粒子对撞中观察到这种转化
3. **静质量 $m_0$ 对应的静能 $m_0c^2$ 是最"浓缩"的能量形式**——微小的质量亏损释放巨大的能量

### 动量-能量关系的应用

$E^2 = p^2c^2 + m_0^2c^4$ 是解决相对论动力学问题的有力工具：

- 已知动量和静质量 → 求总能量
- 已知总能和静质量 → 求动量
- 对极端相对论粒子（$E \gg m_0c^2$）：$E \approx pc$

### 解题策略

| 问题类型 | 解题工具 |
|---------|---------|
| 高速粒子的动能/速度 | $E_k = (\gamma-1)m_0c^2$ |
| 核反应放能 | $\Delta E = \Delta m \cdot c^2$ |
| 粒子碰撞（相对论版） | 动量守恒 $p$ + 能量守恒 $E$ |
| 已知动量求能量 | $E^2 = p^2c^2 + m_0^2c^4$ |

## 5. 例题

### 例题 1：电子在加速器中的动能

电子（$m_0 = 9.11 \times 10^{-31}\,\mathrm{kg}$）被加速到 $v = 0.99c$。求：

(1) 电子的相对论质量
(2) 电子的经典动能和相对论动能
(3) 电子静能是多少 MeV？

**解**：

**建模分析**：$v/c = 0.99$，需用相对论公式。

**(1)** $\gamma = \frac{1}{\sqrt{1 - 0.99^2}} = \frac{1}{\sqrt{0.0199}} = \frac{1}{0.141} \approx 7.09$

$m = \gamma m_0 = 7.09 \times 9.11 \times 10^{-31} \approx 6.46 \times 10^{-30}\,\mathrm{kg}$

**(2)** 经典动能：$E_k^{\text{经典}} = \frac12 m_0 v^2 = 0.5 \times 9.11\times10^{-31} \times (0.99c)^2$

$= 0.5 \times 9.11\times10^{-31} \times 8.82\times10^{16} \approx 4.02\times10^{-14}\,\mathrm{J}$

相对论动能：$E_k = (\gamma - 1)m_0 c^2 = (7.09 - 1) \times 9.11\times10^{-31} \times 9\times10^{16}$

$= 6.09 \times 8.20\times10^{-14} \approx 4.99\times10^{-13}\,\mathrm{J}$

经典动能仅 $4.02\times10^{-14}\,\mathrm{J}$，相对论动能为 $4.99\times10^{-13}\,\mathrm{J}$——相差 12.4 倍！

**(3)** 静能：$E_0 = m_0c^2 = 9.11\times10^{-31} \times 9\times10^{16} = 8.20\times10^{-14}\,\mathrm{J}$

$1\,\mathrm{eV} = 1.602\times10^{-19}\,\mathrm{J}$

$E_0 = \frac{8.20\times10^{-14}}{1.602\times10^{-19}} \approx 5.12\times10^{5}\,\mathrm{eV} = 0.512\,\mathrm{MeV}$

**结论**：$0.99c$ 时电子相对论动能是经典动能的 12.4 倍，需用相对论修正。

### 例题 2：核反应中的质量亏损

一个铀-235 核裂变反应：${}^{235}\mathrm{U} + {}^{1}\mathrm{n} \to {}^{141}\mathrm{Ba} + {}^{92}\mathrm{Kr} + 3{}^{1}\mathrm{n}$

已知静质量：${}^{235}\mathrm{U} = 235.0439\,\mathrm{u}$，${}^{1}\mathrm{n} = 1.0087\,\mathrm{u}$，${}^{141}\mathrm{Ba} = 140.9144\,\mathrm{u}$，${}^{92}\mathrm{Kr} = 91.9262\,\mathrm{u}$（$1\,\mathrm{u} = 1.66\times10^{-27}\,\mathrm{kg}$）。求：

(1) 反应前后的质量亏损
(2) 释放的能量（单位：$\mathrm{J}$ 和 $\mathrm{MeV}$）

**解**：

**建模分析**：质量亏损 = 反应前总静质量 - 反应后总静质量。

**(1)** 反应前总质量：$m_{\text{前}} = 235.0439 + 1.0087 = 236.0526\,\mathrm{u}$

反应后总质量：$m_{\text{后}} = 140.9144 + 91.9262 + 3 \times 1.0087 = 140.9144 + 91.9262 + 3.0261 = 235.8667\,\mathrm{u}$

质量亏损：$\Delta m = 236.0526 - 235.8667 = 0.1859\,\mathrm{u}$

换算为千克：$\Delta m = 0.1859 \times 1.66\times10^{-27} = 3.086\times10^{-28}\,\mathrm{kg}$

**(2)** 释放能量：$\Delta E = \Delta m \cdot c^2 = 3.086\times10^{-28} \times 9\times10^{16} \approx 2.78\times10^{-11}\,\mathrm{J}$

换算为 MeV：$\Delta E = \frac{2.78\times10^{-11}}{1.602\times10^{-19} \times 10^6} \approx 173\,\mathrm{MeV}$

**结论**：一次铀-235 裂变释放约 $173\,\mathrm{MeV}$ 的能量。$1\,\mathrm{kg}$ 铀-235 完全裂变释放约 $8.2\times10^{13}\,\mathrm{J}$——相当于 $2500$ 吨标准煤。

### 例题 3：能量-动量关系应用

一个静质量为 $m_0$ 的粒子，其动量为 $p = m_0c$。求：

(1) 粒子的总能量
(2) 粒子的速度
(3) 粒子的动能

**解**：

**建模分析**：已知 $p = m_0c$，利用能量-动量关系求解。

**(1)** $E^2 = p^2c^2 + m_0^2c^4 = (m_0c)^2 c^2 + m_0^2c^4 = m_0^2c^4 + m_0^2c^4 = 2m_0^2c^4$

$E = \sqrt{2}\,m_0c^2$

**(2)** 由 $p = \gamma m_0 v$ 和 $E = \gamma m_0 c^2$，得 $p/E = v/c^2$

$v = \frac{pc^2}{E} = \frac{m_0c \cdot c^2}{\sqrt{2}\,m_0c^2} = \frac{c}{\sqrt{2}} \approx 0.707c$

**(3)** $E_k = E - m_0c^2 = \sqrt{2}\,m_0c^2 - m_0c^2 = (\sqrt{2} - 1)m_0c^2 \approx 0.414\,m_0c^2$

**验证**：$\gamma = \frac{1}{\sqrt{1 - v^2/c^2}} = \frac{1}{\sqrt{1 - 0.5}} = \frac{1}{\sqrt{0.5}} = \sqrt{2} \approx 1.414$，$E = \gamma m_0c^2 = \sqrt{2}m_0c^2$，$p = \gamma m_0 v = \sqrt{2} \times m_0 \times c/\sqrt{2} = m_0c$，结果一致。✅

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|------|---------|
| "$E = mc^2$ 意味着质量可以转化为能量" | 质量不是"转化"为能量——质量和能量是等价的。核反应中总能量（包括静能）守恒，只是静质量减少的部分以辐射动能的形式释放 |
| "相对论中动能就是 $E_k = \frac12 mv^2$，只需把 $m$ 换成 $\gamma m_0$" | 相对论动能是 $(\gamma-1)m_0c^2$，**不是** $\frac12(\gamma m_0)v^2$ |
| "$m = \gamma m_0$ 意味着质量真实增加了" | 相对论质量是**测量效应**——在不同参考系中测量的质量值不同。粒子的固有属性仍是静质量 $m_0$ |
| "光子有能量 $E = h\nu$，所以光子的静质量不为零" | 光子 $m_0 = 0$，$E = pc$，$\gamma \to \infty$。光子的动量由能量决定，不需要静质量 |
| "低速下相对论动能公式和经典动能公式完全一样" | 低速下 $E_k \approx \frac12 m_0 v^2$，是泰勒展开的一级近似，二者并不完全相同（相差高阶项） |

### 检查点

- [ ] 能否写出相对论动能、总能量、静能三者的关系？
- [ ] 能量-动量关系 $E^2 = p^2c^2 + m_0^2c^4$ 对零质量粒子（光子）是否成立？
- [ ] 核反应 $\Delta E = \Delta m \cdot c^2$ 中的 $\Delta m$ 是何含义？
- [ ] 能否计算一个给定 $v/c$ 的粒子的 $\gamma$ 和动能？
- [ ] 相对论中动量守恒是否还成立？（是——但动量是 $\mathbf{p} = \gamma m_0 \mathbf{v}$）

## 练习题

### 练习1（基础巩固）

一个质子（$m_0 = 1.673 \times 10^{-27}\,\mathrm{kg}$）被加速到 $v = 0.8c$。求：

(1) 质子的相对论质量
(2) 质子的总能量和动能（用 $\mathrm{J}$ 和 $\mathrm{MeV}$ 表示）

<details>
<summary><b>参考答案</b></summary>

$\gamma = \frac{1}{\sqrt{1 - 0.8^2}} = \frac{1}{0.6} = 1.667$

**(1)** $m = \gamma m_0 = 1.667 \times 1.673\times10^{-27} \approx 2.79\times10^{-27}\,\mathrm{kg}$

**(2)** 静能 $E_0 = m_0c^2 = 1.673\times10^{-27} \times 9\times10^{16} = 1.506\times10^{-10}\,\mathrm{J}$

$E_0(\mathrm{MeV}) = \frac{1.506\times10^{-10}}{1.602\times10^{-19} \times 10^6} \approx 940\,\mathrm{MeV}$

总能量 $E = \gamma m_0c^2 = 1.667 \times 1.506\times10^{-10} = 2.51\times10^{-10}\,\mathrm{J}$

$E(\mathrm{MeV}) = 1.667 \times 940 \approx 1567\,\mathrm{MeV}$

动能 $E_k = E - E_0 = 2.51\times10^{-10} - 1.506\times10^{-10} = 1.004\times10^{-10}\,\mathrm{J}$

$E_k(\mathrm{MeV}) = 1567 - 940 = 627\,\mathrm{MeV}$

</details>

### 练习2（迁移应用）

一个静质量为 $m_0$ 的粒子以 $v = 0.6c$ 运动，与另一个静止的相同粒子发生完全非弹性碰撞（碰后粘合在一起）。求碰后合成粒子的静质量。

<details>
<summary><b>参考答案</b></summary>

**建模分析**：相对论中，碰撞仍满足动量守恒和能量（总能量）守恒。注意碰后合成粒子的静质量会大于 $2m_0$（部分动能转化为静质量）。

运动粒子的 $\gamma = 1/\sqrt{1-0.6^2} = 1/0.8 = 1.25$

**碰撞前**：

运动粒子：动量 $p = \gamma m_0 v = 1.25 m_0 \times 0.6c = 0.75 m_0c$

运动粒子总能量 $E_1 = \gamma m_0c^2 = 1.25 m_0c^2$

静止粒子总能量 $E_2 = m_0c^2$

系统总能量 $E = E_1 + E_2 = 1.25 m_0c^2 + m_0c^2 = 2.25 m_0c^2$

系统总动量 $p = 0.75 m_0c$

**碰撞后**：

合成粒子（静质量 $M_0$，速度 $V$）的总能量 $E' = \Gamma M_0 c^2$，动量 $p' = \Gamma M_0 V$

能量守恒：$E' = E = 2.25 m_0c^2$

动量守恒：$p' = p = 0.75 m_0c$

利用能量-动量关系：$E'^2 = p'^2c^2 + M_0^2c^4$

$(2.25 m_0c^2)^2 = (0.75 m_0c)^2c^2 + M_0^2c^4$

$5.0625 m_0^2c^4 = 0.5625 m_0^2c^4 + M_0^2c^4$

$M_0^2 = 5.0625 - 0.5625 = 4.5\,m_0^2$

$M_0 = \sqrt{4.5}\,m_0 \approx 2.121\,m_0$

**结论**：合成粒子的静质量约为 $2.12\,m_0$，大于 $2m_0$——多出的 $0.12\,m_0$ 来自运动粒子的动能转化为静质量。

在经典力学中，完全非弹性碰撞的动能损失转化为内能（热能）；在相对论中，这部分能量表现为静质量的增加。这就是质能关系 $E = mc^2$ 的直接体现。

</details>
