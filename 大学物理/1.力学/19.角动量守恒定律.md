## 2. 背景与应用场景

花样滑冰运动员的贝尔曼旋转是角动量守恒最优雅的展示：运动员单腿站立旋转，先将手臂和自由腿向前伸展，然后缓慢收回身体——转速越来越大。当手臂完全收拢贴在胸前时，转速可以达到每秒 3 圈以上。整个过程中，运动员对旋转轴的 $J\omega$ 保持不变。

> **直觉理解**：坐在办公椅上旋转，双手抱胸转得快，双手伸直转得慢——你什么都没做，只是改变了手臂的位置，转速就自动变化了。这是因为角动量 $J\omega$ 守恒，你改变了 $J$（转动惯量），$\omega$ 就必须反比地变化。

天体物理中的角动量守恒更令人惊叹。蟹状星云中心的脉冲星（中子星）每秒旋转约 30 次。它的前身是一颗质量约 $8\!-\!10$ 倍太阳质量的恒星，核心半径约 $10^5\,\mathrm{km}$，自转周期约数天。恒星核心坍缩成半径仅 $10\,\mathrm{km}$ 的中子星时，半径缩小到原来的 $10^{-4}$，角速度增大到 $10^4$ 倍。角动量守恒使这颗中子星获得了极为稳定的周期，成为宇宙中最精确的"时钟"之一。

## 3. 核心概念

### 角动量守恒定律

系统所受合外力矩为零时，系统对同一转轴的角动量守恒：

$$
\sum\mathbf{M}_{\text{外}} = 0 \implies \mathbf{L} = \sum \mathbf{L}_i = \text{常量}
$$

对定轴转动刚体：$J_1\omega_1 = J_2\omega_2$

**注意**：合外力矩为零不同于合外力为零。力通过转轴时对该轴的力矩为零。

### 动量守恒 vs 角动量守恒

| 对比项 | 动量守恒 | 角动量守恒 |
|--------|---------|-----------|
| 条件 | $\sum\mathbf{F}_{\text{外}} = 0$ | $\sum\mathbf{M}_{\text{外}} = 0$ |
| 守恒量 | $\mathbf{p} = m\mathbf{v}$ | $\mathbf{L} = \mathbf{r}\times\mathbf{p}$ |
| 对应变换 | 空间平移对称性 | 空间旋转对称性 |
| 惯性量度 | 质量 $m$ | 转动惯量 $J$ |
| 生活中的例子 | 台球碰撞前后总动量不变 | 滑冰运动员收臂加速 |

### 角动量守恒时动能为什么不守恒？

设角动量 $L = J\omega$ 守恒，则动能：

$$
E_k = \frac12 J\omega^2 = \frac12 J\left(\frac{L}{J}\right)^2 = \frac{L^2}{2J}
$$

$L$ 不变时 $E_k \propto 1/J$。$J$ 减小则 $E_k$ 增大，反之亦然。这说明角动量守恒时动能**可以**变化。多出的能量来自系统内部（如肌肉做功），损失的能量转化为内能。

## 4. 角动量守恒的典型应用

### 判断方法

1. **选系统**：确定研究对象
2. **选转轴**：明确对哪个轴的角动量
3. **分析外力矩**：外力是否对该轴产生力矩
4. **判断守恒**：若合外力矩为零，角动量守恒

### 常见情形中的角动量守恒

| 情形 | 为什么角动量守恒 | 结果 |
|------|----------------|------|
| 旋转舞者收臂 | 重力与转轴平行，摩擦力矩可忽略 | $J\downarrow \Rightarrow \omega\uparrow$ |
| 跳水运动员屈体 | 空中只有重力（通过质心，无力矩） | 角动量守恒，控制翻转速度 |
| 卫星绕行星运动 | 万有引力为有心力（力矩为零） | $r_pv_p = r_av_a$（开普勒第二定律） |
| 两圆盘摩擦耦合 | 摩擦力为内力，合外力矩为零 | 总角动量守恒 |
| 子弹射入杆端 | 碰撞中冲击力为内力，重力力矩可忽略 | 角动量近似守恒 |

### 开普勒第二定律的角动量解释

行星在近日点速度大、远日点速度小——这是角动量守恒的直接结果：

$L = mr_pv_p = mr_av_a \Rightarrow r_pv_p = r_av_a$

## 5. 例题

### 例题 1：旋转椅

旋转椅对竖直轴的转动惯量 $J_0 = 100\,\mathrm{kg\cdot m^2}$，初始 $\omega_0 = 1.0\,\mathrm{rad/s}$。$m = 60\,\mathrm{kg}$ 的人站在椅中心。

(1) 人走到距轴 $R=2\,\mathrm{m}$ 的台边后，角速度变为多少？
(2) 动能变化了多少？解释原因。

**解**：

**(1)** 系统（椅+人）对转轴的合外力矩为零，角动量守恒。

人在中心：$L_1 = J_0\omega_0 = 100\times1.0 = 100\,\mathrm{kg\cdot m^2/s}$

人在台边：$L_2 = (J_0 + mR^2)\omega = (100 + 60\times4)\omega = 340\omega$

由 $L_1 = L_2$：$100 = 340\omega$，$\omega = 0.294\,\mathrm{rad/s}$

**(2)** 初始动能 $E_{k1} = \frac12 J_0\omega_0^2 = 0.5\times100\times1 = 50\,\mathrm{J}$

末动能 $E_{k2} = \frac12 (J_0+mR^2)\omega^2 = 0.5\times340\times0.0864 = 14.7\,\mathrm{J}$

动能减少 $35.3\,\mathrm{J}$。人和转台之间没有能量输入，但人向外走时脚与台面之间的静摩擦力做负功，将部分机械能转化为内能。

### 例题 2：子弹射入杆端

一根质量 $m_0 = 1\,\mathrm{kg}$、长度 $L = 1\,\mathrm{m}$ 的均匀细棒，可绕其一端在竖直平面内自由转动。初始时棒静止在水平位置。质量为 $m = 10\,\mathrm{g} = 0.01\,\mathrm{kg}$ 的子弹以水平速度 $v_0 = 400\,\mathrm{m/s}$ 击中棒的自由端并嵌入棒内。求碰撞后瞬间棒的角速度。

**解**：

**建模**：子弹与棒系统对转轴的合外力矩为零（碰撞力为内力，重力通过转轴），角动量守恒。

子弹入射方向与棒垂直，对转轴的角动量 $L_{\text{子}} = mv_0L$。

棒对端点的转动惯量 $J_{\text{棒}} = \frac13 m_0L^2 = \frac13\times1\times1 = 0.333\,\mathrm{kg\cdot m^2}$

子弹嵌入棒端后的转动惯量 $J_{\text{子}} = mL^2 = 0.01\times1 = 0.01\,\mathrm{kg\cdot m^2}$

碰前角动量：$L_1 = mv_0L = 0.01\times400\times1 = 4\,\mathrm{kg\cdot m^2/s}$

碰后角动量：$L_2 = (J_{\text{棒}} + J_{\text{子}})\omega = (0.333+0.01)\omega = 0.343\omega$

由 $L_1 = L_2$：$4 = 0.343\omega$，$\omega = 4/0.343 = 11.66\,\mathrm{rad/s}$

## 6. 常见误区与检查点

| 误区 | 正确理解 |
|------|---------|
| "合外力为零则角动量守恒" | 需要**合外力矩**为零——力通过转轴时力矩为零 |
| "角动量守恒就是转速不变" | $J\omega$ 不变，$J$ 变化时 $\omega$ 随之反比变化 |
| "角动量守恒时动能也守恒" | 动能 $E_k = L^2/(2J)$，$J$ 变化时动能必变 |
| "有摩擦力就不守恒" | 摩擦力力矩为零时仍守恒（如静摩擦力通过转轴） |

## 练习题

### 练习1（基础巩固）

旋转椅上的人双臂伸展时 $J_1 = 5\,\mathrm{kg\cdot m^2}$，$\omega_1 = 2\,\mathrm{rad/s}$；收臂后 $J_2 = 2\,\mathrm{kg\cdot m^2}$。求收臂后的 $\omega_2$ 和动能变化，并解释能量来源。

<details>
<summary><b>参考答案</b></summary>

角动量守恒：$J_1\omega_1 = J_2\omega_2$

$\omega_2 = 5\times2/2 = 5\,\mathrm{rad/s}$

收臂后转速变为原来的 $2.5$ 倍。

动能变化：$E_{k1} = \frac12\times5\times4 = 10\,\mathrm{J}$

$E_{k2} = \frac12\times2\times25 = 25\,\mathrm{J}$

动能增加了 $15\,\mathrm{J}$。这额外的能量来自人收缩手臂时肌肉所做的功——手臂向中心移动时，受到惯性离心力的反抗，肌肉做正功转化为转动动能。

</details>

### 练习2（迁移应用）

一颗中子星的半径为 $R_1 = 10^5\,\mathrm{km}$，自转周期 $T_1 = 30$ 天。坍缩后半径变为 $R_2 = 10\,\mathrm{km}$。假设质量分布均匀且角动量守恒，求：

(1) 中子星的自转周期
(2) 表面赤道上的线速度
(3) 这个结果说明了什么？

<details>
<summary><b>参考答案</b></summary>

**(1)** 球体的转动惯量 $J = \frac25 mR^2$。角动量守恒：

$\frac25 mR_1^2\omega_1 = \frac25 mR_2^2\omega_2$

$R_1^2\cdot\frac{2\pi}{T_1} = R_2^2\cdot\frac{2\pi}{T_2}$

$T_2 = T_1\left(\frac{R_2}{R_1}\right)^2 = 30\times\left(\frac{10}{10^5}\right)^2 = 30\times10^{-8} = 3\times10^{-7}\,\text{天}$

$T_2 = 3\times10^{-7}\times24\times3600 \approx 0.026\,\mathrm{s}$

实际中子星的周期约 $0.001\!-\!1\,\mathrm{s}$，与计算结果在同一量级。

**(2)** 赤道线速度 $v = \frac{2\pi R_2}{T_2} = \frac{2\pi\times10^4}{0.026} \approx 2.4\times10^6\,\mathrm{m/s}$

约 $0.8\%$ 光速——这是宇宙中已知最快的宏观自转速度之一。

**(3)** 角动量守恒是宇宙中普遍成立的定律。恒星坍缩时半径急剧减小，转速急剧增大，最终可以产生每秒几百转的中子星。

</details>
