# 07. 相对运动与伽利略变换

> 所属切块：01-力学  |  块内序号：07  |  难度：基础
> 常见混淆点：速度合成时易忽略参考系转换的方向性

## 1. 学习目标与先修前置

### 学习目标
- 理解相对运动的概念——同一物体在不同参考系中有不同描述
- 掌握伽利略速度变换公式及矢量合成方法
- 学会用相对运动分析"雨滴方向""船渡河""飞机导航"等实际问题
- 理解经典力学中加速度的绝对性

### 先修知识
- 质点运动学基础（文件 02-04）
- 矢量加法和减法
- 运动的相对性直觉（如火车上看到树在动）

---

## 2. 背景与应用场景

空中加油是相对运动的经典应用——加油机和受油机需要在同一时刻保持相对静止，即相对速度为零。这要求两架飞机的绝对速度完全相等。稍有偏差就会导致碰撞。

> **直觉理解**：雨天你在公交车上，车停时看到雨是竖直下落的。车开动后，你看到雨是斜着向你飘来的——雨相对地面没变，但相对你却有了水平速度分量。

其他应用场景：
- **船渡河问题**：船头指向哪里，船实际就往哪里走吗？不对——水流会把船冲向下游
- **飞机航线规划**：顺风飞行省油，逆风飞行费油，侧风需要修正航向
- **多普勒雷达测速**：利用电磁波在运动参考系中的频率变化测量车速

---

## 3. 核心概念与符号约定

### 伽利略速度变换

设有三个对象：被研究的物体 $A$、基本参考系 $K$（通常取地面）、运动参考系 $K'$。三者速度满足：

$$
\mathbf{v}_{AK} = \mathbf{v}_{AK'} + \mathbf{v}_{K'K}
$$

其中三个速度的含义：

| 速度 | 名称 | 含义 |
|------|------|------|
| $\mathbf{v}_{AK}$ | 绝对速度 | 物体 $A$ 相对于 $K$ 系的速度 |
| $\mathbf{v}_{AK'}$ | 相对速度 | 物体 $A$ 相对于 $K'$ 系的速度 |
| $\mathbf{v}_{K'K}$ | 牵连速度 | $K'$ 系相对于 $K$ 系的速度 |

### 伽利略加速度变换

$$
\mathbf{a}_{AK} = \mathbf{a}_{AK'} + \mathbf{a}_{K'K}
$$

当 $K'$ 系相对 $K$ 系匀速直线运动时（$\mathbf{a}_{K'K} = 0$），有 $\mathbf{a}_{AK} = \mathbf{a}_{AK'}$——**加速度是绝对量**，与惯性系的选择无关。这正是牛顿第二定律在所有惯性系中形式不变的原因。

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $\mathbf{v}_{AK}$ | 绝对速度 | $\mathrm{m/s}$ |
| $\mathbf{v}_{AK'}$ | 相对速度 | $\mathrm{m/s}$ |
| $\mathbf{v}_{K'K}$ | 牵连速度 | $\mathrm{m/s}$ |

---

## 4. 原理与方法

### 解题四步法

1. **确定三个对象**：明确哪个是物体 $A$，哪个是基本系 $K$，哪个是运动系 $K'$
2. **写出已知速度**：包括大小和方向
3. **画出速度矢量图**：$\mathbf{v}_{AK} = \mathbf{v}_{AK'} + \mathbf{v}_{K'K}$ 构成三角形或平行四边形
4. **解三角形**：利用正弦定理、余弦定理或坐标分解求解

### 经典力学的时空观

伽利略变换隐含了经典力学的绝对时空观——时间在所有参考系中均匀流逝（$t = t'$），空间长度在不同参考系中相同。这仅在低速时成立（$v \ll c$），高速时需用洛伦兹变换。

### 适用条件

- $K'$ 系相对于 $K$ 系平动（无转动）
- 物体速度远小于光速（$v \ll c$）

## 5. 例题

### 例题 1：雨中行人

无风时雨滴相对地面以 $v_r = 5\,\mathrm{m/s}$ 竖直下落。人持伞以 $v_p = 1.5\,\mathrm{m/s}$ 在水平路面行走。

(1) 为不被淋湿，伞柄应向前倾还是向后倾？与竖直方向成多大角度？
(2) 若风速为 $2\,\mathrm{m/s}$ 迎面吹来（即雨滴有水平速度分量），角度如何变化？

**解**：

**建模**：以地面为 $K$ 系、人为 $K'$ 系。雨相对地面的速度 $\mathbf{v}_{AK} = (0, -5)\,\mathrm{m/s}$。人相对地面的速度 $\mathbf{v}_{K'K} = (1.5, 0)\,\mathrm{m/s}$。

**(1)** 雨相对人的速度 $\mathbf{v}_{AK'} = \mathbf{v}_{AK} - \mathbf{v}_{K'K} = (-1.5, -5)\,\mathrm{m/s}$

方向与竖直方向夹角 $\theta = \arctan(1.5/5) = \arctan(0.3) \approx 16.7^\circ$

伞应向前倾斜 $16.7^\circ$（指向行走方向）。

**(2)** 若迎面有 $2\,\mathrm{m/s}$ 的风，雨的水平速度变为 $-2\,\mathrm{m/s}$（迎面吹来）。则 $\mathbf{v}_{AK} = (-2, -5)\,\mathrm{m/s}$。雨相对人的速度 $\mathbf{v}_{AK'} = (-2-1.5, -5) = (-3.5, -5)\,\mathrm{m/s}$

$\theta = \arctan(3.5/5) \approx 35.0^\circ$——伞需要更向前倾。

### 例题 2：船渡河

河宽 $d = 200\,\mathrm{m}$，水流速度 $\mathbf{v}_{\text{水}} = 3\,\mathrm{m/s}$ 向东。船在静水中的航速 $v_{\text{船}} = 5\,\mathrm{m/s}$。

(1) 船头指向正北时，船的实际航速和航向是什么？到达对岸时在下游何处？
(2) 船头应朝什么方向才能垂直过河到达正对岸？过河需时多少？
(3) 要使渡河时间最短，船头应指向何方？最短时间是多少？

**解**：

**建模**：以地面为 $K$ 系、河水为 $K'$ 系。船相对水的速度 $\mathbf{v}_{AK'} = 5\,\mathrm{m/s}$，方向由船头指向决定。水相对地的速度 $\mathbf{v}_{K'K} = 3\,\mathrm{m/s}$ 向东。

**(1)** 船头指北时 $\mathbf{v}_{AK'} = (0, 5)\,\mathrm{m/s}$（北为正 $y$）。

绝对速度 $\mathbf{v}_{AK} = (0,5) + (3,0) = (3,5)\,\mathrm{m/s}$

大小 $v = \sqrt{3^2+5^2} = \sqrt{34} \approx 5.83\,\mathrm{m/s}$，方向北偏东 $\alpha = \arctan(3/5) \approx 31.0^\circ$

过河时间 $t = d/v_y = 200/5 = 40\,\mathrm{s}$，被水冲下的距离 $x = v_x t = 3\times40 = 120\,\mathrm{m}$（下游）。

**(2)** 要垂直过河，须绝对速度 $\mathbf{v}_{AK}$ 的 $x$ 分量为零。设船头与正北夹角 $\theta$（偏向上游）：

$5\sin\theta - 3 = 0$，$\sin\theta = 0.6$，$\theta = \arcsin(0.6) \approx 36.9^\circ$

过河速度 $v_y = 5\cos\theta = 5\times0.8 = 4\,\mathrm{m/s}$

时间 $t = 200/4 = 50\,\mathrm{s}$

**(3)** 渡河时间仅由 $y$ 方向分速度决定。船头指向正北时 $v_y$ 最大（$=5\,\mathrm{m/s}$），时间最短。$t_{\min} = 200/5 = 40\,\mathrm{s}$（与(1)相同）。代价是在下游 $120\,\mathrm{m}$ 处登岸。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|------|---------|
| "船头指向就是船的行进方向" | 船的行进方向是绝对速度方向，船头指向是相对水的速度方向，两者受水流影响而不同 |
| "风从正北来"意味着风向北吹 | "风从某方向来"是指风的来向，即风的速度方向与来向相反——"从北来"的风速度向南 |
| "相对运动只适用于直线运动" | 相对运动适用于任意运动，但伽利略速度变换仅适用于平动参考系 |
| "在不同参考系中加速度不同" | 经典力学中，惯性系之间的加速度是绝对量 |

### 检查点

- [ ] 能否正确识别速度变换中的三个对象（$A, K, K'$）？
- [ ] 能否画出速度矢量三角形？
- [ ] 能否求解船渡河的最短时间和最短路径问题？
- [ ] 是否理解"相对运动趋势"与"相对运动"的区别？

## 练习题

### 练习1（基础巩固）

某人以 $v_1 = 4\,\mathrm{km/h}$ 向东行走时，感觉风从正北吹来。将速度增加一倍（$v_2 = 8\,\mathrm{km/h}$ 向东）时，感觉风从东北方向吹来。求真实风速和风向。

<details>
<summary><b>参考答案</b></summary>

**建模**：以地面为 $K$ 系、人为 $K'$ 系。设真实风速 $\mathbf{v}_{AK} = (v_x, v_y)$（$x$ 东、$y$ 北）。

**情形一**：人向东 $4\,\mathrm{km/h}$，$\mathbf{v}_{K'K}^{(1)} = (4, 0)$。

风从正北来，即风相对人向南吹：$\mathbf{v}_{AK'}^{(1)} = (0, -u)$。

$\mathbf{v}_{AK} = (0, -u) + (4, 0) = (4, -u)$ —— ①

**情形二**：人向东 $8\,\mathrm{km/h}$，$\mathbf{v}_{K'K}^{(2)} = (8, 0)$。

风从东北来，即风相对人向西南吹：$\mathbf{v}_{AK'}^{(2)} = (-w/\sqrt{2}, -w/\sqrt{2})$（与正东成 $135^\circ$ 的西南方向）。

$\mathbf{v}_{AK} = (8 - w/\sqrt{2}, -w/\sqrt{2})$ —— ②

**列方程**：真实风速不变，①②两式相等：

$v_x = 4 = 8 - w/\sqrt{2}$，解得 $w = 4\sqrt{2}$

$v_y = -u = -w/\sqrt{2} = -4$

**结论**：$\mathbf{v}_{AK} = (4, -4)\,\mathrm{km/h}$

风速大小 $v = \sqrt{4^2 + 4^2} = 4\sqrt{2} \approx 5.66\,\mathrm{km/h}$

风向与正东方向夹角 $\arctan(4/4) = 45^\circ$，即**东偏南 $45^\circ$**（东南风，由东南向西北吹）。

</details>

### 练习2（迁移应用）

一架飞机在静止空气中以 $v_p = 300\,\mathrm{km/h}$ 飞行。现要由 $A$ 地直线飞往正东 $600\,\mathrm{km}$ 处的 $B$ 地。已知风速 $v_w = 60\,\mathrm{km/h}$ 从正北方向吹来。

(1) 飞机应朝什么方向飞行（机头指向）？
(2) 从 $A$ 到 $B$ 需要多长时间？
(3) 如果风速方向变为从正东方向吹来，答案如何变化？

<details>
<summary><b>参考答案</b></summary>

**建模**：以地面为 $K$ 系、空气为 $K'$ 系。风从北来即空气相对地面向南运动：$\mathbf{v}_{K'K} = (0, -60)\,\mathrm{km/h}$。

**(1)** 要使飞机相对地面的速度 $\mathbf{v}_{AK}$ 沿正东方向，$y$ 分量必须为零。

设机头（相对空气的速度）与正东方向夹角为 $\theta$（北偏），则 $\mathbf{v}_{AK'} = (300\cos\theta, 300\sin\theta)$。

由 $\mathbf{v}_{AK} = \mathbf{v}_{AK'} + \mathbf{v}_{K'K}$ 的 $y$ 方向分量为零：

$300\sin\theta - 60 = 0$，$\sin\theta = 0.2$，$\theta = \arcsin(0.2) \approx 11.5^\circ$

飞机应朝**东偏北 $11.5^\circ$** 方向飞行。

**(2)** 对地速度 $x$ 分量：$v_{AK,x} = 300\cos 11.5^\circ = 300 \times 0.9798 = 293.9\,\mathrm{km/h}$

所需时间 $t = \frac{600}{293.9} = 2.04\,\mathrm{h} \approx 2\,\mathrm{h}\,2\,\mathrm{min}$

比无风时的 $2\,\mathrm{h}$ 略长，因为需要偏转航向产生了水平分量损失。

**(3)** 东风（风从东来）即空气向西运动：$\mathbf{v}_{K'K} = (-60, 0)\,\mathrm{km/h}$。

这时不需要偏航，机头直接指向正东：$\mathbf{v}_{AK'} = (300, 0)$。

对地速度 $v_{AK} = 300 - 60 = 240\,\mathrm{km/h}$

时间 $t = 600/240 = 2.5\,\mathrm{h}$（逆风飞行，时间更长）

若从 $B$ 返回 $A$（向西），则变为顺风：$v = 300 + 60 = 360\,\mathrm{km/h}$，$t = 600/360 = 1.67\,\mathrm{h}$。来回时间不同，这就是航空中常说的"风对飞行时间的影响"。

</details>
