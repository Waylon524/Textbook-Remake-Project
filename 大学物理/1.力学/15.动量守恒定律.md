## 2. 背景与应用场景

2020 年 12 月，嫦娥五号从月球表面起飞，将月壤样本送入环月轨道。嫦娥五号的上升器与着陆器分离的过程中，利用动量守恒原理——上升器向一个方向获得动量，着陆器提供反方向的动量补偿。没有发射塔、没有导流槽，在真空环境中顺利完成起飞。

> **直觉理解**：两个人背对背站在光滑冰面上，一个人推开另一个人——两人会向相反方向滑行。他们初始总动量为零，推人后两人各获得大小相等、方向相反的动量，总动量仍然为零。如果其中一个人的质量更大，他获得的速度就更小。

动量守恒定律是物理学中最基本、最普适的定律之一——它适用于从亚原子粒子（如中微子与电子的碰撞）到星系（如星系碰撞合并）的各个尺度。它适用于经典力学和相对论力学。

### 动量守恒 vs 动能守恒

| 性质 | 动量守恒 | 动能守恒 |
|------|---------|---------|
| 条件 | 合外力为零 | 合外力做功为零（或非保守力不做功） |
| 矢量/标量 | 矢量 | 标量 |
| 适用范围 | 所有物理领域（包括相对论） | 经典力学为主 |
| 碰撞中 | 始终守恒 | 仅弹性碰撞守恒 |

## 3. 核心概念

### 动量守恒定律

系统所受合外力为零时，系统总动量保持不变：

$$
\sum\mathbf{F}_{\text{外}} = 0 \implies \mathbf{P} = \sum m_i\mathbf{v}_i = \text{常量}
$$

### 分量形式

合外力在 $x$ 方向分量为零时：

$$
\sum m_i v_{ix} = \text{常量}
$$

这在分析二维和三维碰撞时非常有用——可以分别研究各个方向的动量守恒。

### 近似守恒

碰撞、爆炸、发射等过程的共同特点：作用时间极短（毫秒级），内力极大（百倍千倍于外力）。此时即使合外力不严格为零，外力的冲量与内力的冲量相比可忽略，系统动量**近似守恒**。

### 三种碰撞类型

| 碰撞类型 | 动量守恒 | 动能守恒 | 碰后特征 |
|---------|---------|---------|---------|
| 完全弹性 | 是 | 是 | 碰后分开，无能量损失 |
| 非弹性 | 是 | 否 | 碰后分开，部分动能转化为内能 |
| 完全非弹性 | 是 | 否（损失最大） | 碰后粘合以共同速度运动 |

## 4. 例题

### 例题 1：完全非弹性碰撞

$m_1 = 2\,\mathrm{kg}$ 以 $v_1 = 3\,\mathrm{m/s}$ 向右运动，$m_2 = 1\,\mathrm{kg}$ 以 $v_2 = 4\,\mathrm{m/s}$ 向左运动。发生完全非弹性碰撞（碰后粘合）。

(1) 求碰后速度
(2) 求碰撞中的动能损失
(3) 损失的动能去了哪里？

**解**：

取向右为正方向。

**(1)** 碰前总动量：

$p = m_1v_1 + m_2v_2 = 2\times3 + 1\times(-4) = 6 - 4 = 2\,\mathrm{kg\cdot m/s}$

碰后两物体粘合：$(m_1 + m_2)v = 2$

$v = \frac{2}{3} \approx 0.667\,\mathrm{m/s}$（向右）

**(2)** 碰前动能：

$E_{k1} = \frac12 m_1v_1^2 + \frac12 m_2v_2^2 = \frac12\times2\times9 + \frac12\times1\times16 = 9 + 8 = 17\,\mathrm{J}$

碰后动能：

$E_{k2} = \frac12 (m_1+m_2)v^2 = \frac12\times3\times(0.667)^2 = 1.5\times0.444 = 0.667\,\mathrm{J}$

动能损失 $\Delta E_k = 17 - 0.667 = 16.33\,\mathrm{J}$

**(3)** 损失的 $16.33\,\mathrm{J}$ 转化为碰撞瞬间的形变能，最终以热量的形式耗散（物体发生永久形变或温度升高）。

### 例题 2：爆炸反冲

一个质量为 $M = 6\,\mathrm{kg}$ 的物体静止在光滑水平面上。内部爆炸后分裂为 $m_1 = 2\,\mathrm{kg}$ 和 $m_2 = 4\,\mathrm{kg}$ 的两部分。测得 $m_1$ 以 $v_1 = 6\,\mathrm{m/s}$ 向右运动。

(1) 求 $m_2$ 的速度
(2) 求爆炸释放的能量（转化为动能的部分）
(3) 验证系统的动量守恒

**解**：

**(1)** 爆炸前总动量为 $0$，爆炸内力>>外力，动量守恒。

$2\times6 + 4\times v_2 = 0$

$v_2 = -\frac{12}{4} = -3\,\mathrm{m/s}$（向左）

**(2)** 爆炸释放的动能即为碎片的总动能：

$E_k = \frac12 m_1v_1^2 + \frac12 m_2v_2^2 = \frac12\times2\times36 + \frac12\times4\times9$

$= 36 + 18 = 54\,\mathrm{J}$

**(3)** 验证动量守恒：碰后总动量 $p = 2\times6 + 4\times(-3) = 12 - 12 = 0$，与碰前相同 ✅

### 例题 3：一维弹性碰撞公式

证明：两个物体发生完全弹性正碰时，$v_1' = \frac{(m_1-m_2)v_1 + 2m_2v_2}{m_1+m_2}$

**证**：

碰前 $m_1$ 速度 $v_1$，$m_2$ 速度 $v_2$。弹性碰撞同时满足动量守恒和动能守恒：

动量守恒：$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$ ... ①

动能守恒：$\frac12 m_1v_1^2 + \frac12 m_2v_2^2 = \frac12 m_1v_1'^2 + \frac12 m_2v_2'^2$ ... ②

从 ① 得：$m_1(v_1 - v_1') = m_2(v_2' - v_2)$ ... ③

从 ② 得：$m_1(v_1^2 - v_1'^2) = m_2(v_2'^2 - v_2^2)$ ... ④

④ ÷ ③：$v_1 + v_1' = v_2 + v_2'$（关键关系：弹性碰撞中相对速度反向）

结合 ① 解方程组得：

$v_1' = \frac{(m_1-m_2)v_1 + 2m_2v_2}{m_1+m_2}$，$v_2' = \frac{(m_2-m_1)v_2 + 2m_1v_1}{m_1+m_2}$

**特例**：$m_1 = m_2$ 时 $v_1' = v_2$，$v_2' = v_1$（速度交换）。

## 6. 常见误区与检查点

| 误区 | 正确理解 |
|------|---------|
| "动量守恒要求合外力严格为零" | 碰撞中内力 >> 外力时可近似守恒，工程上就是够精确 |
| "动量守恒就是速度不变" | 总动量不变，但各物体的速度可以重新分配 |
| "完全非弹性碰撞后都停下来" | 仅当 $m_1v_1 + m_2v_2 = 0$ 时才停下 |
| "动量守恒和动能守恒是一回事" | 这是两种不同的守恒律，动量始终守恒而动能不一定 |
| "质量大的物体动量一定大" | $p=mv$，质量大但速度小也可能动量小 |

## 练习题

### 练习1（基础巩固）

$m_1 = 4\,\mathrm{kg}$ 以 $2\,\mathrm{m/s}$ 向右运动，与 $m_2 = 2\,\mathrm{kg}$ 静止物体发生完全非弹性碰撞。

(1) 求碰后共同速度
(2) 求碰撞前后动能变化

<details>
<summary><b>参考答案</b></summary>

**(1)** 取向右为正。碰前 $p = 4\times2 = 8\,\mathrm{kg\cdot m/s}$

碰后 $(4+2)v = 8$，$v = 8/6 = 1.33\,\mathrm{m/s}$

**(2)** 碰前 $E_{k1} = \frac12\times4\times4 = 8\,\mathrm{J}$

碰后 $E_{k2} = \frac12\times6\times(1.33)^2 = 3\times1.78 = 5.33\,\mathrm{J}$

损失 $\Delta E_k = 2.67\,\mathrm{J}$，转化为内能。

</details>

### 练习2（迁移应用）

一个静止的原子核质量 $M$，内部衰变后分裂为质量 $m_1$ 和 $m_2$ 的两部分，$m_1$ 的动能为 $E_1$。

(1) 求 $m_2$ 的动能 $E_2$
(2) 证明 $E_1/E_2 = m_2/m_1$

<details>
<summary><b>参考答案</b></summary>

**(1)** 衰变前静止，总动量为零。衰变过程中内力 >> 外力，动量守恒。

$m_1v_1 + m_2v_2 = 0 \Rightarrow v_2 = -\frac{m_1}{m_2}v_1$

速度大小比 $v_1/v_2 = m_2/m_1$

$E_1 = \frac12 m_1v_1^2$，$E_2 = \frac12 m_2v_2^2 = \frac12 m_2\left(\frac{m_1}{m_2}v_1\right)^2 = \frac12 \frac{m_1^2}{m_2}v_1^2$

$\frac{E_1}{E_2} = \frac{\frac12 m_1v_1^2}{\frac12 \frac{m_1^2}{m_2}v_1^2} = \frac{m_2}{m_1}$

**(2)** 得证。$E_1/E_2 = m_2/m_1$，即质量较小的碎片获得较多的动能。

**数值验证**：设 $M=120$，$m_1=40$，$m_2=80$，$E_1=100\,\mathrm{MeV}$。

$E_2 = E_1 \times m_1/m_2 = 100\times40/80 = 50\,\mathrm{MeV}$

重碎片获得的动能较少，这与重核裂变的实验观察一致。

</details>
