# 26. 刚体运动学

## 2. 背景与应用场景

工业机器人的机械臂由多个刚体臂段通过关节连接而成。每个关节的转动带动臂段运动，工程师必须精确计算各臂段的角位置、角速度和角加速度，才能控制末端执行器到达指定位置。这正是刚体运动学的核心任务。

> **直觉理解**：想象一把直尺——平推时尺上各点运动完全相同（平动）；固定一端旋转时，尺上不同点的速度不同（离轴越远越快），但它们转过的角度相同（转动）。刚体模型的核心假设是：无论受到多大的力，尺子都不会弯曲或伸长。

刚体模型简化了大量工程问题：齿轮传动、车轮滚动、陀螺仪、机器人运动学分析，都可以用刚体运动学描述。

## 3. 核心概念与符号约定

### 刚体的定义

刚体是指在运动或受力过程中，**任意两点之间的距离始终保持不变**的理想物体。这是质点模型之后又一重要的理想化模型——将物体的形变忽略不计。

### 刚体的基本运动形式

| 运动形式 | 特征 | 举例 |
|---------|------|------|
| 平动 | 刚体内任意两点连线的方向保持不变，各点位移/速度/加速度相同 | 电梯升降、抽屉推拉 |
| 定轴转动 | 刚体上各点绕固定轴做圆周运动，角速度相同 | 门绕铰链转动、风扇转动 |
| 平面运动 | 平动 + 转动的组合，各点运动平行于某一固定平面 | 车轮滚动、滑冰运动员 |
| 一般运动 | 平动 + 转动（三维空间） | 掷出的飞盘、翻滚的体操运动员 |

### 定轴转动的角量描述

角位置：$\theta(t)$，单位为 $\mathrm{rad}$

角速度：$\omega = \dfrac{\mathrm{d}\theta}{\mathrm{d}t}$，单位为 $\mathrm{rad/s}$

角加速度：$\alpha = \dfrac{\mathrm{d}\omega}{\mathrm{d}t} = \dfrac{\mathrm{d}^2\theta}{\mathrm{d}t^2}$，单位为 $\mathrm{rad/s^2}$

### 匀变速转动的运动学公式

与匀变速直线运动在数学形式上完全对应：

| 直线运动 | 定轴转动 |
|---------|---------|
| $v = v_0 + at$ | $\omega = \omega_0 + \alpha t$ |
| $x = x_0 + v_0t + \frac12 at^2$ | $\theta = \theta_0 + \omega_0t + \frac12 \alpha t^2$ |
| $v^2 = v_0^2 + 2a(x-x_0)$ | $\omega^2 = \omega_0^2 + 2\alpha(\theta-\theta_0)$ |

### 线量与角量的关系

刚体上与转轴距离为 $R$ 的一点：

| 线量 | 与角量的关系 | 方向 |
|------|------------|------|
| 弧长 | $s = R\theta$ | — |
| 线速度 | $v = \omega R$ | 沿圆周切线方向 |
| 切向加速度 | $a_\tau = \alpha R$ | 沿圆周切线方向 |
| 法向加速度 | $a_n = \omega^2 R = v^2/R$ | 指向圆心 |

矢量形式：$\mathbf{v} = \boldsymbol{\omega} \times \mathbf{r}$（其中 $\mathbf{r}$ 为点到转轴的位矢）

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $\theta$ | 角位置 | $\mathrm{rad}$ |
| $\omega$ | 角速度 | $\mathrm{rad/s}$ |
| $\alpha$ | 角加速度 | $\mathrm{rad/s^2}$ |
| $n$ | 转速 | $\mathrm{r/min}$（转/分） |
| $s$ | 弧长 | $\mathrm{m}$ |
| $\omega R$ | 线速度 | $\mathrm{m/s}$ |

转速与角速度换算：$\omega = \dfrac{2\pi n}{60}$

## 4. 原理与方法

### 平动分析的核心方法

平动时刚体上各点运动完全相同，因此**可以用质心的运动代表整个刚体**。平动的解题方法与质点运动学完全相同。

### 定轴转动的运动学分析步骤

1. **判断运动类型**：确定是匀变速转动（$\alpha$ 恒定）还是变角速转动（$\alpha$ 变化）
2. **选择参考方向**：规定正方向（通常取逆时针为正）
3. **列运动学方程**：对应 $\omega = \omega_0 + \alpha t$ 或积分形式
4. **求线量**：通过 $v = \omega R$、$a_\tau = \alpha R$、$a_n = \omega^2 R$ 求指定点的线速度、加速度
5. **结果检验**：量纲检查、边界条件验证

### 角加速度的非均匀情况

当 $\alpha$ 不是常数（如 $\alpha = \alpha(\theta)$ 或 $\alpha = \alpha(t)$），需用积分：

$$
\omega = \omega_0 + \int_0^t \alpha(t)\,\mathrm{d}t,\quad
\theta = \theta_0 + \int_0^t \omega(t)\,\mathrm{d}t
$$

或利用 $\alpha = \omega\frac{\mathrm{d}\omega}{\mathrm{d}\theta}$ 进行变量变换后积分。

### 刚体的自由度

| 约束情况 | 自由度数 | 描述 |
|---------|:-------:|------|
| 自由刚体（三维空间） | 6 | 3 平动 + 3 转动 |
| 定轴转动 | 1 | 只有绕轴的角位置可变 |
| 平面运动 | 3 | 2 平动 + 1 转动 |
| 固定点转动 | 3 | 如陀螺，质心固定，三个欧拉角 |

### 角速度的矢量性

角速度 $\boldsymbol{\omega}$ 是矢量，方向由右手螺旋定则确定：**右手四指弯曲方向为转动方向，拇指指向即为 $\boldsymbol{\omega}$ 方向**。

**重要注意**：虽然角速度是矢量，但**有限角位移 $\Delta\boldsymbol{\theta}$ 不是矢量**——因为它不满足矢量加法交换律（两次不同顺序的有限转动结果不同）。只有**无穷小角位移 $\mathrm{d}\boldsymbol{\theta}$ 才是矢量**。这是刚体运动学中常被忽略的细节。

## 5. 例题

### 例题 1：匀变速转动的基本计算

一台工业风扇启动后从静止以恒定角加速度 $\alpha = 1.5\,\mathrm{rad/s^2}$ 加速转动。求：

(1) $t = 4\,\mathrm{s}$ 时的角速度
(2) $t = 4\,\mathrm{s}$ 内转过的角度（用弧度制和圈数表示）
(3) $t = 4\,\mathrm{s}$ 时风扇叶片尖端的速度和加速度（叶片半径 $R = 0.6\,\mathrm{m}$）

**解**：

**建模分析**：匀变速转动，$\alpha =$ 常数，使用匀变速转动公式。$\omega_0 = 0$，$\theta_0 = 0$。

**(1)** $\omega = \omega_0 + \alpha t = 0 + 1.5 \times 4 = 6.0\,\mathrm{rad/s}$

**(2)** $\theta = \omega_0 t + \frac12 \alpha t^2 = 0 + \frac12 \times 1.5 \times 16 = 12\,\mathrm{rad}$

转过的圈数：$N = \frac{\theta}{2\pi} = \frac{12}{2\pi} \approx 1.91$ 圈

**(3)** 叶片尖端的线速度：$v = \omega R = 6.0 \times 0.6 = 3.6\,\mathrm{m/s}$

切向加速度：$a_\tau = \alpha R = 1.5 \times 0.6 = 0.9\,\mathrm{m/s^2}$

法向加速度：$a_n = \omega^2 R = 6.0^2 \times 0.6 = 21.6\,\mathrm{m/s^2}$

总加速度大小：$a = \sqrt{a_\tau^2 + a_n^2} = \sqrt{0.9^2 + 21.6^2} \approx 21.62\,\mathrm{m/s^2}$

**量纲检查**：$v = \omega R$ 量纲：$\mathrm{rad/s \cdot m} \to \mathrm{m/s}$，正确（$\mathrm{rad}$ 无量纲）。

**结论解释**：$4\,\mathrm{s}$ 后叶片转约 1.91 圈，尖端线速度 $3.6\,\mathrm{m/s}$。法向加速度远大于切向加速度，说明此时向心效应占主导。

### 例题 2：由角速度函数求运动学量

一刚体绕定轴转动，其角速度随时间变化为 $\omega(t) = 3t^2 - 2t$（SI 单位制）。$t=0$ 时角位置 $\theta_0 = 0$。求：

(1) 角加速度 $\alpha(t)$ 的表达式
(2) $t = 2\,\mathrm{s}$ 时的角位置和角加速度
(3) $t = 1\,\mathrm{s}$ 到 $t = 3\,\mathrm{s}$ 期间转过的角度

**解**：

**建模分析**：已知 $\omega(t)$ 是变角速转动，需通过求导和积分求解。

**(1)** $\alpha(t) = \dfrac{\mathrm{d}\omega}{\mathrm{d}t} = 6t - 2$

**(2)** $\theta(2) = \theta_0 + \int_0^2 \omega(t)\,\mathrm{d}t = 0 + \int_0^2 (3t^2 - 2t)\,\mathrm{d}t$

$\theta(2) = [t^3 - t^2]_0^2 = (8 - 4) - 0 = 4\,\mathrm{rad}$

$\alpha(2) = 6 \times 2 - 2 = 10\,\mathrm{rad/s^2}$

**(3)** $t=1$ 到 $t=3$ 的角位移：

$\Delta\theta = \int_1^3 (3t^2 - 2t)\,\mathrm{d}t = [t^3 - t^2]_1^3 = (27 - 9) - (1 - 1) = 18\,\mathrm{rad}$

**结论解释**：角加速度随时间线性增大（从 $t=0$ 时的 $-2\,\mathrm{rad/s^2}$ 到 $t=3$ 时的 $16\,\mathrm{rad/s^2}$），说明转动在加速。

### 例题 3：刚体平面运动——无滑滚动

半径为 $R = 0.3\,\mathrm{m}$ 的均匀圆柱体在水平面上做无滑滚动。圆柱体质心的速度 $v_C = 2\,\mathrm{m/s}$，加速度 $a_C = 0.5\,\mathrm{m/s^2}$。求：

(1) 圆柱体的角速度 $\omega$ 和角加速度 $\alpha$
(2) 圆柱体上与地面接触点 $P$ 的速度
(3) 圆柱体顶部点 $Q$ 的速度

**解**：

**建模分析**：无滑滚动条件是圆柱与地面接触点的瞬时速度为零（纯滚动，不打滑）。此时质心速度与角速度满足 $v_C = \omega R$。

**(1)** 角速度：$\omega = \frac{v_C}{R} = \frac{2}{0.3} \approx 6.67\,\mathrm{rad/s}$

角加速度：$\alpha = \frac{a_C}{R} = \frac{0.5}{0.3} \approx 1.67\,\mathrm{rad/s^2}$

**(2)** 接触点 $P$：平动速度 $v_C$ 向右，转动速度 $\omega R = v_C$ 向左（接触点相对质心向后运动）

合成速度：$v_P = v_C - \omega R = v_C - v_C = 0$ ✅（无滑滚动条件满足）

**(3)** 顶部点 $Q$：平动速度 $v_C$ 向右，转动速度 $\omega R = v_C$ 向右（顶部点相对质心向前运动）

合成速度：$v_Q = v_C + \omega R = v_C + v_C = 2v_C = 4\,\mathrm{m/s}$

**结论解释**：无滑滚动中，圆柱顶部速度是质心速度的 **2 倍**，而接触点瞬时静止。这解释了为什么传送带上圆柱体的顶部比底部运动得快。

**一致性检查**：若 $v_C$ 和 $\omega$ 不满足 $v_C = \omega R$，则存在滑动（打滑），滚动摩擦变为滑动摩擦。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|------|---------|
| "刚体上各点的速度和加速度都相同" | 仅**平动**时相同；转动时线速度 $v = \omega R$ 与半径成正比，不同点速度不同 |
| "角速度大小等于转速" | 角速度 $\omega$ 的单位是 $\mathrm{rad/s}$，转速 $n$ 的单位是 $\mathrm{r/min}$，换算关系 $\omega = 2\pi n / 60$ |
| "有限角位移和角速度一样是矢量" | 有限角位移**不满足矢量加法交换律**，不是矢量。只有无穷小角位移 $\mathrm{d}\boldsymbol{\theta}$ 是矢量 |
| "刚体转动中所有点的角加速度相同" | **正确**：定轴转动中所有点的角速度和角加速度相同（因为各点一起转过相同的角度） |
| "无滑滚动时接触点速度为零，所以该点加速度也为零" | 接触点速度瞬时的确为零，但加速度**不为零**——该点同时有向心加速度和切向加速度分量 |

### 检查点

- [ ] 给定刚体的运动状态，能否准确区分平动和转动分量？
- [ ] 已知角速度 $\omega$，能否写出刚体上任意点的速度矢量表达式 $\mathbf{v} = \boldsymbol{\omega} \times \mathbf{r}$？
- [ ] 无滑滚动的约束条件 $v_C = \omega R$ 是否理解其几何含义？
- [ ] 能否进行转速 $n$（r/min）与角速度 $\omega$（rad/s）的快速换算？
- [ ] 计算法向加速度时是否用了正确的公式 $a_n = \omega^2 R = v^2/R$？

## 练习题

### 练习1（基础巩固）

一台离心机的转子从静止开始匀加速转动，$5\,\mathrm{s}$ 内达到转速 $n = 3000\,\mathrm{r/min}$。转子半径 $R = 0.15\,\mathrm{m}$。

(1) 求角加速度 $\alpha$
(2) 求 $5\,\mathrm{s}$ 内转过的总圈数
(3) 达到额定转速时，转子边缘点的线速度、切向加速度和法向加速度

<details>
<summary><b>参考答案</b></summary>

**建模分析**：从静止匀加速转动到额定转速，$\omega_0 = 0$，$\alpha$ 恒定。

**(1)** 额定角速度：$\omega = \frac{2\pi n}{60} = \frac{2\pi \times 3000}{60} = 100\pi \approx 314.16\,\mathrm{rad/s}$

由 $\omega = \omega_0 + \alpha t$：$\alpha = \frac{\omega}{t} = \frac{314.16}{5} \approx 62.83\,\mathrm{rad/s^2}$

**(2)** 总角位移：$\theta = \frac12 \alpha t^2 = \frac12 \times 62.83 \times 25 \approx 785.4\,\mathrm{rad}$

转过的圈数：$N = \frac{\theta}{2\pi} = \frac{785.4}{2\pi} \approx 125$ 圈

（也可用平均角速度：$\bar\omega = \omega/2 = 157.08\,\mathrm{rad/s}$，$\theta = \bar\omega t = 157.08 \times 5 = 785.4\,\mathrm{rad}$）

**(3)** 边缘点线速度：$v = \omega R = 314.16 \times 0.15 \approx 47.12\,\mathrm{m/s}$

切向加速度：$a_\tau = \alpha R = 62.83 \times 0.15 \approx 9.42\,\mathrm{m/s^2}$

法向加速度：$a_n = \omega^2 R = 314.16^2 \times 0.15 \approx 14804\,\mathrm{m/s^2}$

$a_n \approx 1510g$（约 1510 倍重力加速度！这就是离心机分离效应的来源）

</details>

### 练习2（迁移应用）

一个半径为 $R = 0.25\,\mathrm{m}$ 的自行车轮，在水平地面上以 $v = 5\,\mathrm{m/s}$ 的速度匀速直线运动（无滑滚动）。车轮边缘有一红色标记点，初始时刻位于轮轴正上方最高点。

(1) 求车轮的角速度 $\omega$
(2) 求标记点经过 $\frac14$ 周期后的位置和速度（相对地面）

<details>
<summary><b>参考答案</b></summary>

**建模分析**：匀速无滑滚动，质心速度恒定 $v_C = 5\,\mathrm{m/s}$，角速度恒定 $v_C = \omega R$。标记点的运动是平动和转动的合成。

**(1)** $\omega = \frac{v_C}{R} = \frac{5}{0.25} = 20\,\mathrm{rad/s}$

转动周期 $T = \frac{2\pi}{\omega} = \frac{2\pi}{20} \approx 0.314\,\mathrm{s}$

**(2)** $\frac14$ 周期后，车轮转过了 $\theta = \omega \cdot \frac{T}{4} = 20 \times \frac{0.314}{4} = \frac{\pi}{2}\,\mathrm{rad} = 90^\circ$

初始时标记点在最高点（轮轴正上方 $R$ 处）。转过 $90^\circ$ 后，标记点位于轮轴的正**前方** $R$ 处。

与此同时，质心在 $\frac14$ 周期内前进了 $\Delta x = v_C \cdot \frac{T}{4} = 5 \times \frac{0.314}{4} \approx 0.393\,\mathrm{m}$

标记点相对地面的位置（以初始质心位置为原点）：

$x\text{-坐标} = \Delta x + R = 0.393 + 0.25 = 0.643\,\mathrm{m}$

$y\text{-坐标} = R$（与轮轴同高，因为从最高点转 $90^\circ$）

标记点的速度：平动速度 $v_C = 5\,\mathrm{m/s}$ 向右，转动速度 $\omega R = 5\,\mathrm{m/s}$ 向下（$90^\circ$ 位置）

合成速度大小：$v = \sqrt{5^2 + 5^2} = 5\sqrt{2} \approx 7.07\,\mathrm{m/s}$

方向：与水平方向成 $45^\circ$ 向下右方。

**物理意义**：无滑滚动中，车轮上不同点的运动轨迹不同——轮缘点的轨迹为**摆线**。在最高点时，该点速度为 $2v_C$（向前）；在接触点时，速度为零。这正是自行车轮看起来"上半部分比下半部分模糊"的原因——上半部分运动更快。

</details>
