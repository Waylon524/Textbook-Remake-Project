## 2. 背景与应用场景

航天飞机再入大气层时，宇航员承受 $3g$ 的减速加速度。在航天器参考系（非惯性系）中，宇航员感觉自己被一个巨大的"惯性力"压在座椅上。这个力没有施力物体，但必须考虑它才能正确分析宇航员的受力。

> **直觉理解**：公交車紧急刹车时，你感觉身体向前倾——没有真实的力在推你，是身体保持前进的惯性。以公交车为参考系（非惯性系），需要引入一个向前的"惯性力"来解释你的前倾。惯性力的大小等于你的质量乘以公交车的减速度，方向与减速度方向相反。

地球自转引起的惯性力对大气环流有显著影响。北半球的台风都是逆时针旋转——这是因为空气向低压中心汇聚时，受到向右的科里奥利力偏转。如果不考虑科里奥利力（即不在转动的地球参考系中引入惯性力），就无法准确预报台风路径。

## 3. 核心概念

### 平动非惯性系的惯性力

$K'$ 系相对惯性系 $K$ 以 $\mathbf{a}_0$ 平动加速，在 $K'$ 系中观察者看到——即使物体不受真实力，它也会以 $-\mathbf{a}_0$ 加速运动。为了在 $K'$ 系中使用牛顿第二定律，必须引入：

$$
\mathbf{F}_{\text{惯}} = -m\mathbf{a}_0
$$

**特点**：
- 方向与 $K'$ 系的加速度方向相反
- 大小正比于物体质量 $m$（与引力相同，这正是等效原理的基础）
- 虚拟力——无施力物体，无反作用力

### 转动非惯性系的惯性力

**惯性离心力**（与物体是否运动无关）：

$$
F_{\text{离}} = m\omega^2 r
$$

方向沿径向向外。在转动系中看，所有物体都受到这个"向外甩"的力。

**科里奥利力**（仅当物体在转动系中运动时出现）：

$$
\mathbf{F}_{\text{科}} = -2m\boldsymbol{\omega} \times \mathbf{v}'
$$

- 与物体在转动系中的速度 $\mathbf{v}'$ 有关
- 方向垂直于 $\boldsymbol{\omega}$ 和 $\mathbf{v}'$ 决定的平面
- 在北半球使运动物体向右偏，南半球向左偏

### 三种力的对比

| 类型 | 真实力 | 惯性力（平动） | 惯性离心力 | 科里奥利力 |
|------|--------|---------------|-----------|-----------|
| 施力物体 | 有 | 无 | 无 | 无 |
| 反作用力 | 有 | 无 | 无 | 无 |
| 依赖参考系 | 否 | 是 | 是 | 是 |
| 依赖速度 | 不一定 | 否 | 否 | 是 |

## 4. 原理与方法

### 超重与失重的非惯性系解释

以加速上升的电梯为参考系（非惯性系），电梯加速度 $a$ 向上，需引入向下的惯性力 $F_{\text{惯}} = ma$。在电梯系中物体静止：

$$N - mg - ma = 0 \Rightarrow N = m(g+a)$$

**物理图像**：在非惯性系中，惯性力 $ma$ 像附加的重力——"有效重力"变为 $m(g+a)$，所以感觉变重了。

### 转动系中解题

在匀速转动参考系中处理问题：
1. 确定转动角速度 $\omega$
2. 在物体上添加惯性离心力 $m\omega^2 r$（径向向外）
3. 若物体相对转系运动，添加科里奥利力
4. 按平衡或牛顿定律列出方程

## 5. 例题

### 例题 1：水平转盘上的平衡

水平转盘以 $\omega = 3\,\mathrm{rad/s}$ 匀速转动。$m = 0.2\,\mathrm{kg}$ 的物体放在距转轴 $r = 0.4\,\mathrm{m}$ 处，与转盘相对静止。分别从地面参考系和转盘参考系分析物体的受力，并求静摩擦力。

**解**：

**地面系（惯性系）**：物体做匀速圆周运动，向心力由静摩擦力提供。

$f_s = m\omega^2 r = 0.2\times 3^2 \times 0.4 = 0.2\times9\times0.4 = 0.72\,\mathrm{N}$（指向圆心）

**转盘系（非惯性系）**：物体相对转盘静止，引入惯性离心力：

$F_{\text{离}} = m\omega^2 r = 0.72\,\mathrm{N}$（径向向外）

物体在转盘系中受力平衡：$f_s = F_{\text{离}} = 0.72\,\mathrm{N}$（指向圆心）

**对比**：两个参考系得到相同的摩擦力 $0.72\,\mathrm{N}$，但解释不同——惯性系说"摩擦力提供向心力"，非惯性系说"摩擦力平衡惯性离心力"。

### 例题 2：科里奥利力与河流侵蚀

北半球一条南北走向的河流，水从北向南流动，流速 $v=2\,\mathrm{m/s}$。当地纬度 $\varphi = 40^\circ$。求每千克河水受到的科里奥利力的大小和方向，并分析对河岸的影响。

**解**：

地球自转角速度 $\omega = \frac{2\pi}{T} = \frac{2\pi}{86400} = 7.27\times10^{-5}\,\mathrm{rad/s}$

在北纬 $40^\circ$ 处，地球自转的角速度矢量在竖直方向的分量为 $\omega\sin\varphi$。

科里奥利力 $F_{\text{科}} = -2m\boldsymbol{\omega} \times \mathbf{v}$。对向南流动的水（北向南）：

$F_{\text{科}}/m = 2\omega v\sin\varphi = 2\times7.27\times10^{-5}\times2\times\sin40^\circ$

$= 2.91\times10^{-4}\times0.643 = 1.87\times10^{-4}\,\mathrm{N/kg}$

方向——北半球指向运动方向的**右侧**，即向西（河水的右岸）。

**影响**：虽然每千克水仅受约 $2\times10^{-4}\,\mathrm{N}$ 的力，但长期地质作用下，北半球的河流右岸比左岸更陡峭。这就是**贝尔定律**——由科里奥利力引起的河流不对称侵蚀。

## 6. 常见误区与检查点

| 误区 | 正确理解 |
|------|---------|
| "惯性离心力是向心力的反作用力" | 向心力是真实力（或合力），惯性离心力是假想力，两者不是作用力与反作用力 |
| "在非惯性系中完全不能用牛顿定律" | 引入惯性力后牛顿定律形式上成立 |
| "惯性力与真实力无法区分" | 这正是爱因斯坦等效原理的核心——均匀引力场与加速参考系等价 |
| "科里奥利力很小可以忽略" | 对日常小尺度运动可忽略，对大尺度（台风、洋流、弹道导弹）必须考虑 |

## 练习题

### 练习1（基础巩固）

电梯以 $a = 3\,\mathrm{m/s^2}$ 减速下降。$m = 2\,\mathrm{kg}$ 的物体放在电梯地板上。

(1) 用惯性系方法求物体对地板的压力
(2) 用非惯性系方法（以电梯为参考系）求解同样问题
(3) 这是超重还是失重？为什么？

<details>
<summary><b>参考答案</b></summary>

**(1)** 惯性系（取向上为正）：减速下降即加速度向上 $a=3\,\mathrm{m/s^2}$。

$N - mg = ma$

$N = m(g+a) = 2\times(9.8+3) = 2\times12.8 = 25.6\,\mathrm{N}$

**(2)** 非惯性系（以电梯为参考系）：电梯加速度 $a=3\,\mathrm{m/s^2}$ 向上，引入向下的惯性力 $F_{\text{惯}} = ma = 6\,\mathrm{N}$。

在电梯系中物体静止（受力平衡）：

$N - mg - F_{\text{惯}} = 0$

$N = mg + ma = 19.6 + 6 = 25.6\,\mathrm{N}$

**(3)** 超重（$N > mg$）。减速下降时加速度向上，视重大于实重。口诀：加速上升和减速下降都是超重。

</details>

### 练习2（迁移应用）

为什么赤道处测得的重力加速度小于两极处？定量计算差异的大小。

<details>
<summary><b>参考答案</b></summary>

以地面为参考系（非惯性系），地球自转使赤道上的物体受到向外的惯性离心力。

$F_{\text{离}} = m\omega^2 R$，其中 $\omega = 7.27\times10^{-5}\,\mathrm{rad/s}$，$R = 6.37\times10^6\,\mathrm{m}$。

单位质量受到的惯性离心力（即"丢失"的重力加速度）：

$a_{\text{离}} = \omega^2 R = (7.27\times10^{-5})^2 \times 6.37\times10^6$

$= 5.29\times10^{-9} \times 6.37\times10^6 = 0.0337\,\mathrm{m/s^2}$

此外，地球并非完美球体而是椭球——赤道半径比两极半径大约 $21\,\mathrm{km}$，这使赤道处的 $g$ 又减小约 $0.018\,\mathrm{m/s^2}$。

**总差异**：$g_{\text{两极}} - g_{\text{赤道}} \approx 0.052\,\mathrm{m/s^2}$，与实测值一致（$9.832$ vs $9.780$）。

**生活感受**：在赤道用弹簧秤称 $60\,\mathrm{kg}$ 的物体，比在两极轻约 $0.3\,\mathrm{N}$——微乎其微，无法察觉。

</details>

### 等效原理——惯性力与引力的统一

爱因斯坦在 1907 年提出了等效原理：**在局部区域内，一个均匀引力场与一个加速参考系在物理上无法区分**。这是广义相对论的出发点。

想象一个密封的火箭舱：若在太空中以 $g$ 加速上升，舱内人会感觉自己被"拉"向地板——与地球表面的感受完全一样。引力和惯性力在局部区域不可区分。

惯性质量（出现在 $F=ma$ 中）与引力质量（出现在 $F=GmM/r^2$ 中）在经典力学中含义不同，但实验表明它们严格相等——精度达到 $10^{-12}$ 量级。这正是等效原理的实验基础，也是爱因斯坦提出广义相对论的关键线索。

### 非惯性系中的能量问题

在非惯性系中，惯性力可以做功，因此机械能守恒定律需要修正。在加速上升的电梯中，引入惯性力后，电梯系中的"有效重力势能"变为 $m(g+a)h$，这提供了在非惯性系中分析能量问题的框架。
