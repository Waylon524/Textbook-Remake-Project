## 2. 背景与应用场景

汽车的仪表盘上显示的是**瞬时速度**，而导航软件计算的"平均速度"是根据总路程除以总时间得出的。两者完全不同——你在高速上以 120 km/h 巡航时（瞬时速度），算上休息时间后的平均速度可能只有 80 km/h。理解二者的区别在生活中就有实用价值。

> **直觉理解**：速度就是"位置变化的快慢"，加速度就是"速度变化的快慢"。开过车的人都知道：踩油门（加速度为正）车加速，踩刹车（加速度为负）车减速。

实际应用场景：
- **列车运行控制**：加速度过大乘客会不舒服，高铁的加速度控制在约 $0.5\,\mathrm{m/s^2}$ 以内
- **安全气囊触发**：通过加速度传感器检测到剧烈减速时触发气囊
- **航天器交会对接**：需要精确控制相对速度和加速度

## 3. 核心概念与符号约定

### 平均速度与瞬时速度

平均速度描述一段时间内的整体运动趋势：

$$
\bar{\mathbf{v}} = \frac{\Delta\mathbf{r}}{\Delta t}
$$

瞬时速度描述某一时刻的运动状态，是平均速度在 $\Delta t\to 0$ 时的极限：

$$
\mathbf{v}(t) = \lim_{\Delta t\to 0} \frac{\Delta\mathbf{r}}{\Delta t} = \frac{\mathrm{d}\mathbf{r}}{\mathrm{d}t}
$$

> **关键区别**：平均速度依赖于时间段的选择，瞬时速度是时刻的函数。

### 速度与速率

- **速度** $\mathbf{v}$：矢量，有大小和方向
- **速率** $v = |\mathbf{v}|$：标量，等于速度的大小

注意：平均速度的大小 $|\bar{\mathbf{v}}|$ **一般不等于**平均速率 $\bar{v}=\Delta s/\Delta t$。

### 加速度

加速度是速度随时间的变化率：

$$
\mathbf{a}(t) = \frac{\mathrm{d}\mathbf{v}}{\mathrm{d}t} = \frac{\mathrm{d}^2\mathbf{r}}{\mathrm{d}t^2}
$$

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $\mathbf{v}$ | 速度（矢量） | $\mathrm{m/s}$ |
| $v$ | 速率（标量） | $\mathrm{m/s}$ |
| $\bar{\mathbf{v}}$ | 平均速度 | $\mathrm{m/s}$ |
| $\mathbf{a}$ | 加速度 | $\mathrm{m/s^2}$ |
| $\bar{a}$ | 平均加速度 | $\mathrm{m/s^2}$ |

## 4. 原理与方法

### 由位置求速度——微分法

已知运动学方程 $\mathbf{r}(t)$，对时间求一阶导得速度，求二阶导得加速度：

$$
\mathbf{v}(t) = \frac{\mathrm{d}\mathbf{r}}{\mathrm{d}t},\qquad
\mathbf{a}(t) = \frac{\mathrm{d}\mathbf{v}}{\mathrm{d}t} = \frac{\mathrm{d}^2\mathbf{r}}{\mathrm{d}t^2}
$$

在直角坐标系中，各分量独立求导：

$$
v_x = \frac{\mathrm{d}x}{\mathrm{d}t},\quad v_y = \frac{\mathrm{d}y}{\mathrm{d}t},\quad v_z = \frac{\mathrm{d}z}{\mathrm{d}t}
$$

### 由加速度求位置——积分法

已知加速度 $\mathbf{a}(t)$ 和初始条件 $\mathbf{r}_0, \mathbf{v}_0$，逐次积分：

$$
\mathbf{v}(t) = \mathbf{v}_0 + \int_0^t \mathbf{a}(t)\,\mathrm{d}t
$$

$$
\mathbf{r}(t) = \mathbf{r}_0 + \int_0^t \mathbf{v}(t)\,\mathrm{d}t
$$

### 直线运动的特殊公式

匀变速直线运动（$a = \text{常量}$）：

$$
v = v_0 + at,\quad x = x_0 + v_0t + \frac12 at^2,\quad v^2 - v_0^2 = 2a(x-x_0)
$$

### 适用条件

- 微分法：$\mathbf{r}(t)$ 必须可导
- 积分法：$\mathbf{a}(t)$ 可积，且初始条件已知
- 匀变速直线运动公式仅适用于加速度为常量的情形

## 5. 例题

### 例题 1：速度与加速度的微分计算

质点的运动学方程为 $\mathbf{r}(t) = (t^3)\,\mathbf{i} + (4t)\,\mathbf{j} + (2t^2)\,\mathbf{k}$（SI 单位）。求 $t=2\,\mathrm{s}$ 时的速度和加速度。

**解**：

对各分量求导：

$$
v_x = \frac{\mathrm{d}x}{\mathrm{d}t} = 3t^2,\quad
v_y = \frac{\mathrm{d}y}{\mathrm{d}t} = 4,\quad
v_z = \frac{\mathrm{d}z}{\mathrm{d}t} = 4t
$$

$t=2\,\mathrm{s}$ 时：

$$
\mathbf{v}(2) = 12\,\mathbf{i} + 4\,\mathbf{j} + 8\,\mathbf{k}\;(\mathrm{m/s})
$$

速率 $v(2) = \sqrt{144+16+64} = \sqrt{224} \approx 14.97\,\mathrm{m/s}$。

再求加速度：

$$
a_x = \frac{\mathrm{d}^2x}{\mathrm{d}t^2} = 6t,\quad
a_y = 0,\quad
a_z = 4
$$

$t=2\,\mathrm{s}$ 时 $\mathbf{a}(2) = 12\,\mathbf{i} + 4\,\mathbf{k}\;(\mathrm{m/s^2})$。

**一致性检查**：量纲 $[v]=\mathrm{m/s}$，$[a]=\mathrm{m/s^2}$，正确。

### 例题 2：匀变速直线运动

一辆汽车以 $v_0 = 20\,\mathrm{m/s}$ 行驶，司机发现前方 $50\,\mathrm{m}$ 处有障碍物后紧急刹车。汽车刹车时的加速度为 $a = -5\,\mathrm{m/s^2}$。

(1) 汽车是否会在撞到障碍物前停下？
(2) 从开始刹车到停下，汽车行驶了多少米？

**解**：

(1) 先求停车时间：$v = v_0 + at = 0 \Rightarrow t = \frac{-v_0}{a} = \frac{-20}{-5} = 4\,\mathrm{s}$。

(2) 求刹车距离：

$$
x = v_0 t + \frac12 a t^2 = 20\times4 + \frac12\times(-5)\times16 = 80 - 40 = 40\,\mathrm{m}
$$

或者用公式 $v^2 - v_0^2 = 2ax$ 直接求：

$$
0 - 20^2 = 2\times(-5)\times x \Rightarrow x = \frac{-400}{-10} = 40\,\mathrm{m}
$$

汽车在 $40\,\mathrm{m}$ 处停下，距离障碍物还有 $10\,\mathrm{m}$，**不会撞上**。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|------|---------|
| "平均速度的大小就是平均速率" | 只有当质点沿直线单向运动时才相等 |
| "加速度为负就是减速" | 加速度与速度方向相反时才减速，负号取决于坐标正向的选择 |
| "速度大则加速度一定大" | 匀速运动速度很大但加速度为零 |
| "$v = \frac{s}{t}$ 适用于一切运动" | 这只适合匀速直线运动 |

### 检查点

- [ ] 能否正确求导得到速度和加速度？
- [ ] 能否从 $a(t)$ 和初始条件积分得到运动学方程？
- [ ] 是否理解"瞬时"的含义（极限过程）？
- [ ] 匀变速公式的适用条件是什么？

## 练习题

### 练习1（基础巩固）

一质点沿 $x$ 轴运动，位置随时间的变化为 $x(t) = 2t^3 - 9t^2 + 12t + 1$（SI 单位）。

(1) 求速度函数 $v(t)$ 和加速度函数 $a(t)$
(2) 求质点速度为零的时刻及对应位置
(3) 分析质点的运动状态（何时加速、何时减速）

<details>
<summary><b>参考答案</b></summary>

**练习1(1)**：由速度定义 $v(t) = \frac{\mathrm{d}x}{\mathrm{d}t}$，对 $x(t)$ 求导：

$v(t) = \frac{\mathrm{d}}{\mathrm{d}t}(2t^3 - 9t^2 + 12t + 1) = 6t^2 - 18t + 12 = 6(t^2 - 3t + 2) = 6(t-1)(t-2)$

由加速度定义 $a(t) = \frac{\mathrm{d}v}{\mathrm{d}t}$，对 $v(t)$ 求导：

$a(t) = \frac{\mathrm{d}}{\mathrm{d}t}(6t^2 - 18t + 12) = 12t - 18 = 6(2t-3)$

**练习1(2)**：令 $v(t)=0$，即 $6(t-1)(t-2)=0$，解得 $t_1=1\,\mathrm{s}$，$t_2=2\,\mathrm{s}$。

代入 $x(t)$：
- $t=1$：$x = 2(1)^3 - 9(1)^2 + 12(1) + 1 = 2 - 9 + 12 + 1 = 6\,\mathrm{m}$
- $t=2$：$x = 2(8) - 9(4) + 12(2) + 1 = 16 - 36 + 24 + 1 = 5\,\mathrm{m}$

**练习1(3)**：根据 $v$ 和 $a$ 的符号关系判断加速/减速：

- $t<1\,\mathrm{s}$（如 $t=0$）：$v(0)=12>0$，$a(0)=-18<0$，$v$ 与 $a$ 异号 → **减速**，质点向 $x$ 正方向运动但速度在减小

- $1<t<2\,\mathrm{s}$（如 $t=1.5$）：$v(1.5)=6(0.5)(-0.5)=-1.5<0$，$a(1.5)=12(1.5)-18=0$→（实际 $a=0$ 是转折点）在 $1<t<1.5$ 时 $a<0$，$v<0$，$v$ 与 $a$ 同号 → **加速**，质点向 $x$ 负方向运动且速度增大

- $t>2\,\mathrm{s}$（如 $t=3$）：$v(3)=6(2)(1)=12>0$，$a(3)=36-18=18>0$，$v$ 与 $a$ 同号 → **加速**，质点向 $x$ 正方向运动且速度增大

</details>

### 练习2（迁移应用）

一架飞机在跑道上从静止开始匀加速滑行，$20\,\mathrm{s}$ 后速度达到 $80\,\mathrm{m/s}$ 时起飞。

(1) 求滑行加速度
(2) 求跑道至少需要多长
(3) 如果跑道只有 $600\,\mathrm{m}$，飞机需要多大的加速度才能起飞？（假设仍用 $20\,\mathrm{s}$ 加速到 $80\,\mathrm{m/s}$）

<details>
<summary><b>参考答案</b></summary>

**练习2(1)**：已知 $v_0=0$，$v=80\,\mathrm{m/s}$，$t=20\,\mathrm{s}$。由匀加速直线运动公式 $v = v_0 + at$：

$a = \frac{v - v_0}{t} = \frac{80 - 0}{20} = 4\,\mathrm{m/s^2}$

加速度为 $4\,\mathrm{m/s^2}$，相当于 $0.41g$，是一个中等强度的加速感。

**练习2(2)**：由匀加速位移公式 $x = v_0t + \frac12 at^2$：

$x = 0\times20 + \frac12 \times 4 \times 20^2 = \frac12 \times 4 \times 400 = 800\,\mathrm{m}$

机场跑道至少需要 $800\,\mathrm{m}$。实际上机场跑道通常在 $2000\!-\!3000\,\mathrm{m}$ 左右，这是因为大型飞机速度要求更高且考虑安全余量。

**练习2(3)**：已知 $x=600\,\mathrm{m}$，$t=20\,\mathrm{s}$，$v_0=0$。由 $x = v_0t + \frac12 at^2$ 反推加速度：

$a = \frac{2x}{t^2} = \frac{2\times600}{20^2} = \frac{1200}{400} = 3\,\mathrm{m/s^2}$

此时 $t=20\,\mathrm{s}$ 时的速度 $v = at = 3\times20 = 60\,\mathrm{m/s}$，远低于起飞所需速度 $80\,\mathrm{m/s}$。

**结论**：在 $600\,\mathrm{m}$ 跑道上仅用 $3\,\mathrm{m/s^2}$ 的加速度无法达到 $80\,\mathrm{m/s}$ 的起飞速度。实际需要的加速度为 $a = v^2/(2x) = 80^2/(2\times600) = 6400/1200 \approx 5.33\,\mathrm{m/s^2}$，或者需要更长的跑道。

</details>
