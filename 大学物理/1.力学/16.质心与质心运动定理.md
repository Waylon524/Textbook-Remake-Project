## 2. 背景与应用场景

跳水运动员从跳板跃起后，在空中做出转体、翻腾、抱膝等各种复杂动作——身体各部分在剧烈运动，但**质心始终沿着一条抛物线运动**。这是因为运动员在空中的合外力只有重力（忽略空气阻力）。不论他如何收紧或伸展身体，不论他怎样旋转，质心的轨迹不会改变。

> **直觉理解**：质心就是系统所有质量"平均"所在的位置。把系统的总质量想象成集中在一点上，这一点就代表了系统整体的运动。好比一锅汤——不用逐一分析每种食材的位置，只看"整体的重心"就够了。

工程师在设计汽车时，会计算整车的质心位置。质心越低、越靠近几何中心，汽车的操控性和稳定性就越好。SUV 比轿车更容易侧翻的根本原因就是质心更高。

## 3. 核心概念

### 质心的定义

**离散质点系**：

$$
\mathbf{r}_C = \frac{\sum_{i=1}^{n} m_i\mathbf{r}_i}{\sum_{i=1}^{n} m_i}
$$

**连续体**：

$$
\mathbf{r}_C = \frac{\int \mathbf{r}\,\mathrm{d}m}{\int \mathrm{d}m} = \frac{\int \mathbf{r}\rho\,\mathrm{d}V}{\int \rho\,\mathrm{d}V}
$$

### 质心的速度与加速度

对离散点系求导：

$$
\mathbf{v}_C = \frac{\sum m_i\mathbf{v}_i}{M},\qquad
\mathbf{a}_C = \frac{\sum m_i\mathbf{a}_i}{M}
$$

### 质心运动定理

$$
\sum\mathbf{F}_{\text{外}} = M\mathbf{a}_C
$$

**深刻含义**：系统质心的运动完全等同于这样一个质点的运动——该质点的质量等于系统的总质量，所受的力等于系统所受的合外力。系统内部各物体如何相互作用（碰撞、摩擦、弹簧连接……），都不影响质心的运动。

**推论**：
- 合外力为零时，质心保持匀速直线运动或静止
- 合外力在某个方向的分量为零时，质心在该方向做匀速运动

### 质心与重心的区别

| 概念 | 质心 | 重心 |
|------|------|------|
| 定义 | 质量分布的中心 | 重力合力的作用点 |
| 公式 | $\frac{\sum m_i\mathbf{r}_i}{\sum m_i}$ | $\frac{\sum m_i g_i\mathbf{r}_i}{\sum m_i g_i}$ |
| 依赖重力场 | 不依赖 | 依赖（$g$ 随位置变化） |
| 均匀重力场中 | 与重心重合 | 与质心重合 |

在均匀重力场（地球表面附近）中，两者重合。但在非均匀重力场中（如卫星在大尺度空间中的运动），两者不重合。

## 4. 原理与方法

### 质心运动定理的证明

对质点系的总动量 $\mathbf{P} = \sum m_i\mathbf{v}_i$ 求时间导数：

$$\frac{\mathrm{d}\mathbf{P}}{\mathrm{d}t} = \sum m_i\mathbf{a}_i = \sum\mathbf{F}_i^{(\text{外})} + \sum\mathbf{F}_i^{(\text{内})}$$

由牛顿第三定律，内力和两两抵消：$\sum\mathbf{F}_i^{(\text{内})} = 0$

又 $\mathbf{P} = M\mathbf{v}_C$，所以 $\frac{\mathrm{d}\mathbf{P}}{\mathrm{d}t} = M\frac{\mathrm{d}\mathbf{v}_C}{\mathrm{d}t} = M\mathbf{a}_C$

因此 $M\mathbf{a}_C = \sum\mathbf{F}_{\text{外}}$，得证。

### 质心系

以质心为原点、坐标轴方向与惯性系平行的参考系称为**质心系**。

质心系的重要性质：
1. 在质心系中，系统的总动量始终为零——$\sum m_i\mathbf{v}_i' = 0$
2. 质心系可能不是惯性系（质心加速时），需引入惯性力
3. 在质心系中分析两体碰撞问题特别方便——两物体以等大反向的动量相互接近

### 常见形状的质心位置

| 形状 | 质心位置 |
|------|---------|
| 均匀细棒 | 中点 |
| 均匀三角形 | 三条中线的交点（距底边 $1/3$ 高） |
| 均匀圆环 | 环心（在物体外部） |
| 均匀圆盘 | 圆心 |
| 均匀球体 | 球心 |
| 均匀半球体 | 距球心 $3R/8$ 处 |

## 5. 例题

### 例题 1：两质点的质心及运动

$m_1 = 2\,\mathrm{kg}$ 在原点 $(0,0)$，$m_2 = 3\,\mathrm{kg}$ 在 $(4,0)\,\mathrm{m}$。

(1) 求系统质心位置
(2) 若 $m_1$ 受 $F = 10\,\mathrm{N}$ 水平向右的恒力，$m_2$ 不受力，求质心加速度
(3) 如果 $m_1$ 和 $m_2$ 用轻弹簧连接，上述质心加速度是否变化？

**解**：

**(1)** $x_C = \frac{m_1x_1 + m_2x_2}{m_1+m_2} = \frac{2\times0 + 3\times4}{5} = \frac{12}{5} = 2.4\,\mathrm{m}$

质心在 $x = 2.4\,\mathrm{m}$ 处——更靠近质量较大的 $m_2$。

**(2)** 系统受合外力 $F = 10\,\mathrm{N}$。由质心运动定理：

$a_C = \frac{F}{M} = \frac{10}{5} = 2\,\mathrm{m/s^2}$（向右）

**(3)** **不变化**。因为质心运动定理只依赖于合外力，与内力（弹簧力）无关。无论 $m_1$ 和 $m_2$ 之间通过弹簧如何运动，质心的加速度都是 $2\,\mathrm{m/s^2}$ 向右。

### 例题 2：人和船

质量为 $M = 150\,\mathrm{kg}$ 的船长 $L = 4\,\mathrm{m}$，静止在平静湖面上。质量为 $m = 50\,\mathrm{kg}$ 的人从船头走到船尾。不计水的阻力。

(1) 求人和船相对地面的位移
(2) 人相对地面走了多远？

**解**：

**建模**：人+船系统水平方向不受外力，质心位置保持不变。取船初始位置前端为坐标系原点，$x$ 轴指向船头为正。初始时人在船头（$x=0$ 处），船质心在 $x = L/2$ 处。

设人走到船尾后，船向右移动了 $s$，则船的新位置范围从 $s$ 到 $s+L$，船质心在 $s+L/2$，人在 $s+L$ 处。

初始质心位置：

$$
x_{C0} = \frac{M\cdot\frac{L}{2} + m\cdot0}{M+m} = \frac{150\times2}{200} = 1.5\,\mathrm{m}
$$

末态质心位置：

$$
x_C = \frac{M\cdot(s+\frac{L}{2}) + m\cdot(s+L)}{M+m} = \frac{150(s+2) + 50(s+4)}{200}
$$

由 $x_C = x_{C0}$：

$150(s+2) + 50(s+4) = 1.5\times200 = 300$

$150s + 300 + 50s + 200 = 300$

$200s + 500 = 300$

$200s = -200$，$s = -1\,\mathrm{m}$

船相对地面向左移动 $1\,\mathrm{m}$。

人相对地面的位移：$x_{\text{人}} = s + L = -1 + 4 = 3\,\mathrm{m}$（向左）

**验证**：人在船上走了 $4\,\mathrm{m}$，但相对地面只走了 $3\,\mathrm{m}$，因为船向反方向移动了 $1\,\mathrm{m}$。

### 例题 3：爆炸后质心的运动

一个质量为 $M$ 的炮弹以初速度 $v_0$、仰角 $\theta$ 发射。在最高点爆炸为质量相等的两片，一片竖直下落，另一片水平飞出。

**分析**：爆炸前后合外力均为重力，质心轨迹不变——仍为原抛物线。无论碎片如何飞散，系统质心的运动不受爆炸影响。

## 6. 常见误区与检查点

| 误区 | 正确理解 |
|------|---------|
| "质心一定在物体内部" | 质心可以在物体外部——圆环的质心在环心（空腔处） |
| "质心与重心总是重合" | 均匀重力场中重合；非均匀重力场中不重合 |
| "内力能改变质心的运动" | 内力可以改变各部分的运动，**不能**改变质心的运动 |
| "质心就是密度最大的点" | 质心是质量加权平均位置，不一定在密度最大的点 |
| "质心运动定理太理论化没用" | 恰恰相反——它把多体复杂运动简化为单个质点的运动 |

## 练习题

### 练习1（基础巩固）

两个质点 $m_1 = 1\,\mathrm{kg}$ 在 $(0,0)$，$m_2 = 2\,\mathrm{kg}$ 在 $(3,4)\,\mathrm{m}$。

(1) 求系统质心坐标
(2) 如果 $m_1$ 增加到 $3\,\mathrm{kg}$，质心向哪个方向移动？

<details>
<summary><b>参考答案</b></summary>

**(1)** $x_C = \frac{1\times0 + 2\times3}{1+2} = \frac{6}{3} = 2\,\mathrm{m}$

$y_C = \frac{1\times0 + 2\times4}{3} = \frac{8}{3} \approx 2.67\,\mathrm{m}$

**(2)** $m_1=3\,\mathrm{kg}$：$x_C' = \frac{3\times0 + 2\times3}{5} = \frac{6}{5} = 1.2\,\mathrm{m}$

$y_C' = \frac{3\times0 + 2\times4}{5} = \frac{8}{5} = 1.6\,\mathrm{m}$

质心从 $(2.00,2.67)$ 移动到 $(1.20,1.60)$——向 $m_1$ 方向移动，因为 $m_1$ 的质量增大。
