## 2. 背景与应用场景

运动学两类问题是联系"运动描述"和"运动预测"的桥梁。

- **第一类问题（求导）**：雷达测得飞机位置数据 $\mathbf{r}(t)$，通过求导实时计算速度和加速度，用于航迹追踪
- **第二类问题（积分）**：火箭的加速度由推力决定，通过积分预测其飞行轨迹

> **直觉理解**：第一类是"已知位置历史，问现在多快"；第二类是"已知加速能力，问未来在哪"。前者是微分（看变化率），后者是积分（积累变化）。

---

## 3. 核心概念与符号约定

### 第一类问题（微分法）

已知运动学方程 $\mathbf{r} = \mathbf{r}(t)$，求导得速度和加速度：

$$
\mathbf{v}(t) = \frac{\mathrm{d}\mathbf{r}}{\mathrm{d}t},\quad
\mathbf{a}(t) = \frac{\mathrm{d}\mathbf{v}}{\mathrm{d}t} = \frac{\mathrm{d}^2\mathbf{r}}{\mathrm{d}t^2}
$$

### 第二类问题（积分法）

已知加速度 $\mathbf{a}(t)$ 和初始条件 $\mathbf{r}_0, \mathbf{v}_0$，积分求速度和运动学方程：

$$
\mathbf{v}(t) = \mathbf{v}_0 + \int_{0}^{t} \mathbf{a}(t)\,\mathrm{d}t
$$

$$
\mathbf{r}(t) = \mathbf{r}_0 + \int_{0}^{t} \mathbf{v}(t)\,\mathrm{d}t
$$

### 初始条件的重要性

第二类问题中，积分常数由初始条件确定——**不同的初始条件导致不同的运动**，即使加速度相同。

### 符号约定

| 符号 | 含义 | SI 单位 |
|------|------|--------|
| $\mathbf{r}(t)$ | 运动学方程 | $\mathrm{m}$ |
| $\mathbf{v}(t)$ | 速度函数 | $\mathrm{m/s}$ |
| $\mathbf{a}(t)$ | 加速度函数 | $\mathrm{m/s^2}$ |
| $\mathbf{r}_0, \mathbf{v}_0$ | 初始位置和速度 | $\mathrm{m},\mathrm{m/s}$ |

---

## 4. 原理与方法

### 第一类问题的标准流程

```
位置   r(t)  ──求一阶导──→  速度 v(t)  ──求一阶导──→  加速度 a(t)
```

例：$x(t) = 3t^2 - 2t + 1 \xrightarrow{\frac{\mathrm{d}}{\mathrm{d}t}} v(t) = 6t - 2 \xrightarrow{\frac{\mathrm{d}}{\mathrm{d}t}} a(t) = 6$

### 第二类问题的标准流程

```
加速度 a(t)  ──积分──→  速度 v(t)  ──积分──→  位置 x(t)
    ↑                     ↑
  已知               初始条件 v₀,x₀
```

### 三类常见加速度的处理

| 加速度形式 | 处理方法 | 示例 |
|-----------|---------|------|
| $a = a(t)$ | 直接对时间积分 | $a = 6t$，$v = v_0 + 3t^2$ |
| $a = a(v)$ | 分离变量：$\frac{\mathrm{d}v}{a(v)} = \mathrm{d}t$ | $a = -kv$，$v = v_0 e^{-kt}$ |
| $a = a(x)$ | 利用 $a = v\frac{\mathrm{d}v}{\mathrm{d}x}$，分离变量：$v\mathrm{d}v = a(x)\mathrm{d}x$ | $a = -kx$，$v^2 = v_0^2 - kx^2$ |

### 适用条件

- $\mathbf{r}(t)$ 二阶可导（第一类）
- $\mathbf{a}(t)$ 可积、初始条件已知（第二类）
- 变量替换积分时注意积分上下限的对应关系

## 5. 例题

### 例题 1：第一类问题（微分）

质点的运动学方程为 $\mathbf{r}(t) = (6t^2 - t^3)\,\mathbf{i} + (4t)\,\mathbf{j}$（SI 单位）。求 $t=2\,\mathrm{s}$ 时的速度和加速度，并分析运动特征。

**解**：

**建模**：二维运动，$x$ 和 $y$ 方向独立分析。

**列式**：

$x$ 方向：$x = 6t^2 - t^3$，$v_x = 12t - 3t^2$，$a_x = 12 - 6t$

$y$ 方向：$y = 4t$，$v_y = 4$，$a_y = 0$

**计算**：

$t=2\,\mathrm{s}$：

$v_x = 24 - 12 = 12\,\mathrm{m/s}$，$v_y = 4\,\mathrm{m/s}$

$\mathbf{v} = 12\,\mathbf{i} + 4\,\mathbf{j}\;(\mathrm{m/s})$，$v = \sqrt{144+16} = \sqrt{160} \approx 12.65\,\mathrm{m/s}$

$a_x = 12 - 12 = 0$，$a_y = 0$，$\mathbf{a} = 0$

**结论**：$t=2\,\mathrm{s}$ 时加速度恰为零，这是速度变化的转折点——之前减速，之后加速。

### 例题 2：第二类问题（积分）

一质点沿 $x$ 轴运动，加速度 $a = 4t - 6$（SI 单位）。$t=0$ 时 $x_0 = 0$，$v_0 = 2\,\mathrm{m/s}$。求 $t=3\,\mathrm{s}$ 时的速度和位置。

**解**：

**建模**：已知 $a(t)$ 和初始条件，逐次积分。

**列式并计算**：

速度：$v(t) = v_0 + \int_0^t a(t)\,\mathrm{d}t = 2 + \int_0^t (4t - 6)\,\mathrm{d}t$

$= 2 + [2t^2 - 6t]_0^t = 2 + 2t^2 - 6t$

$v(t) = 2t^2 - 6t + 2$

$t=3\,\mathrm{s}$：$v(3) = 18 - 18 + 2 = 2\,\mathrm{m/s}$

位置：$x(t) = x_0 + \int_0^t v(t)\,\mathrm{d}t = 0 + \int_0^t (2t^2 - 6t + 2)\,\mathrm{d}t$

$= \left[\frac{2}{3}t^3 - 3t^2 + 2t\right]_0^t = \frac{2}{3}t^3 - 3t^2 + 2t$

$x(t) = \frac{2}{3}t^3 - 3t^2 + 2t$

$t=3\,\mathrm{s}$：$x(3) = \frac{2}{3}\times27 - 27 + 6 = 18 - 27 + 6 = -3\,\mathrm{m}$

**验证**：$v(3) = 2\,\mathrm{m/s}$，与初速度相同，但质点已经过了一个来回运动（从 $x=0$ 到正方向又回到负方向）。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|------|---------|
| "已知 $a$ 求 $v$ 不用加常数" | 不定积分必须加积分常数，由初始条件 $v_0$ 确定 |
| "第二类问题一定可以解析积分" | 只有少数简单形式的 $a(t)$ 可解析积分，复杂情况需数值方法 |
| "$v(t)$ 为零时质点一定在原点" | $v=0$ 是速度为零，位置由 $x(t)$ 决定，不一定在原点 |
| "只要加速度为负就是减速" | 加速度为负且速度也为负时是加速 |

### 检查点

- [ ] 第一类问题：能否熟练求导？
- [ ] 第二类问题：能否正确确定积分常数？
- [ ] 能否处理 $a = a(v)$ 形式的分离变量积分？
- [ ] 求解后是否检查了量纲？（位置 m，速度 m/s，加速度 m/s²）

## 练习题

### 练习1（基础巩固）

一质点沿 $x$ 轴运动，加速度 $a = -2v$（SI 单位，$v$ 为瞬时速度）。$t=0$ 时 $x_0 = 0$，$v_0 = 10\,\mathrm{m/s}$。

(1) 求速度函数 $v(t)$
(2) 求运动学方程 $x(t)$
(3) 求质点最终的位置（$t \to \infty$ 时的 $x$）

<details>
<summary><b>参考答案</b></summary>

**练习1(1)**：已知 $a = -2v$，属于第二类问题中 $a = a(v)$ 的形式。由 $a = \frac{\mathrm{d}v}{\mathrm{d}t}$ 得：

$\frac{\mathrm{d}v}{\mathrm{d}t} = -2v$

分离变量：$\frac{\mathrm{d}v}{v} = -2\,\mathrm{d}t$

两边积分：$\int_{v_0}^{v} \frac{\mathrm{d}v}{v} = \int_0^t -2\,\mathrm{d}t$

$[\ln v]_{v_0}^{v} = -2t$

$\ln\frac{v}{v_0} = -2t$

$v(t) = v_0 e^{-2t} = 10 e^{-2t}\;(\mathrm{m/s})$

速度随时间指数衰减，$t \to \infty$ 时 $v \to 0$。

**练习1(2)**：由 $v = \frac{\mathrm{d}x}{\mathrm{d}t}$，对速度积分求位置：

$x(t) = x_0 + \int_0^t v(t)\,\mathrm{d}t = 0 + \int_0^t 10 e^{-2t}\,\mathrm{d}t$

$= 10 \times \left[-\frac{1}{2}e^{-2t}\right]_0^t = 10 \times \left(-\frac{1}{2}e^{-2t} + \frac{1}{2}\right)$

$x(t) = 5(1 - e^{-2t})\;(\mathrm{m})$

**练习1(3)**：$t \to \infty$ 时，$e^{-2t} \to 0$，所以 $x(\infty) = 5\,\mathrm{m}$。

**结论**：质点在阻尼作用下速度趋于零，最终停在 $x=5\,\mathrm{m}$ 处。这是一个典型的"阻尼运动"模型——如粘性流体中物体在初始推动后的运动。

</details>

### 练习2（迁移应用）

一质点加速度与位移的关系为 $a = -4x$（SI 单位）。$t=0$ 时 $x_0 = 2\,\mathrm{m}$，$v_0 = 0$。

(1) 利用 $a = v\frac{\mathrm{d}v}{\mathrm{d}x}$ 求 $v(x)$ 关系式
(2) 求 $x=0$ 处的速度
(3) 已知该运动为简谐振动，求运动学方程 $x(t)$

<details>
<summary><b>参考答案</b></summary>

**练习2(1)**：已知 $a = -4x$，且 $a = v\frac{\mathrm{d}v}{\mathrm{d}x}$，所以：

$v\frac{\mathrm{d}v}{\mathrm{d}x} = -4x$

分离变量：$v\,\mathrm{d}v = -4x\,\mathrm{d}x$

两边积分：$\int_{v_0}^{v} v\,\mathrm{d}v = \int_{x_0}^{x} -4x\,\mathrm{d}x$

$\frac12 v^2 - \frac12 v_0^2 = -2(x^2 - x_0^2)$

代入 $v_0 = 0$，$x_0 = 2$：

$\frac12 v^2 = -2(x^2 - 4) = 8 - 2x^2$

$v^2 = 16 - 4x^2$

所以 $v(x) = \pm\sqrt{16 - 4x^2}$

**练习2(2)**：代入 $x=0$，$v(0) = \pm\sqrt{16} = \pm 4\,\mathrm{m/s}$

质点从 $x=2$ 向原点运动时取负号（向左运动），$v = -4\,\mathrm{m/s}$；经过原点后继续向左运动。

**练习2(3)**：已知 $a = -4x$，对比简谐振动微分方程 $a = -\omega^2 x$，得 $\omega^2 = 4$，$\omega = 2\,\mathrm{rad/s}$。

简谐振动的一般形式为 $x(t) = A\cos(\omega t + \varphi)$。代入初始条件：

$t=0$：$x(0) = A\cos\varphi = 2$，$v(0) = -A\omega\sin\varphi = 0$

由 $v(0)=0$ 得 $\sin\varphi=0$，$\varphi=0$。代入 $x(0)=2$ 得 $A=2$。

运动学方程：$x(t) = 2\cos(2t)\;(\mathrm{m})$

验证：求导得 $v(t) = -4\sin(2t)$，再求导得 $a(t) = -8\cos(2t) = -4x(t)$，与已知 $a=-4x$ 一致。

</details>
