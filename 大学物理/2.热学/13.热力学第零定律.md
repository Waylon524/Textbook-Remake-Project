# 13. 热力学第零定律

> 所属切块：02-热学  |  块内序号：13  |  难度：基础

## 2. 背景与应用场景

耳温枪能在 $1$ 秒内测出体温，其原理是：耳膜的红外辐射强度与温度一一对应——这是测温技术中的一个实例。但所有温度计背后的理论基础是同一个：热力学第零定律。

> **直觉理解**：如果你摸了冰水混合物（$0^\circ\mathrm{C}$），又摸了一杯开水（$100^\circ\mathrm{C}$），你立刻知道哪个更冷哪个更热。但你要精确知道温度，就需要一个"第三方"——温度计。温度计放入开水后，它与开水达到热平衡，温度计的读数就是开水的温度。这就是第零定律：A 与 B 热平衡，A 与 C 热平衡，则 B 与 C 也处于热平衡。

第零定律之所以重要，是因为它为"温度"这个基本物理量提供了逻辑基础——没有它，我们就无法定义温度。

---

## 3. 核心概念与符号约定

### 热力学第零定律的正式表述

> 如果两个热力学系统中的每一个都与第三个系统处于热平衡，则它们彼此也处于热平衡。

**数学表达**：若 $A \sim C$ 且 $B \sim C$，则 $A \sim B$（$\sim$ 表示"与……处于热平衡"）。

### 第零定律的逻辑地位

第零定律是热力学的**逻辑起点**——它保证了"温度"概念的合理性。

| 定律 | 提出时间 | 核心贡献 |
|:----|:--------:|:---------|
| 第零定律 | 1930s（命名） | 定义了温度概念的存在性 |
| 第一定律 | 1850s | 能量守恒 |
| 第二定律 | 1850s | 过程方向性 |
| 第三定律 | 1900s | 绝对零度不可达到 |

第零定律在热力学建立后才被认识到其基础地位，因此被编号为"第零"。

### 温度的定义

根据第零定律：**温度是决定一个系统是否与其他系统处于热平衡的物理量**。

- 两个系统处于热平衡 $\iff$ 它们具有相同的温度
- 温度相等是热平衡的**充要条件**

### 温度计的原理

所有温度计都基于第零定律：

1. 温度计与待测物体接触
2. 经过足够时间，温度计与待测物体达到热平衡
3. 温度计内测温物质的某种性质（体积、电阻、辐射等）与温度单值对应
4. 读取该性质的值，换算为温度

### 常用温度计

| 类型 | 测温性质 | 适用范围 |
|:----|:---------|:---------|
| 水银/酒精温度计 | 热胀冷缩 | $-30^\circ\mathrm{C}$ 到 $300^\circ\mathrm{C}$ |
| 铂电阻温度计 | 电阻随温度变化 | $-260^\circ\mathrm{C}$ 到 $660^\circ\mathrm{C}$ |
| 热电偶 | 温差电动势 | $-200^\circ\mathrm{C}$ 到 $1600^\circ\mathrm{C}$ |
| 光学高温计 | 热辐射 | $800^\circ\mathrm{C}$ 以上 |
| 气体温度计 | 定容气体压强 | 很宽范围（最精确） |

## 4. 原理与方法

### 第零定律为什么不是"显然"的

初学者常觉得第零定律是"废话"。但仔细思考以下问题：

- 如果没有第零定律——如果 $A \sim C$ 且 $B \sim C$ 不能推出 $A \sim B$——那么"温度"这个概念就毫无意义
- 因为你会遇到这样的情况：A 和 C 温度相同，B 和 C 温度相同，但 A 和 B 温度不同——这在逻辑上意味着温度不是良定义的
- 第零定律保证了"温度相等"是一种**等价关系**，使温度成为良定义的物理量

### 热平衡的传递性

第零定律的实质是：**热平衡具有传递性**。

这种传递性是建立温度概念的基础：我们不需要把每个物体和其他所有物体接触来比较温度，只需用温度计作为"中介"，分别与被测物体达到热平衡即可。

### 构建温标的逻辑步骤

1. 选定测温物质（如水银）
2. 选定测温性质（如水银柱高度）
3. 选定固定点（如冰点、沸点）
4. 规定温度与测温性质的关系（如线性关系）
5. 利用第零定律进行温度测量

## 5. 例题

### 例题 1：第零定律与温度测量

将一支 $0^\circ\mathrm{C}$ 的体温计放入一个 $37^\circ\mathrm{C}$ 的患者口中。描述热平衡建立的过程，并用第零定律解释为什么足够长时间后体温计的读数就是患者的体温。

**解**：

**过程描述**：
1. 初始：体温计 $0^\circ\mathrm{C}$，患者 $37^\circ\mathrm{C}$——两者未达到热平衡
2. 热传递开始：热量从温度较高的患者口腔传到温度较低的体温计
3. 体温计温度升高，口腔局部温度略微降低
4. 一段时间后，两者温度达到相同——热平衡建立
5. 体温计读数稳定，此时体温计的温度等于患者口腔的温度

**第零定律的应用**：体温计与患者口腔最终处于热平衡状态。由第零定律，如果我们预先校准了体温计（即知道体温计在特定读数下与标准温度源处于热平衡），那么患者口腔也就与标准温度源有相同温度——因此读数就是患者的体温。

**为什么需要足够长时间**：热传导需要时间。水的比热容大，人体组织含大量水分，热容量大，需要足够时间才能使测温元件与人体组织达到热平衡。

### 例题 2：不同温标的校准

一支未经校准的水银温度计，水银柱高度 $h$ 与温度 $t$ 的关系为 $h = a + bt$。冰点读数 $h_0 = 2.0\,\mathrm{cm}$，沸点读数 $h_{100} = 12.0\,\mathrm{cm}$。

(1) 求 $a$ 和 $b$
(2) 高度 $h = 7.0\,\mathrm{cm}$ 对应的温度是多少？

**解**：

**建模**：摄氏温标的两个固定点：冰点 $0^\circ\mathrm{C}$，沸点 $100^\circ\mathrm{C}$。

由 $h = a + bt$：

**列方程**：

$2.0 = a + b \times 0 \Rightarrow a = 2.0\,\mathrm{cm}$

$12.0 = 2.0 + b \times 100 \Rightarrow b = \frac{10.0}{100} = 0.10\,\mathrm{cm/^\circ C}$

**求温度**：

$h = 7.0 = 2.0 + 0.10t \Rightarrow t = \frac{5.0}{0.10} = 50^\circ\mathrm{C}$

**结论**：水银柱高度 $7.0\,\mathrm{cm}$ 对应摄氏温度 $50^\circ\mathrm{C}$。

### 例题 3：温度计的选择

实验室需要测量以下温度，选择什么温度计合适？并说明理由。

(1) $-200^\circ\mathrm{C}$ 的液氮
(2) $1500^\circ\mathrm{C}$ 的钢水
(3) $1000^\circ\mathrm{C}$ 的工业炉（需实时数据传输）

**解**：

| 场景 | 推荐温度计 | 理由 |
|:----|:----------|:------|
| (1) $-200^\circ\mathrm{C}$ 液氮 | 铂电阻温度计 | 水银/酒精温度计在低温下凝固，铂电阻在低温范围线性好、精度高 |
| (2) $1500^\circ\mathrm{C}$ 钢水 | 光学高温计 | 钢水温度远超接触式温度计的量程，利用热辐射测温，不需接触 |
| (3) $1000^\circ\mathrm{C}$ 工业炉 | 热电偶 | 热电偶结构简单、耐高温、信号适合远距离传输和计算机采集 |

**说明**：温度计的选择需要综合考虑量程、精度、响应时间、环境条件和经济成本。第零定律告诉我们——只要能正确建立热平衡，任何测温方法都是可行的。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|:----|:---------|
| "第零定律是热力学中最重要的定律" | 第零定律是**最基础**的（定义了温度），但并非"最重要"——第一、第二定律在应用上更重要 |
| "第零定律只适用于温度测量" | 第零定律的核心是**热平衡的传递性**——它是温度作为状态参量的存在性基础，不仅仅是测温 |
| "温度计读数就是被测物体的真实温度" | 温度计读数代表温度计本身的热平衡温度。只有正确使用（充分接触、达到平衡），它才等于被测物体的温度 |
| "水银温度计比电子温度计更准" | 精度取决于校准和设计，与原理无关。铂电阻温度计比水银温度计精确得多 |
| "冰点和沸点是摄氏温度的自然固定点" | 固定点是人为定义的。热力学温标（开尔文）只有一个固定点——水的三相点 |

## 练习题

### 练习1（基础巩固）

简述热力学第零定律的内容，并解释为什么需要这个定律来定义温度。如果没有第零定律，温度测量会遇到什么问题？

<details>
<summary><b>参考答案</b></summary>

**第零定律**：如果两个热力学系统中的每一个都与第三个系统处于热平衡，则它们彼此也处于热平衡。

**为什么需要第零定律**：第零定律保证了"温度"是一个良定义的物理量。它使"温度相等"成为等价关系，满足自反性、对称性和传递性。没有它，我们无法确保：如果 A 的温度与 C 相同，B 的温度也与 C 相同，那么 A 与 B 的温度一定相同。

**没有第零定律的问题**：如果没有传递性，可能出现 A 与 C 热平衡、B 与 C 热平衡、但 A 与 B 不热平衡的矛盾情况——这意味着"温度"概念根本不成立，温度测量也失去了逻辑基础。

</details>

### 练习2（迁移应用）

一支电阻温度计的电阻 $R$ 与温度 $t$ 的关系为 $R(t) = R_0(1 + \alpha t)$，其中 $R_0 = 100.0\,\Omega$，$\alpha = 0.00400/^\circ\mathrm{C}$。

(1) 冰点时电阻是多少？沸点时又是多少？
(2) 插入某液体中测得电阻 $R = 115.2\,\Omega$，液体的温度是多少？
(3) 如果用同一温度计测量 $500^\circ\mathrm{C}$ 的高温，该线性关系还成立吗？为什么？

<details>
<summary><b>参考答案</b></summary>

**(1)** 冰点 $t_0 = 0^\circ\mathrm{C}$：

$R(0) = R_0(1 + \alpha \times 0) = R_0 = 100.0\,\Omega$

沸点 $t_{100} = 100^\circ\mathrm{C}$：

$R(100) = 100.0 \times (1 + 0.00400 \times 100) = 100.0 \times (1 + 0.40) = 140.0\,\Omega$

**(2)** $R = 115.2 = 100.0(1 + 0.00400 t)$

$1 + 0.00400t = \frac{115.2}{100} = 1.152$

$0.00400t = 0.152$

$t = \frac{0.152}{0.00400} = 38.0^\circ\mathrm{C}$

**(3)** $500^\circ\mathrm{C}$ 时，纯金属的电阻-温度关系不再是简单的线性关系——电阻率 $\rho$ 与温度的关系包含高阶项（通常用 Callendar-Van Dusen 方程描述）。如果仍用线性近似，会引入显著误差。实际精密测温中，铂电阻温度计的使用范围通常在 $-260^\circ\mathrm{C}$ 到 $660^\circ\mathrm{C}$，但需要非线性校准。

</details>
