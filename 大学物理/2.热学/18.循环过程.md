## 2. 背景与应用场景

所有热机（蒸汽轮机、汽油发动机、燃气轮机）都基于循环过程——工质从高温热源吸热，部分转化为机械功，剩余热量排向低温热源。发电厂的蒸汽动力循环中，锅炉中水吸收热量变为高温高压蒸汽，推动汽轮机做功，乏汽在冷凝器中凝结放热——水又回到初始状态，循环往复。

> **直觉理解**：热机循环就像"借鸡生蛋"——从高温热源"借"来热量，一部分"孵出"机械功（鸡蛋），剩下的"还"给低温热源。效率就是"鸡蛋"占"借来热量"的比例。

---

## 3. 核心概念与符号约定

### 循环过程的特征

系统经历一系列变化后回到初始状态——$\Delta U = 0$（内能为状态量，循环一圈后不变）。

由第一定律：$Q_{\text{net}} = W_{\text{net}}$

即：循环中系统从外界吸收的净热量等于系统对外做的净功。

### 正循环（热机）

$p$-$V$ 图上沿**顺时针**方向进行的循环。

- 系统从高温热源吸热 $Q_1$
- 系统向低温热源放热 $Q_2$（$Q_2 < 0$，取绝对值 $|Q_2|$）
- 对外做功 $W = Q_1 - |Q_2|$

**热机效率**：

$$
\eta = \frac{W}{Q_1} = \frac{Q_1 - |Q_2|}{Q_1} = 1 - \frac{|Q_2|}{Q_1}
$$

### 逆循环（制冷机/热泵）

$p$-$V$ 图上沿**逆时针**方向进行的循环。

- 系统从低温热源吸热 $Q_2$
- 外界对系统做功 $W$
- 系统向高温热源放热 $Q_1 = Q_2 + W$

**制冷系数**：

$$
\varepsilon = \frac{Q_2}{W} = \frac{Q_2}{Q_1 - Q_2}
$$

**热泵系数**：$\varepsilon' = \frac{Q_1}{W} = \frac{Q_1}{Q_1 - Q_2}$

### 符号约定

| 符号 | 含义 |
|:----:|:------|
| $Q_1$ | 从高温热源吸收的热量（正循环） |
| $Q_2$ | 向低温热源放出的热量（取绝对值） |
| $W$ | 系统对外做的净功 |
| $\eta$ | 热机效率 |
| $\varepsilon$ | 制冷系数 |

---

## 4. 原理与方法

### $p$-$V$ 图上循环的分析

1. 将循环分解为若干基本过程（等容、等压、等温、绝热）
2. 分别计算每个过程的 $Q$、$W$、$\Delta U$
3. 汇总得到循环净功 $W_{\text{net}}$ 和净热 $Q_{\text{net}}$
4. 确定 $Q_1$（吸热总量）和 $|Q_2|$（放热总量）
5. 计算效率 $\eta = W_{\text{net}}/Q_1$

### 循环净功的几何意义

$p$-$V$ 图上，循环曲线所围成的面积等于系统对外做的净功。

- 顺时针循环：围成的面积为**正**（热机）
- 逆时针循环：围成的面积为**负**（制冷机）

### 永动机的不可能性

第一类永动机（不消耗能量而持续对外做功）违反了热力学第一定律——因为循环一次 $\Delta U = 0$，$W$ 必须有 $Q$ 来补偿。

## 5. 例题

### 例题 1：简单循环效率

$1\,\mathrm{mol}$ 单原子理想气体经历一个循环：$A\to B$ 等容升温（$300\to600\,\mathrm{K}$），$B\to C$ 等压膨胀，$C\to A$ 等温压缩。求循环效率。

**解**：

**过程 $A\to B$（等容）**：

$W_{AB} = 0$

$Q_{AB} = nC_V\Delta T = 1 \times \frac32 \times 8.314 \times 300 = 3741\,\mathrm{J}$（吸热）

**过程 $B\to C$（等压）**：

先确定 $C$ 状态。$C$ 与 $A$ 等温，$T_C = T_A = 300\,\mathrm{K}$。等压过程 $\frac{V}{T}$ 为常数：

$V_B = V_A$，$\frac{V_C}{V_B} = \frac{T_C}{T_B} = \frac{300}{600} = 0.5$

$W_{BC} = p_B(V_C - V_B) = nR(T_C - T_B) = 8.314 \times (300 - 600) = -2494\,\mathrm{J}$（被压缩）

$Q_{BC} = nC_p\Delta T = 1 \times \frac52 \times 8.314 \times (300-600) = -6236\,\mathrm{J}$（放热）

**过程 $C\to A$（等温）**：

$\Delta U_{CA} = 0$

$W_{CA} = nRT_C\ln\frac{V_A}{V_C} = 1 \times 8.314 \times 300 \times \ln\frac{V_A}{0.5V_A} = 2494 \times \ln 2 = 1728\,\mathrm{J}$（膨胀做功）

$Q_{CA} = W_{CA} = 1728\,\mathrm{J}$（吸热）

**总吸热**：$Q_1 = Q_{AB} + Q_{CA} = 3741 + 1728 = 5469\,\mathrm{J}$

**净功**：$W_{\text{net}} = 0 + (-2494) + 1728 = -766\,\mathrm{J}$

**热机效率**：$\eta = \frac{W_{\text{net}}}{Q_1} = \frac{766}{5469} \approx 0.140 = 14.0\%$

### 例题 2：制冷机

一台冰箱从冷冻室（$-10^\circ\mathrm{C}$）吸热 $Q_2 = 120\,\mathrm{J}$，向环境（$25^\circ\mathrm{C}$）放热 $Q_1 = 150\,\mathrm{J}$。求：

(1) 压缩机做的功
(2) 制冷系数

**解**：

**(1)** $W = Q_1 - Q_2 = 150 - 120 = 30\,\mathrm{J}$

**(2)** $\varepsilon = \frac{Q_2}{W} = \frac{120}{30} = 4.0$

**结论**：压缩机每做 $1\,\mathrm{J}$ 的功，能从冷冻室抽出 $4\,\mathrm{J}$ 的热量。

### 例题 3：循环的 $p$-$V$ 图分析

某热机循环由两个等容过程和两个等压过程组成。工质为 $2\,\mathrm{mol}$ 双原子气体。$T_A = 300\,\mathrm{K}$，$T_B = 500\,\mathrm{K}$，$T_C = 400\,\mathrm{K}$，$T_D = 240\,\mathrm{K}$。求循环效率。

**解**：

**建模**：循环顺序为 $A\to B$（等容），$B\to C$（等压），$C\to D$（等容），$D\to A$（等压）。

$A\to B$（等容吸热）：$Q_{AB} = nC_V(T_B - T_A) = 2 \times \frac52 R \times 200 = 2 \times 20.8 \times 200 = 8320\,\mathrm{J}$

$B\to C$（等压放热）：$Q_{BC} = nC_p(T_C - T_B) = 2 \times \frac72 R \times (-100) = 2 \times 29.1 \times (-100) = -5820\,\mathrm{J}$

$C\to D$（等容放热）：$Q_{CD} = nC_V(T_D - T_C) = 2 \times 20.8 \times (-160) = -6656\,\mathrm{J}$

$D\to A$（等压吸热）：$Q_{DA} = nC_p(T_A - T_D) = 2 \times 29.1 \times 60 = 3492\,\mathrm{J}$

**总吸热**：$Q_1 = Q_{AB} + Q_{DA} = 8320 + 3492 = 11812\,\mathrm{J}$

**总放热**：$|Q_2| = |Q_{BC}| + |Q_{CD}| = 5820 + 6656 = 12476\,\mathrm{J}$

**净功**：$W = Q_1 - |Q_2| = 11812 - 12476 = -664\,\mathrm{J}$

净功为负——这不是热机循环，而是制冷机循环！

**重新判断**：检查循环方向——$A\to B$ 等容升温（压强增大），$B\to C$ 等压降温（体积缩小），$C\to D$ 等容降温（压强减小），$D\to A$ 等压升温（体积增大）。在 $p$-$V$ 图上为逆时针——制冷机。

制冷系数：$\varepsilon = \frac{Q_2}{W} = \frac{12476}{664} \approx 18.8$

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|:----|:---------|
| "循环效率可以超过 $100\%$" | 由热力学第一定律，$W \le Q_1$，所以 $\eta \le 100\%$——实际还远小于 $100\%$ |
| "$p$-$V$ 图上任意闭合曲线都代表热机循环" | 顺时针为热机，逆时针为制冷机 |
| "效率只取决于 $Q_1$ 和 $Q_2$ 怎么算" | 吸热 $Q_1$ 是**总吸热**（所有正的热量之和），不是"某个过程的吸热" |
| "循环过程中 $\Delta U = 0$ 意味着系统状态不变" | $\Delta U = 0$ 是循环一周的内能变化，但过程中内能可以变化（回零） |
| "制冷机的制冷系数可以无限大" | 由第二定律的限制，$\varepsilon$ 有上限 |

## 练习题

### 练习1（基础巩固）

$1\,\mathrm{mol}$ 单原子理想气体循环：$A(300\,\mathrm{K})\to B(600\,\mathrm{K})$ 等容，$B\to C$ 等温膨胀至 $V_C = 2V_B$，$C\to A$ 等压压缩。求循环效率和净功。

<details>
<summary><b>参考答案</b></summary>

$A\to B$（等容）：$W_{AB}=0$，$Q_{AB}=nC_V\Delta T = \frac32 R \times 300 = 3741\,\mathrm{J}$

$B\to C$（等温）：$\Delta U=0$，$W_{BC}=nRT_B\ln\frac{V_C}{V_B} = 8.314\times600\times\ln2 = 3457\,\mathrm{J}$，$Q_{BC}=3457\,\mathrm{J}$

$C\to A$（等压）：$T_C = T_B = 600\,\mathrm{K}$，$T_A = 300\,\mathrm{K}$

$W_{CA} = nR(T_A-T_C) = 8.314\times(300-600) = -2494\,\mathrm{J}$

$Q_{CA} = nC_p(T_A-T_C) = \frac52 R \times (-300) = -6236\,\mathrm{J}$

**净功**：$W_{\text{net}} = 0 + 3457 + (-2494) = 963\,\mathrm{J}$

**总吸热**：$Q_1 = 3741 + 3457 = 7198\,\mathrm{J}$

**效率**：$\eta = 963/7198 \approx 13.4\%$

</details>

### 练习2（迁移应用）

一个热机在 $T_1 = 400\,\mathrm{K}$ 和 $T_2 = 300\,\mathrm{K}$ 之间工作，每个循环从高温热源吸热 $Q_1 = 2000\,\mathrm{J}$，向低温热源放热 $Q_2 = 1600\,\mathrm{J}$。

(1) 求循环效率
(2) 如果该热机改为逆循环做制冷机，在相同温度区间内工作，输入 $W = 400\,\mathrm{J}$ 的功，能从低温热源吸取多少热量？
(3) 比较该效率与工作在相同温度区间的卡诺热机效率

<details>
<summary><b>参考答案</b></summary>

**(1)** $\eta = \frac{W}{Q_1} = \frac{Q_1-Q_2}{Q_1} = \frac{2000-1600}{2000} = 0.20 = 20\%$

**(2)** 逆循环（制冷机）：$W = Q_1' - Q_2'$，假设同样的温度区间

制冷系数 $\varepsilon = \frac{Q_2}{W} = \frac{1600}{400} = 4.0$

若输入 $W = 400\,\mathrm{J}$：$Q_2 = \varepsilon W = 4.0 \times 400 = 1600\,\mathrm{J}$

**(3)** 工作在 $400\,\mathrm{K}$ 和 $300\,\mathrm{K}$ 之间的卡诺热机效率：

$\eta_C = 1 - \frac{T_2}{T_1} = 1 - \frac{300}{400} = 0.25 = 25\%$

该热机效率 $20\%$ 低于卡诺效率 $25\%$——符合卡诺定理（任何实际热机效率不超过工作在相同温度区间的卡诺热机效率）。

</details>
