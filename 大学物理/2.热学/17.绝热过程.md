# 17. 绝热过程

## 2. 背景与应用场景

青藏铁路的冻土问题是工程建设中的一大挑战。工程师在部分路段使用了"热棒"技术——热棒中填充氨水，氨在气温低时凝结放热（向下散热），气温高时蒸发吸热（向上吸热）。这一过程涉及绝热膨胀和压缩的循环，核心是利用了绝热过程中温度随压强的变化。

> **直觉理解**：用打气筒快速打气时，气筒下端会明显发热——你快速压缩空气，热量来不及散发（近似绝热），空气温度升高。反过来，快速打开一罐压缩气体，气罐会变冷——气体快速膨胀（近似绝热），温度急剧下降。绝热过程就是"快得来不及散热"的过程。

绝热过程在工程中有着广泛应用：内燃机的压缩冲程、空气分离液化、气动工具和制冷循环。

---

## 3. 核心概念与符号约定

### 绝热过程

系统与外界之间没有热量交换的过程，$Q = 0$。

**近似条件**：过程进行得非常快（远快于热传导速度），或系统绝热良好。

### 绝热过程方程

$$
pV^\gamma = \text{常量},\quad TV^{\gamma-1} = \text{常量},\quad p^{1-\gamma}T^\gamma = \text{常量}
$$

其中 $\gamma = C_p/C_V$。

### 绝热过程中的能量转换

由 $Q = 0$ 和 $Q = \Delta U + W$：

$W = -\Delta U = -nC_V\Delta T$

- 绝热膨胀（$W > 0$）：$\Delta U < 0$，温度降低
- 绝热压缩（$W < 0$）：$\Delta U > 0$，温度升高

### 绝热线 vs 等温线

| 特征 | 等温线 | 绝热线 |
|:----|:------|:------|
| 方程 | $pV = \text{常}$ | $pV^\gamma = \text{常}$ |
| 斜率 | $\left(\dfrac{\mathrm{d}p}{\mathrm{d}V}\right)_T = -\dfrac{p}{V}$ | $\left(\dfrac{\mathrm{d}p}{\mathrm{d}V}\right)_S = -\gamma\dfrac{p}{V}$ |
| 陡峭程度 | 较平坦 | 较陡（$\gamma > 1$） |

**为什么绝热线更陡**：等温膨胀时压强降低仅因为体积增大；绝热膨胀时压强降低既因为体积增大，又因为温度降低（内能做功）。

---

## 4. 原理与方法

### 绝热过程的定量分析

**绝热过程的功**（由 $W = -\Delta U$ 直接推导）：

方法一（从内能变化计算）：$W = nC_V(T_1 - T_2)$

方法二（从 $p$-$V$ 积分计算）：$W = \frac{p_1V_1 - p_2V_2}{\gamma - 1} = \frac{nR(T_1 - T_2)}{\gamma - 1}$

两种方法等价（因为 $C_V = R/(\gamma-1)$）。

### 常见气体的 $\gamma$ 值

| 气体 | $\gamma$ | 说明 |
|:----|:--------:|:------|
| $\mathrm{He, Ne, Ar}$ | $1.67$ | 单原子 |
| $\mathrm{N_2, O_2, H_2}$ | $1.40$ | 双原子（室温） |
| $\mathrm{CO_2}$ | $1.30$ | 三原子线性 |
| 湿蒸汽 | $1.13$ | — |
| 过热水蒸气 | $1.30$ | — |

### 解题方法

1. 确认 $Q = 0$
2. 用 $\Delta U = nC_V\Delta T$ 求内能变化
3. $W = -\Delta U$
4. 如需 $p$-$V$-$T$ 关系，使用绝热过程方程
5. 比较 $p$-$V$ 图上绝热线和等温线的斜率

## 5. 例题

### 例题 1：绝热膨胀降温

$1\,\mathrm{mol}$ 单原子理想气体从 $p_1 = 5.0\,\mathrm{atm}$、$V_1 = 2.0\,\mathrm{L}$ 绝热膨胀到 $V_2 = 4.0\,\mathrm{L}$。求末态压强、温度和做功。

**解**：

**建模**：绝热过程 $p_1V_1^\gamma = p_2V_2^\gamma$，$\gamma = 5/3$。

**初态**：$T_1 = \frac{p_1V_1}{nR} = \frac{5 \times 1.013\times10^5 \times 2.0\times10^{-3}}{1 \times 8.314} = \frac{1013}{8.314} \approx 122\,\mathrm{K}$

**末态压强**：

$p_2 = p_1\left(\frac{V_1}{V_2}\right)^\gamma = 5.0 \times \left(\frac{2}{4}\right)^{5/3} = 5.0 \times (0.5)^{1.67}$

$0.5^{1.67} = e^{1.67\ln0.5} = e^{1.67 \times (-0.693)} = e^{-1.157} = 0.315$

$p_2 = 5.0 \times 0.315 = 1.58\,\mathrm{atm}$

**末态温度**：

$T_2 = T_1\left(\frac{V_1}{V_2}\right)^{\gamma-1} = 122 \times (0.5)^{0.67}$

$0.5^{0.67} = e^{0.67\ln0.5} = e^{-0.464} = 0.629$

$T_2 = 122 \times 0.629 = 76.7\,\mathrm{K}$

**做功**：

$\Delta U = nC_V(T_2 - T_1) = 1 \times \frac32 \times 8.314 \times (76.7 - 122) = 1.5 \times 8.314 \times (-45.3) = -565\,\mathrm{J}$

$W = -\Delta U = 565\,\mathrm{J}$

**结论**：绝热膨胀后，压强降到 $1.58\,\mathrm{atm}$，温度降到 $76.7\,\mathrm{K}$（$-196^\circ\mathrm{C}$），气体对外做功 $565\,\mathrm{J}$。

### 例题 2：绝热压缩的热效应

柴油机的压缩比为 $20:1$，吸入空气温度 $27^\circ\mathrm{C}$。假设空气绝热压缩，$\gamma = 1.40$。求压缩后的温度。这个温度能否使柴油自燃（柴油自燃点约 $210^\circ\mathrm{C}$）？

**解**：

**建模**：绝热压缩，$TV^{\gamma-1} = \text{常}$。$V_2/V_1 = 1/20$。

$T_2 = T_1\left(\frac{V_1}{V_2}\right)^{\gamma-1} = 300 \times 20^{0.40}$

$20^{0.40} = e^{0.40\ln20} = e^{0.40 \times 3.00} = e^{1.20} = 3.32$

$T_2 = 300 \times 3.32 = 996\,\mathrm{K} \approx 723^\circ\mathrm{C}$

**结论**：压缩后温度高达 $723^\circ\mathrm{C}$，远高于柴油的自燃点（$210^\circ\mathrm{C}$）。因此柴油喷入气缸后立即自燃，无需火花塞。

**现实对比**：实际柴油机中空燃比、传热损失和燃烧室冷却会使温度低于理论值，但仍足以自燃。

### 例题 3：绝热线与等温线的比较

在同一 $p$-$V$ 图上画出一理想气体的等温线和绝热线。证明在任意交点处绝热线的斜率绝对值是等温线的 $\gamma$ 倍。

**解**：

**等温线**：$pV = \text{常}$，两边对 $V$ 求导：

$p + V\frac{\mathrm{d}p}{\mathrm{d}V} = 0 \Rightarrow \left(\frac{\mathrm{d}p}{\mathrm{d}V}\right)_T = -\frac{p}{V}$

**绝热线**：$pV^\gamma = \text{常}$，两边对 $V$ 求导：

$V^\gamma\frac{\mathrm{d}p}{\mathrm{d}V} + p\gamma V^{\gamma-1} = 0$

$\frac{\mathrm{d}p}{\mathrm{d}V} + \frac{\gamma p}{V} = 0 \Rightarrow \left(\frac{\mathrm{d}p}{\mathrm{d}V}\right)_S = -\frac{\gamma p}{V}$

**比较**：$\left|\frac{\mathrm{d}p}{\mathrm{d}V}\right|_S = \gamma \cdot \left|\frac{\mathrm{d}p}{\mathrm{d}V}\right|_T$

因为 $\gamma > 1$，在任意交点处，绝热线斜率绝对值是等温线的 $\gamma$ 倍，所以绝热线更陡。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|:----|:---------|
| "绝热过程就是温度不变" | 绝热是 $Q=0$，不是 $T=\text{常}$。绝热膨胀温度降低，绝热压缩温度升高 |
| "绝热过程 $\gamma$ 对所有气体都相同" | $\gamma = (i+2)/i$，取决于分子自由度——单原子 $1.67$，双原子 $1.40$ |
| "绝热线和等温线在 $p$-$V$ 图上会交叉两次" | 从同一初态出发，绝热线在等温线下方（膨胀时）或上方（压缩时），不会再次交叉 |
| "绝热过程的功只能用 $W = \int p\,\mathrm{d}V$ 计算" | $W = -\Delta U = nC_V(T_1-T_2)$ 往往更简便——无需积分 |
| "快速膨胀一定是绝热" | 快速膨胀近似绝热（来不及热交换），但严格绝热需要系统完全隔热 |

## 练习题

### 练习1（基础巩固）

$1\,\mathrm{mol}$ 双原子理想气体从 $T_1 = 300\,\mathrm{K}$、$p_1 = 3.0\,\mathrm{atm}$ 绝热膨胀到 $p_2 = 1.0\,\mathrm{atm}$。求 $T_2$、$W$、$\Delta U$。

<details>
<summary><b>参考答案</b></summary>

双原子 $\gamma = 1.40$

由 $p^{1-\gamma}T^\gamma = \text{常}$，$p_1^{1-\gamma}T_1^\gamma = p_2^{1-\gamma}T_2^\gamma$

$T_2 = T_1\left(\frac{p_2}{p_1}\right)^{(1-\gamma)/\gamma} = 300 \times \left(\frac{1}{3}\right)^{(1-1.4)/1.4} = 300 \times 3^{(0.4)/(1.4)}$

$T_2 = 300 \times 3^{0.286} = 300 \times 1.369 = 411\,\mathrm{K}$

$\Delta U = nC_V(T_2 - T_1) = 1 \times \frac52 \times 8.314 \times (411 - 300) = 2.5 \times 8.314 \times 111 = 2307\,\mathrm{J}$

$W = -\Delta U = -2307\,\mathrm{J}$

绝热膨胀后温度上升到 $411\,\mathrm{K}$——压强是原来的 $1/3$ 但温度反而升高了？检查数据。

问题发现：从 $3\,\mathrm{atm}$ 到 $1\,\mathrm{atm}$ 的**膨胀**（$p$ 减小），$V$ 应增大。但 $T_2 > T_1$ 意味着温度升高——这在绝热膨胀中不可能发生！

**错误原因**：应该用 $pV^\gamma$ 规律。从 $p_1=3\to p_2=1$ 是膨胀过程（$V$ 增大），绝热膨胀应该降温。公式应使用 $T \propto p^{(\gamma-1)/\gamma}$：

$T_2 = T_1 \left(\frac{p_2}{p_1}\right)^{(\gamma-1)/\gamma} = 300 \times \left(\frac{1}{3}\right)^{0.4/1.4} = 300 \times 3^{-0.286} = 300 \times 0.731 = 219\,\mathrm{K}$

$\Delta U = 1 \times 2.5 \times 8.314 \times (219-300) = 2.5 \times 8.314 \times (-81) = -1684\,\mathrm{J}$

$W = -\Delta U = 1684\,\mathrm{J}$

$T_2 = 219\,\mathrm{K} = -54^\circ\mathrm{C}$——绝热膨胀显著降温。

</details>

### 练习2（迁移应用）

空气 ($\gamma = 1.40$) 从 $27^\circ\mathrm{C}$、$1.0\,\mathrm{atm}$ 绝热压缩到压强为 $10\,\mathrm{atm}$。

(1) 求压缩后的温度
(2) 如果改为等温压缩到同样的末压强，体积比 $V_2/V_1$ 是多少？
(3) 绝热和等温两种情况下，外界对气体做的功哪个更大？为什么？

<details>
<summary><b>参考答案</b></summary>

**(1)** $T_2 = T_1 \left(\frac{p_2}{p_1}\right)^{(\gamma-1)/\gamma} = 300 \times 10^{(0.4/1.4)} = 300 \times 10^{0.286}$

$10^{0.286} = e^{0.286\ln10} = e^{0.286 \times 2.303} = e^{0.658} = 1.931$

$T_2 = 300 \times 1.931 = 579\,\mathrm{K} \approx 306^\circ\mathrm{C}$

**(2)** 等温压缩：$p_1V_1 = p_2V_2$，$\frac{V_2}{V_1} = \frac{p_1}{p_2} = \frac{1}{10} = 0.1$

**(3)** 绝热压缩的末态温度更高（$579\,\mathrm{K}$ vs $300\,\mathrm{K}$），因此 $\Delta U$ 更大。由 $W = -\Delta U$（绝热过程），所以外界对气体做的功更多。

物理原因：等温压缩时热量可以散发到恒温热源，温度不变；绝热压缩时所有压缩功都转化为内能，温度升高、压强增大更快，需要更多压缩功才能达到同样的末压强。

</details>
