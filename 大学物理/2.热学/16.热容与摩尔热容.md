## 2. 背景与应用场景

暖气片的热容远大于同质量的水——水比热容大（$C = 4.2\,\mathrm{J/(g\cdot K)}$），是铜的 $10$ 倍。这就是为什么热水供暖系统用"水"作为热载体：水能储存大量热量，通过管道输送到各房间后缓慢释放。理解热容是能量管理和热工设计的核心。

> **直觉理解**：热容就是"物体的储热能力"——同样升 $1^\circ\mathrm{C}$，水比铁需要更多热量，所以水更适合做储热介质。而对气体来说，同样升温 $1^\circ\mathrm{C}$，等压过程比等容过程需要更多热量——因为膨胀时还要对外做功。

---

## 3. 核心概念与符号约定

### 热容的定义

**比热容**：单位质量物质温度升高 $1\,\mathrm{K}$ 所需的热量，$c = \frac{Q}{m\Delta T}$，单位 $\mathrm{J/(kg\cdot K)}$。

**摩尔热容**：$1\,\mathrm{mol}$ 物质温度升高 $1\,\mathrm{K}$ 所需的热量，$C = \frac{Q}{n\Delta T}$，单位 $\mathrm{J/(mol\cdot K)}$。

**关键点**：热容与过程有关——对气体，等容和等压条件下的热容不同。

### 定容摩尔热容 $C_V$

$C_V = \left(\frac{Q}{n\Delta T}\right)_V = \frac{\Delta U}{n\Delta T} = \frac{i}{2} R$

等容下，$Q = \Delta U$，所以 $C_V = \frac{1}{n}\frac{\mathrm{d}U}{\mathrm{d}T}$。

### 定压摩尔热容 $C_p$

$C_p = \left(\frac{Q}{n\Delta T}\right)_p = \frac{\Delta U + p\Delta V}{n\Delta T} = C_V + R$

### 迈耶公式

$$
C_p - C_V = R
$$

**物理意义**：相同 $1\,\mathrm{mol}$ 气体升高 $1\,\mathrm{K}$，等压过程比等容过程多吸收 $R \approx 8.314\,\mathrm{J}$ 的热量——这部分热量用于对外膨胀做功。

### 比热比 $\gamma$

$$
\gamma = \frac{C_p}{C_V} = \frac{i + 2}{i}
$$

| 分子类型 | $i$ | $C_V$ | $C_p$ | $\gamma$ |
|:--------:|:---:|:----:|:----:|:--------:|
| 单原子 | 3 | $\frac32 R$ | $\frac52 R$ | $1.67$ |
| 双原子刚性 | 5 | $\frac52 R$ | $\frac72 R$ | $1.40$ |
| 多原子非线性 | 6 | $3R$ | $4R$ | $1.33$ |

---

## 4. 原理与方法

### 为什么 $C_p > C_V$？

等容过程：吸收的热量 $Q_V$ 全部用于增加内能（升温），不做功。

等压过程：吸收的热量 $Q_p$ 一部分增加内能（升温），一部分对外做功（膨胀）。

因为升温相同（内能增量相同），等压过程需要多吸收用于做功的那部分热量：$Q_p - Q_V = p\Delta V = nR\Delta T$。

对 $1\,\mathrm{mol}$ 气体：$C_p - C_V = R$。

### 经典热容的局限

经典理论（能量均分定理）预测：
- 单原子：$C_V = 3R/2$
- 双原子：$C_V = 5R/2$
- 多原子：$C_V = 3R$

但实验表明：
- 低温下 $C_V$ 低于经典预测——量子效应使自由度"冻结"
- 高温下 $C_V$ 趋于经典值——自由逐渐"解冻"

### 热容的应用

| 应用 | 公式 | 说明 |
|:----|:-----|:------|
| 等容过程吸热 | $Q_V = nC_V\Delta T$ | 体积不变时 |
| 等压过程吸热 | $Q_p = nC_p\Delta T$ | 压强不变时 |
| 绝热过程 $\gamma$ | $\gamma = C_p/C_V$ | 决定绝热线形状 |
| 内能变化 | $\Delta U = nC_V\Delta T$ | 与过程无关 |

## 5. 例题

### 例题 1：$C_V$ 和 $C_p$ 的计算

求氧气（$\mathrm{O_2}$，刚性双原子）的 $C_V$、$C_p$ 和 $\gamma$。

**解**：

双原子 $i=5$：

$C_V = \frac52 R = \frac52 \times 8.314 = 20.79\,\mathrm{J/(mol\cdot K)}$

$C_p = C_V + R = 20.79 + 8.314 = 29.10\,\mathrm{J/(mol\cdot K)}$

$\gamma = \frac{C_p}{C_V} = \frac{29.10}{20.79} = 1.40$

**验证**：$\gamma = \frac{i+2}{i} = \frac{7}{5} = 1.40$ ✅

### 例题 2：等容 vs 等压加热

$2\,\mathrm{mol}$ 氮气（$\mathrm{N_2}$，双原子）从 $300\,\mathrm{K}$ 加热到 $400\,\mathrm{K}$。分别计算等容和等压过程中气体吸收的热量。

**解**：

$C_V = \frac52 R = 20.79\,\mathrm{J/(mol\cdot K)}$

$C_p = \frac72 R = 29.10\,\mathrm{J/(mol\cdot K)}$

**等容**：$Q_V = nC_V\Delta T = 2 \times 20.79 \times 100 = 4158\,\mathrm{J}$

**等压**：$Q_p = nC_p\Delta T = 2 \times 29.10 \times 100 = 5820\,\mathrm{J}$

**差值**：$Q_p - Q_V = 5820 - 4158 = 1662\,\mathrm{J}$

这个差值正是气体膨胀对外做功：$W = p\Delta V = nR\Delta T = 2 \times 8.314 \times 100 = 1663\,\mathrm{J}$ ✅

### 例题 3：实验测定 $\gamma$

用实验方法测定某气体的比热比 $\gamma$：在绝热条件下使气体迅速膨胀，测得初态 $p_1 = 1.0\,\mathrm{atm}$，$T_1 = 300\,\mathrm{K}$；末态 $p_2 = 0.75\,\mathrm{atm}$，$T_2 = 275\,\mathrm{K}$。求该气体的 $\gamma$，并判断是单原子还是双原子气体。

**解**：

**建模**：绝热过程 $p^{1-\gamma}T^\gamma = \text{常}$。

**列式**：$p_1^{1-\gamma}T_1^\gamma = p_2^{1-\gamma}T_2^\gamma$

$\left(\frac{p_2}{p_1}\right)^{1-\gamma} = \left(\frac{T_2}{T_1}\right)^\gamma$

**取对数**：

$(1-\gamma)\ln\frac{p_2}{p_1} = \gamma\ln\frac{T_2}{T_1}$

$(1-\gamma)\ln\frac{0.75}{1.0} = \gamma\ln\frac{275}{300}$

$(1-\gamma)(-0.288) = \gamma(-0.0870)$

$-0.288 + 0.288\gamma = -0.0870\gamma$

$0.288\gamma + 0.0870\gamma = 0.288$

$0.375\gamma = 0.288$

$\gamma = 0.768$

显然不对——$\gamma$ 不可能小于 $1$。一定有计算错误。

**修正**：绝热过程公式：$p^{1-\gamma}T^\gamma = \text{常}$

取自然对数：$(1-\gamma)\ln p_1 + \gamma\ln T_1 = (1-\gamma)\ln p_2 + \gamma\ln T_2$

$(1-\gamma)(\ln p_1 - \ln p_2) = \gamma(\ln T_2 - \ln T_1)$

$(1-\gamma)\ln\frac{p_1}{p_2} = \gamma\ln\frac{T_2}{T_1}$

$\ln\frac{p_1}{p_2} = \ln\frac{1.0}{0.75} = \ln 1.333 = 0.288$

$\ln\frac{T_2}{T_1} = \ln\frac{275}{300} = \ln 0.917 = -0.0866$

$(1-\gamma) \times 0.288 = \gamma \times (-0.0866)$

$0.288 - 0.288\gamma = -0.0866\gamma$

$0.288 = 0.288\gamma - 0.0866\gamma = 0.2014\gamma$

$\gamma = \frac{0.288}{0.2014} \approx 1.43$

**结论**：$\gamma \approx 1.43$，接近双原子气体的 $\gamma = 1.40$——该气体很可能是双原子分子。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|:----|:---------|
| "热容只取决于物质本身，与过程无关" | 对固体和液体近似成立，对气体**一定**与过程有关——$C_p \neq C_V$ |
| "$C_V = \frac{i}{2}R$ 对所有温度都成立" | 仅在常温下成立——低温时自由度冻结，$C_V$ 小于经典值；高温时振动激发，$C_V$ 大于经典值 |
| "$C_p$ 很大的物质一定能储存很多热量" | $Q = C_p m\Delta T$——储热能力还取决于质量 $\times$ 温升 |
| "迈耶公式 $C_p - C_V = R$ 对所有物质都成立" | 迈耶公式仅适用于理想气体——对固体、液体不成立 |
| "热容越大导热越快" | 热容和导热系数是不同的物理量——水热容大但导热慢（静态水是良好的绝热体） |

## 练习题

### 练习1（基础巩固）

氦气（单原子）的 $C_V = \frac32 R$，$C_p = \frac52 R$。$3\,\mathrm{mol}$ 氦气从 $300\,\mathrm{K}$ 升到 $400\,\mathrm{K}$，求等容和等压过程中的吸热量。

<details>
<summary><b>参考答案</b></summary>

**等容**：$Q_V = nC_V\Delta T = 3 \times \frac32 \times 8.314 \times 100 = 3 \times 1247 = 3741\,\mathrm{J}$

**等压**：$Q_p = nC_p\Delta T = 3 \times \frac52 \times 8.314 \times 100 = 3 \times 2079 = 6237\,\mathrm{J}$

差值 $6237 - 3741 = 2496\,\mathrm{J} \approx nR\Delta T = 3 \times 8.314 \times 100 = 2494\,\mathrm{J}$ ✅

</details>

### 练习2（迁移应用）

$1\,\mathrm{mol}$ 理想气体经历一个过程，吸收 $500\,\mathrm{J}$ 热量，温度从 $300\,\mathrm{K}$ 升到 $330\,\mathrm{K}$。已知该气体的 $C_V = 20.8\,\mathrm{J/(mol\cdot K)}$。

(1) 求 $\Delta U$
(2) 求 $W$
(3) 判断这是等容、等压还是其他过程？

<details>
<summary><b>参考答案</b></summary>

**(1)** $\Delta U = nC_V\Delta T = 1 \times 20.8 \times 30 = 624\,\mathrm{J}$

**(2)** 由第一定律 $Q = \Delta U + W$：

$W = Q - \Delta U = 500 - 624 = -124\,\mathrm{J}$

负号表示外界对气体做功（气体被压缩）。

**(3)** 这不是等容过程（等容$W=0$，但这里$W\neq0$），也不是等压过程（等压$W=nR\Delta T = 8.314\times30=249\,\mathrm{J}$，但实际$W=-124\,\mathrm{J}$——符号相反）。

这是一个**压缩**过程——气体温度升高但对外界放热不足，需要外界做功来补偿。实际中，这个过程可能是**多方过程**（介于绝热和等温之间）。

分析总结：$\Delta U(=624) > Q(=500)$，所以 $W=Q-\Delta U<0$，外界对系统做功 $124\,\mathrm{J}$。

</details>
