## 2. 背景与应用场景

$1824$ 年，年仅 $28$ 岁的法国工程师卡诺发表了一篇论文《关于火的动力之思考》。在这篇论文中，他提出了一个思想实验——卡诺热机——并证明了：**任何热机的效率都不可能超过工作在相同温度区间内的卡诺热机**。这个结论后来成为热力学第二定律的基础。

> **直觉理解**：卡诺热机是所有热机中的"天花板"——即使没有任何摩擦和热损失，最高效率也只由高、低温热源的温度决定。就像瀑布发电——水位差越大，发电潜力越大。想提高效率，要么提高热源温度（$T_1\uparrow$），要么降低冷源温度（$T_2\downarrow$）。

卡诺循环的效率公式 $\eta_C = 1 - T_2/T_1$ 是热力学中最优美、最深刻的公式之一——只依赖于两热源的温度，与工质种类无关。

---

## 3. 核心概念与符号约定

### 卡诺循环的四个步骤

卡诺循环由两个等温过程和两个绝热过程组成。

| 步骤 | 过程 | 描述 | $p$-$V$ 图上的走向 |
|:---:|:----|:-----|:-----------------:|
| $1\to 2$ | 等温膨胀 | 工质与高温热源 $T_1$ 接触，吸热 $Q_1$ 并对外做功 | 向右下（等温线） |
| $2\to 3$ | 绝热膨胀 | 工质与热源隔绝，继续膨胀做功，温度降到 $T_2$ | 向右下（绝热线，更陡） |
| $3\to 4$ | 等温压缩 | 工质与低温热源 $T_2$ 接触，放热 $Q_2$，外界对工质做功 | 向左上（等温线） |
| $4\to 1$ | 绝热压缩 | 工质与热源隔绝，继续压缩，温度升回 $T_1$ | 向左上（绝热线） |

### 卡诺热机效率

$$
\eta_C = 1 - \frac{T_2}{T_1} = \frac{T_1 - T_2}{T_1}
$$

其中 $T_1$ 为高温热源温度，$T_2$ 为低温热源温度（单位：$\mathrm{K}$）。

### 卡诺制冷系数

$$
\varepsilon_C = \frac{T_2}{T_1 - T_2}
$$

**热泵系数**：$\varepsilon_C' = \frac{T_1}{T_1 - T_2}$

### 卡诺定理

> 在相同的高温热源 $T_1$ 和低温热源 $T_2$ 之间工作的一切可逆热机，效率相等，都等于 $\eta_C = 1 - T_2/T_1$。在相同温度区间工作的一切不可逆热机，效率都低于卡诺热机。

**推论**：$\eta_{\text{实际}} \le \eta_C$，等号只在理想可逆过程中成立。

## 4. 原理与方法

### 效率公式的推导

卡诺效率公式可以严格推导。核心在于：

等温膨胀 $1\to 2$ 中：$Q_1 = nRT_1\ln(V_2/V_1)$

等温压缩 $3\to 4$ 中：$|Q_2| = nRT_2\ln(V_3/V_4)$

由绝热过程方程：$T_1V_2^{\gamma-1} = T_2V_3^{\gamma-1}$ 和 $T_1V_1^{\gamma-1} = T_2V_4^{\gamma-1}$

两式相除得：$V_2/V_1 = V_3/V_4$

因此 $\frac{Q_1}{|Q_2|} = \frac{T_1}{T_2}$，$\eta = 1 - \frac{|Q_2|}{Q_1} = 1 - \frac{T_2}{T_1}$

### 提高热机效率的途径

由 $\eta_C = 1 - T_2/T_1$：

1. **提高高温热源温度 $T_1$**——这是最有效的方法（如提高蒸汽参数）
2. **降低低温热源温度 $T_2$**——但受环境温度限制，空间有限

**工程实例**：
- 早期蒸汽机：$T_1 \approx 100^\circ\mathrm{C}$（$373\,\mathrm{K}$），$T_2 \approx 40^\circ\mathrm{C}$（$313\,\mathrm{K}$），$\eta_C \approx 16\%$
- 现代超临界火电厂：$T_1 \approx 600^\circ\mathrm{C}$（$873\,\mathrm{K}$），$T_2 \approx 35^\circ\mathrm{C}$（$308\,\mathrm{K}$），$\eta_C \approx 65\%$
- 实际效率约为卡诺效率的 $60\!-\!70\%$，受摩擦、热损失、不可逆性等因素限制

### 卡诺定理的物理意义

1. 热机效率存在上限，不能无限提高
2. 效率上限仅由热源温度决定，与工质种类无关
3. 温差 $\Delta T = T_1 - T_2$ 是"驱动热机"的势能差
4. 绝对零度时效率才可能达到 $100\%$——但绝对零度不可达到

## 5. 例题

### 例题 1：卡诺热机效率

一卡诺热机在 $500\,\mathrm{K}$ 和 $300\,\mathrm{K}$ 之间工作。

(1) 求热机效率
(2) 若每个循环从高温热源吸热 $Q_1 = 1000\,\mathrm{J}$，求每个循环的净功和向低温热源放出的热量

**解**：

**(1)** $\eta_C = 1 - \frac{T_2}{T_1} = 1 - \frac{300}{500} = 0.40 = 40\%$

**(2)** $W = \eta_C Q_1 = 0.40 \times 1000 = 400\,\mathrm{J}$

$|Q_2| = Q_1 - W = 1000 - 400 = 600\,\mathrm{J}$

**验证**：由卡诺比例 $Q_1/Q_2 = T_1/T_2 = 500/300 = 1.667$，$|Q_2| = 1000/1.667 = 600\,\mathrm{J}$ ✅

### 例题 2：卡诺制冷机

一台卡诺制冷机在 $-20^\circ\mathrm{C}$ 和 $27^\circ\mathrm{C}$ 之间工作。

(1) 求制冷系数
(2) 若要从冷库中吸收 $5000\,\mathrm{J}$ 的热量，压缩机需要做多少功？
(3) 同温限下的卡诺热泵的供热系数是多少？

**解**：

**建模**：$T_2 = -20 + 273 = 253\,\mathrm{K}$（冷库），$T_1 = 27 + 273 = 300\,\mathrm{K}$（环境）

**(1)** $\varepsilon_C = \frac{T_2}{T_1 - T_2} = \frac{253}{300 - 253} = \frac{253}{47} \approx 5.38$

**(2)** $\varepsilon = \frac{Q_2}{W} \Rightarrow W = \frac{Q_2}{\varepsilon} = \frac{5000}{5.38} \approx 929\,\mathrm{J}$

**验证**：$Q_1 = Q_2 + W = 5000 + 929 = 5929\,\mathrm{J}$

$Q_1/Q_2 = 5929/5000 = 1.186$，$T_1/T_2 = 300/253 = 1.186$ ✅

**(3)** 热泵系数 $\varepsilon_C' = \frac{T_1}{T_1 - T_2} = \frac{300}{47} \approx 6.38$

或 $\varepsilon_C' = \varepsilon_C + 1 = 5.38 + 1 = 6.38$

### 例题 3：效率比较

某蒸汽发电厂的锅炉温度为 $T_1 = 560^\circ\mathrm{C}$，冷却水温度为 $T_2 = 30^\circ\mathrm{C}$。

(1) 求卡诺效率
(2) 实际电厂效率约为 $38\%$，实际效率与卡诺效率的比值是多少？
(3) 如果锅炉温度提高到 $700^\circ\mathrm{C}$，卡诺效率提高多少？

**解**：

$T_1 = 560 + 273 = 833\,\mathrm{K}$，$T_2 = 30 + 273 = 303\,\mathrm{K}$

**(1)** $\eta_C = 1 - \frac{303}{833} = 1 - 0.364 = 0.636 = 63.6\%$

**(2)** 实际 $\eta = 38\%$，$\frac{\eta}{\eta_C} = \frac{38\%}{63.6\%} = 0.597 \approx 60\%$

说明该电厂还有相当的改进空间。

**(3)** $T_1' = 700 + 273 = 973\,\mathrm{K}$

$\eta_C' = 1 - \frac{303}{973} = 1 - 0.311 = 0.689 = 68.9\%$

卡诺效率提高 $68.9\% - 63.6\% = 5.3\%$。

**工程意义**：提高 $140^\circ\mathrm{C}$ 才提升 $5.3\%$ 的效率，且对锅炉材料强度要求大幅提高。这就是效率"边际收益递减"的工程现实。

## 6. 常见误区与检查点

### 常见误区

| 误区 | 正确理解 |
|:----|:---------|
| "卡诺循环是实际可行的" | 卡诺循环是**理想**循环——可逆等温过程需要无限慢，绝热过程需要完美绝热，实际无法实现 |
| "卡诺效率公式 $\eta = 1 - T_2/T_1$ 对所有热机都成立" | 该公式只对**卡诺热机**成立——实际热机效率需按实际循环计算，但不超过此值 |
| "低温热源越低越好" | 降低 $T_2$ 确实提高效率，但实际中受环境温度限制（冷却水和空气温度），且过低需要制冷消耗额外能量 |
| "卡诺循环效率为 $100\%$ 可以做到" | 需要 $T_2=0\,\mathrm{K}$ 或 $T_1\to\infty$——两者都不可能 |
| "卡诺制冷机的制冷系数大于 $1$ 意味着违反能量守恒" | 制冷系数 $>1$ 是正常的——因为 $W$（压缩机做功）小于 $Q_2$（从冷库吸收的热量），$Q_1 = Q_2+W$ 能量守恒 |

## 练习题

### 练习1（基础巩固）

一卡诺热机在 $T_1 = 600\,\mathrm{K}$ 和 $T_2 = 400\,\mathrm{K}$ 之间工作。

(1) 求效率
(2) 若从高温热源吸热 $Q_1 = 1500\,\mathrm{J}$，求净功和放热量
(3) 如果 $T_1$ 不变，将效率提高到 $40\%$，$T_2$ 应为多少？

<details>
<summary><b>参考答案</b></summary>

**(1)** $\eta_C = 1 - \frac{400}{600} = 1 - 0.667 = 0.333 = 33.3\%$

**(2)** $W = \eta_C Q_1 = 0.333 \times 1500 = 500\,\mathrm{J}$

$|Q_2| = Q_1 - W = 1500 - 500 = 1000\,\mathrm{J}$

**(3)** $\eta = 1 - \frac{T_2}{600} = 0.40$

$\frac{T_2}{600} = 0.60$，$T_2 = 360\,\mathrm{K}$

</details>

### 练习2（迁移应用）

两个卡诺热机串联工作：热机 A 在 $T_1 = 800\,\mathrm{K}$ 和 $T_2 = 500\,\mathrm{K}$ 之间工作，热机 A 排出的热量全部供给热机 B。热机 B 在 $T_2$ 和 $T_3 = 300\,\mathrm{K}$ 之间工作。

(1) 分别求两台热机的效率
(2) 求总效率（两机之和）
(3) 比较总效率与一台在 $800\,\mathrm{K}$ 和 $300\,\mathrm{K}$ 之间工作的卡诺热机效率

<details>
<summary><b>参考答案</b></summary>

**(1)** 热机 A：$\eta_A = 1 - \frac{500}{800} = 0.375 = 37.5\%$

热机 B：$\eta_B = 1 - \frac{300}{500} = 0.400 = 40.0\%$

**(2)** 设热机 A 从热源吸热 $Q_1$，对外做功 $W_A = \eta_A Q_1$，排热 $Q_2 = Q_1 - W_A = Q_1(1-\eta_A)$

热机 B 从 $Q_2$ 吸热，做功 $W_B = \eta_B Q_2 = \eta_B Q_1(1-\eta_A)$

总功 $W = W_A + W_B = Q_1[\eta_A + \eta_B(1-\eta_A)]$

总效率 $\eta = \frac{W}{Q_1} = \eta_A + \eta_B(1-\eta_A) = 0.375 + 0.400(1-0.375)$

$= 0.375 + 0.400 \times 0.625 = 0.375 + 0.250 = 0.625 = 62.5\%$

**(3)** 单台卡诺热机在 $800\,\mathrm{K}$ 和 $300\,\mathrm{K}$ 之间工作：

$\eta_{C,\text{单}} = 1 - \frac{300}{800} = 0.625 = 62.5\%$

**完全相等！** 这说明：串联的两台卡诺热机，总效率等于在总温度区间工作的单台卡诺热机。这是卡诺定理的推论：可逆热机效率只取决于最高和最低热源温度，与中间过程无关。

</details>
